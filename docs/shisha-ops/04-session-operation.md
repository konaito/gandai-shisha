# セッション運営マニュアル

BiPシーシャにおける予約受付から退店までの運営フロー。

---

## 1. 予約受付から退店までのフロー

### 全体フロー図

```
予約受付 → 事前準備 → お客様来店 → セッション開始 → セッション中管理 → セッション終了 → 片付け・清掃
```

### 詳細フロー

#### (1) 予約受付

| 手順 | 内容 | ツール |
|------|------|--------|
| 1 | 予約リクエスト受付（Instagram DM / LINE等） | SNS |
| 2 | 日時・人数・時間（60分/90分/120分）を確認 | - |
| 3 | 希望フレーバーや好みをヒアリング（任意） | - |
| 4 | 料金を案内し、予約確定 | - |
| 5 | 予約前日にリマインドメッセージ送信 | - |

#### (2) 事前準備（来店30分前〜）

| 手順 | 内容 | 所要時間 |
|------|------|---------|
| 1 | 空間の準備（照明・音楽・エアコン） | 5分 |
| 2 | シーシャ本体の組み立て・水入れ | 5分 |
| 3 | 電熱コンロをONにして炭の着火開始 | 着火に10〜15分 |
| 4 | フレーバーの準備（ボウルへの盛り付け） | 5分 |
| 5 | マウスピース・トング等の備品セット | 2分 |
| 6 | ドリンク・お茶の準備 | 3分 |

#### (3) お客様来店

| 手順 | 内容 |
|------|------|
| 1 | 挨拶・靴の脱ぎ場所等の案内 |
| 2 | 初回のお客様には簡単なシーシャの説明（吸い方・注意点） |
| 3 | フレーバーの説明・希望確認 |
| 4 | シーシャを仕上げて提供 |

#### (4) セッション中

| 手順 | 内容 | タイミング |
|------|------|-----------|
| 1 | 最初の数口は一緒に確認（煙量・味のチェック） | 提供直後 |
| 2 | 炭の状態確認・火力調整 | 10〜15分ごと |
| 3 | 炭交換 | 30〜45分ごと |
| 4 | フレーバー交換（長時間セッションの場合） | 60〜90分で |
| 5 | ドリンクの追加確認 | 適宜 |

#### (5) セッション終了

| 手順 | 内容 |
|------|------|
| 1 | 終了時間の5分前にお知らせ |
| 2 | 料金の精算 |
| 3 | お見送り |

#### (6) 片付け・清掃（次のセッションまでに）

→ 詳細は後述「5. 片付け・清掃の手順」を参照

## 2. シーシャの作り方（ステップバイステップ）

### 必要なもの

- シーシャ本体（組み立て済み）
- ボウル
- HMD（Kaloud Lotus / Provost）
- フレーバー
- 炭（着火済み）
- トング
- アルミホイル（Provost使用時）
- マウスピース

### 手順

#### Step 1: 水の準備

```
ボトルに水を入れる
→ ダウンステム（パイプの先端）が水面下 2.5〜3.5cm に浸かる量が適正
→ 水が多すぎると吸いが重い / 少なすぎると煙がフィルタリングされない
```

> **目安**: ボトルの 1/3〜1/2 程度。氷を入れると冷たい煙になり、夏場はおすすめ。

#### Step 2: 本体の組み立て

```
1. ボトルにステムを差し込む（ゴムグロメットでしっかり密着）
2. ホースポートにホースを接続（グロメットで密着）
3. パージバルブが正常に動作するか確認（軽く息を吹き込む）
4. 吸ってみてエアリークがないか確認
```

> **気密性チェック**: ボウルポートを手で塞いで吸い込む。全く吸えなければ気密OK。少しでも空気が入るならグロメットの締め直しが必要。

#### Step 3: フレーバーの盛り付け

```
1. フレーバーをパッケージからボウルに適量取り出す
2. 大きな茎や塊があればほぐす
3. ボウルにふわっと（ふんわり）盛る
   → 押し固めない！空気の通り道を確保する
4. ボウルの縁から2〜3mm下がった位置まで盛る
   → HMDやアルミホイルに直接触れない高さ
5. 表面を軽く平らに整える
```

**盛り付けのタイプ別ガイド**:

| 盛り方 | 特徴 | 向いているフレーバー |
|--------|------|---------------------|
| **フラフィー（ふわっと）** | 空気が通りやすく、軽い吸い心地。初心者向け | Al Fakher、Fumari |
| **セミデンス** | やや押し気味。味が濃くなる | Azure、Social Smoke |
| **デンスパック（ぎっしり）** | 強く押し込む。上級者向け。バズが強い | Tangiers |

#### Step 4: HMDのセット（Kaloud Lotus の場合）

```
1. ボウルにLotusを載せる（ボウルの縁にフィットしていることを確認）
2. 着火済みの炭を3個、Lotusの中に均等に配置
3. 蓋を閉じる（通気口は半開き）
4. 3〜5分待つ（蒸らし時間）
   → この間にフレーバーが温まり、煙が出始める
```

#### Step 4': HMDのセット（Provost の場合）

```
1. ボウルにアルミホイルをピンと張る
2. フォイルポーカーで穴を開ける（外周に沿って均等に、中心にも数個）
3. Provostをアルミホイルの上に置く
4. 着火済みの炭を2個、Provostの中に配置
5. 蓋を閉じる or 半開きで火力調整
6. 3〜5分蒸らす
```

#### Step 5: 最終チェック・提供

```
1. 蒸らし後、軽く吸ってみて煙が出ることを確認
2. 味・煙量が適正か確認
3. マウスピースを装着
4. お客様に提供
```

## 3. セッション中のメンテナンス

### 炭交換のタイミング

| サイン | 対応 |
|--------|------|
| 炭が半分以上灰になっている | 新しい炭に交換 |
| 煙の量が明らかに減った | 炭交換 or HMDの蓋を調整 |
| 味が薄くなってきた | まず蓋を閉じて火力UP → 改善しなければ炭交換 |
| 約30〜45分経過 | 定期交換のタイミング |

### 炭交換の手順

```
1. トングで古い炭をHMDから取り出し、灰皿に置く
2. 電熱コンロで着火済みの新しい炭をトングで掴む
3. HMDに新しい炭を配置
4. 蓋を調整（新しい炭は火力が強いので、最初は蓋を開け気味に）
5. 1〜2分後、煙量を確認して蓋を調整
```

### 火力調整のガイド

| 状況 | HMDの調整 |
|------|----------|
| 煙が少ない・味が薄い | 蓋を閉じる（火力UP） |
| 味が焦げっぽい・苦い | 蓋を開ける or 炭を1つ減らす（火力DOWN） |
| ちょうど良い | 現状維持。蓋は半開き程度 |

### セッション中の巡回チェックリスト（10〜15分ごと）

- [ ] 炭の状態（灰の量、位置ずれ）
- [ ] 煙量は十分か
- [ ] ボトルの水の色（茶色く濁りすぎていないか）
- [ ] お客様の様子（気分が悪そうでないか）
- [ ] 灰が床やテーブルに落ちていないか
- [ ] 換気の状態（一酸化炭素検知器の確認）

## 4. 片付け・清掃の手順

### セッション直後（所要時間: 約20〜30分）

#### (A) 炭の処理

```
1. HMDからすべての炭を取り出す
2. 耐熱の灰皿または金属バケツに入れる
3. 完全に冷めるまで放置（最低30分）
4. 冷めた灰は不燃ごみとして廃棄
```

> **絶対にNG**: 燃えている/熱い炭を直接ゴミ箱に入れない。火災の原因。

#### (B) シーシャ本体の分解・洗浄

```
1. ボウルを取り外す → 使用済みフレーバーをゴミ箱に捨てる
2. HMDを取り外す → 冷めてから水洗い
3. ホースを取り外す
4. ステムをボトルから外す
5. ボトルの水を捨てる

洗浄:
6. ボトル: 水とボトルブラシで内部を洗浄。週1回は中性洗剤で洗浄
7. ステム: パイプクリーナーまたは長いブラシで内部を通す
8. ホース: 水を流して内部を洗浄。吊るして乾燥
9. ボウル: フレーバーの残りをしっかり洗い流す。焦げがあれば浸け置き
10. HMD: 焦げ付きをトングやブラシで落とす。水洗い
11. すべてのパーツを乾燥させる
```

#### (C) 空間の清掃

| エリア | 清掃内容 |
|--------|---------|
| テーブル | 灰・水滴の拭き取り |
| 床 | 掃除機 + ウェットシートで拭き掃除 |
| ソファ・クッション | ファブリックミストで消臭 |
| 換気 | 窓を開けて換気。空気清浄機を最大運転 |
| マウスピース | 使用済みを廃棄 |
| ドリンク類 | グラスの回収・洗浄 |

#### (D) 次のセッション準備

- 機材の在庫確認（炭の残量、フレーバーの残量）
- 使い捨てマウスピースの補充
- 電熱コンロのコイル状態チェック

## 5. トラブル対応

### よくあるトラブルと対処法

#### 煙が出ない / 煙が少ない

| 原因 | 対処法 |
|------|--------|
| 炭の火力不足 | 炭が完全に赤くなっているか確認。黒い部分があれば再加熱 |
| 蒸らし時間不足 | HMDの蓋を閉じて3〜5分待つ |
| フレーバーの盛りが少ない | フレーバーを追加盛りする |
| フレーバーが乾燥している | 新しいフレーバーに交換 |
| エアリーク | グロメットの密着を確認。ホース接続部・ボウル接続部をチェック |
| 水が多すぎる | ボトルの水を少し減らす |

#### 味がしない / 味が薄い

| 原因 | 対処法 |
|------|--------|
| 火力不足 | 蓋を閉じて火力UP、または炭を追加 |
| フレーバーが燃え尽きた | フレーバーを新しく盛り直す（セッション60分以上経過時） |
| フレーバーが古い | 在庫の鮮度を確認。開封から3ヶ月以上のものは味が落ちる |
| 水が汚れている | ボトルの水を交換 |

#### 味が焦げている / 苦い

| 原因 | 対処法 |
|------|--------|
| 火力が強すぎる | HMDの蓋を開ける。炭を1つ減らす |
| フレーバーがHMDに直接触れている | ボウルを確認し、フレーバーの高さを調整 |
| 炭を置いてすぐ吸った | 蒸らし時間を取る（3〜5分） |
| フレーバーの盛り方が悪い | 盛り直す。ふわっと盛ることを意識 |

#### 吸いが重い

| 原因 | 対処法 |
|------|--------|
| 水が多い | ボトルの水を減らす |
| ホースが詰まっている | ホースを交換または水洗い |
| ボウルの穴が詰まっている | ボウルを確認。フレーバーで穴が塞がっていないか |
| グロメットがきつすぎる | グロメットを調整 |

#### お客様の体調不良（ニコチン酔い）

| 症状 | 対処法 |
|------|--------|
| 頭痛・めまい | すぐにシーシャを中止。水やお茶を飲んでもらう |
| 吐き気 | 横になってもらう。窓を開けて換気 |
| 顔面蒼白 | 上記に加え、甘い飲み物（ジュース等）を提供 |
| 症状が改善しない | 医療機関への相談を勧める |

**予防策**:
- 空腹時のシーシャは避けるよう事前に案内
- 初心者にはニコチンの軽いフレーバーを提供
- こまめに水分補給を促す
- 連続で吸い続けないよう声かけ

#### 一酸化炭素（CO）関連

| 状況 | 対処法 |
|------|--------|
| CO検知器がアラーム | 即座に窓を全開。シーシャを中止。お客様を換気の良い場所に |
| 部屋が煙たい | 換気扇を最大に。窓を開ける |

**予防策**:
- CO検知器を常時設置・稼働
- 換気扇を常に回す
- 1時間に1回は窓を開けて空気を入れ替える
- 狭い密室での長時間使用を避ける
