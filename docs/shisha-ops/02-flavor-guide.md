## 3. ミックスレシピの基本

### ミックスの3つの鉄則

1. **味の相性を考える**: 同じ系統のフレーバー同士は基本的に相性が良い（柑橘系×柑橘系、ベリー系×ベリー系）
2. **配合比率を守る**: メイン:サブ = 7:3 または 6:4 が基本。3種類以上混ぜると個々の味が埋もれやすい
3. **ミントをアクセントに**: ミントは万能なアクセント。10〜20%加えると全体が引き締まる

### 定番ミックスレシピ

| レシピ名 | 構成 | 比率 | 系統 | 難易度 |
|---------|------|------|------|--------|
| **レモネード** | レモン + ミント | 7:3 | さっぱり | ★☆☆ |
| **トロピカルパンチ** | マンゴー + パイナップル + ミント | 5:3:2 | フルーティ | ★★☆ |
| **ベリースムージー** | ブルーベリー + ストロベリー | 5:5 | ベリー | ★☆☆ |
| **メロンクリームソーダ** | メロン + バニラ + ミント | 5:3:2 | スイーツ | ★★☆ |
| **チョコストロベリー** | チョコレート + ストロベリー | 6:4 | デザート | ★☆☆ |
| **ピーチティー** | ピーチ + シナモン | 8:2 | 大人系 | ★★☆ |
| **洋ナシモヒート** | 洋ナシ + ライム + ミント | 5:3:2 | さっぱり | ★★☆ |
| **パッションフルーツ** | パッション + グァバ + ミント | 5:3:2 | トロピカル | ★★★ |

### ミックスのNG例

- **スパイス系 + フルーツ系**: 味がケンカしやすい（ダブルアップル + レモン等）
- **4種類以上のミックス**: 味がぼやける。最大3種類まで
- **強いミント + 繊細なフレーバー**: ミントが全てを打ち消してしまう

## 4. 保存方法と賞味期限

### 保存の基本

| 項目 | 推奨 |
|------|------|
| **保存容器** | 密閉できるジップロックまたはタッパー。開封後は元のパッケージから移し替え推奨 |
| **保存場所** | 直射日光を避け、涼しく乾燥した場所。冷蔵庫は不要（結露でカビの原因に） |
| **保存温度** | 常温（15〜25℃） |
| **賞味期限** | 未開封: 製造から約2年 / 開封後: 3〜6ヶ月が目安 |

### 劣化のサイン

- シロップが分離して底に溜まっている（軽度 → よく混ぜれば使える）
- 乾燥してパサパサになっている（重度 → 煙が少なく味が薄い。廃棄推奨）
- カビが生えている（即廃棄）
- 香りが明らかに劣化している（廃棄推奨）

### 保存のコツ

- 開封したらすぐにジップロックに入れ、空気を抜いて密封
- 複数フレーバーを同じ場所に保管する際は、匂い移りに注意
- 使う分だけ取り出し、残りはすぐに密封
- 在庫回転が遅いフレーバーは50gパックで仕入れる

### 仕入れの考え方

| 戦略 | 詳細 |
|------|------|
| **定番は大容量で** | Al Fakherのダブルアップル、グレープ、ミントなど回転率の高いフレーバーは250gまたは1kgで仕入れてコスト削減 |
| **新フレーバーは小容量で** | 初めて取り扱うフレーバーは50gまたは100gでテスト。お客さんの反応を見てから追加発注 |
| **卸売価格の交渉** | CyberChillやC.S.B Onlineに店舗である旨を伝え、卸売価格を交渉。まとめ買いで10〜20%程度の割引が期待できる |
| **季節に応じた仕入れ** | 夏: ミント・柑橘系を多め / 冬: スパイス・スイーツ系を多め |
