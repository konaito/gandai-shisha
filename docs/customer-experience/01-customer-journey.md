# カスタマージャーニーマップ — BiPシーシャ

> 岩手大学生向け貸切シーシャスペースにおける、認知からリピートまでの顧客体験設計

---

## 1. ジャーニー全体像

| フェーズ | 顧客の状態 | 主なタッチポイント | KPI |
|---------|-----------|------------------|-----|
| 認知 | 「シーシャって何？」「盛岡にあるの？」 | Instagram, TikTok, 友人の口コミ | リーチ数・フォロワー増加率 |
| 興味 | 「行ってみたいかも」 | 投稿の保存・シェア, プロフィール閲覧 | 保存率・プロフィール遷移率 |
| 検討 | 「いつ・誰と行く？いくら？」 | 公式サイト, LINE, Googleマップ | サイト訪問数・LINE登録数 |
| 予約 | 「予約しよう」 | LINE予約・DM | 予約完了率 |
| 来店 | 「ここであってる？」「ドキドキ」 | 道案内・入口の雰囲気 | 到着率（ノーショー率の逆） |
| 体験 | 「楽しい！映える！」 | シーシャ体験・空間・接客 | 滞在時間・満足度 |
| 退店 | 「また来たいな」 | 会計・次回特典の案内 | 次回予約率 |
| 共有 | 「友達にも教えたい」 | SNS投稿・ストーリーズ | UGC投稿数・メンション数 |
| リピート | 「次は別のフレーバー試したい」 | LINE配信・限定メニュー告知 | 2回目来店率・LTV |

## 2. 各フェーズの詳細設計

### 2-1. 認知フェーズ

**Z世代の特徴**: 大学生はInstagramとTikTokを「検索エンジン」として使う。テキストよりビジュアルで情報を取得する。

| アクション | 具体的な施策 |
|-----------|------------|
| Instagram運用 | 煙の映えるリール動画を週2-3本投稿。ハッシュタグ `#盛岡シーシャ` `#岩大` `#盛岡カフェ` |
| TikTok | 「シーシャ初心者がはじめて吸ってみた」系の短尺動画。リアクション重視 |
| 友人の口コミ | 来店者にストーリーズ投稿を促す仕組み（後述） |
| 学内露出 | 岩大の学食・掲示板周辺でのチラシ、サークル勧誘時期に合わせた告知 |

**ポイント**: 大学生の新規認知の9割はSNS経由。Googleマップの口コミも初期段階で3-5件は確保しておく。

### 2-2. 興味フェーズ

**Z世代の特徴**: 「失敗したくない」心理が強い。事前に口コミ・写真を入念にチェックする。

| アクション | 具体的な施策 |
|-----------|------------|
| 投稿の質 | 空間の雰囲気が伝わる写真・動画。「実際に行った人の投稿」が最も信頼される |
| 料金の明示 | 投稿やプロフィールに料金を明記。大学生は価格感度が高い |
| 初心者歓迎の打ち出し | 「シーシャ初めてでもOK」を前面に。ハードルを下げるコピー |
| ハイライトの整備 | Instagramハイライトに「料金」「アクセス」「Q&A」「お客様の声」を常設 |

### 2-3. 検討フェーズ

**大学生の意思決定プロセス**:
1. 誰かが「行きたい」と言い出す（発起人）
2. グループLINEで共有・相談
3. 日程調整（大学生は授業・バイトで予定が流動的）
4. 料金を割り勘で計算
5. 予約

| アクション | 具体的な施策 |
|-----------|------------|
| 割り勘計算の明示 | 「4人で来たら1人○○円」のように人数別料金を表示 |
| 予約の簡便さ | LINEで「日時・人数」を送るだけで予約完了にする |
| FAQ整備 | 「喫煙者じゃなくても大丈夫？」「何を持っていけばいい？」等の不安を事前に解消 |
| Googleマップ最適化 | 写真・営業時間・料金帯を正確に記載。口コミへの返信も丁寧に |

### 2-4. 予約フェーズ

| アクション | 具体的な施策 |
|-----------|------------|
| LINE予約 | 公式LINEから日時・人数を送信→確認返信の2ステップで完了 |
| リマインド | 前日にLINEでリマインドメッセージ（道順も添付） |
| キャンセルポリシー | 当日キャンセルのみペナルティ。学生は予定が変わりやすいので柔軟に |
| 空き状況の可視化 | LINEリッチメニューまたはサイトで空き枠を確認できるようにする |

### 2-5. 来店フェーズ

**初めての来店は最も不安が大きいタイミング**

| アクション | 具体的な施策 |
|-----------|------------|
| 道案内 | 予約確認時にGoogleマップのピン+写真付き道順を送付 |
| 入口の分かりやすさ | 看板 or 目印を設置。暗い時間帯は特に重要 |
| 到着時の声かけ | 「予約の○○さんですね、ようこそ！」で安心感を |
| 靴を脱ぐ等のルール | 事前にLINEで伝えておくと当日スムーズ |

### 2-6. 体験フェーズ

| アクション | 具体的な施策 |
|-----------|------------|
| シーシャの説明 | 初心者には吸い方・フレーバーを丁寧に説明（→サービスマニュアル参照） |
| 空間の演出 | 照明・音楽・香りで非日常感を演出（→空間演出ガイド参照） |
| 写真撮影の促進 | 映えるスポット・小物を用意。「撮って！」と言わなくても撮りたくなる空間 |
| 炭替えのタイミング | 15-20分ごとの炭替えで自然にコミュニケーション。味の感想を聞く |
| ドリンク・軽食 | 持ち込みOKにするか、簡単なドリンクを提供するか（原価管理と相談） |

### 2-7. 退店フェーズ

| アクション | 具体的な施策 |
|-----------|------------|
| 会計 | PayPay等のキャッシュレス対応。割り勘しやすい決済手段 |
| 次回特典の案内 | 「次回使える○○円OFFクーポン」をその場で渡す |
| LINE登録の促進 | まだ登録していない同行者にもLINE登録を案内 |
| 感想のヒアリング | 「今日どうでした？」と軽く聞く。改善のヒントになる |
| 口コミ依頼 | Googleマップの口コミを丁寧にお願い（QRコードを用意） |

### 2-8. 共有フェーズ

| アクション | 具体的な施策 |
|-----------|------------|
| ストーリーズ投稿の促進 | 店内にInstagramネームプレートを設置。「タグ付けで次回○○サービス」 |
| 写真素材の提供 | 店が撮った写真をAirDropやQRで共有 |
| リポスト | 顧客のストーリーズをリポストして感謝を表現 |

### 2-9. リピートフェーズ

| アクション | 具体的な施策 |
|-----------|------------|
| LINE配信 | 新フレーバー入荷・空き枠情報を週1で配信（頻度注意） |
| 限定感 | 「今月限定フレーバー」「雨の日割引」等で再訪動機を作る |
| 紹介制度 | 友達を連れてくると両者に特典（→リピーター施策参照） |

## 3. 大学生の意思決定における重要ポイント

### グループ意思決定の構造
- **発起人（1人）** が情報を発見し、グループLINEに共有
- **同調圧力** で参加が決まる（「みんな行くなら行く」）
- → 発起人になりやすい情報発信（シェアしやすいコンテンツ）が重要

### 価格感度
- 大学生の飲み会予算は1回3,000-4,000円が目安
- 割り勘前提で「1人あたりいくらか」が最重要情報
- 「飲み会1回分でシーシャ体験できる」という比較訴求が有効

### 時間的制約
- 授業・バイト・サークルで予定が不規則
- 平日夜（18:00-22:00）と週末が主な利用時間帯
- 予約変更のしやすさが重要

### SNS行動
- 体験を「記録」としてストーリーズに投稿する習慣がある
- 投稿のハードルは低い（特にストーリーズは24時間で消える安心感）
- ただしフィード投稿は「本当に良かった」ものだけ → 空間の質が問われる

| フェーズ | SNS施策 | 期待効果 |
|---------|--------|---------|
| 認知 | リール・TikTokで煙の動画 | 拡散・新規リーチ |
| 興味 | ハイライトで情報整理 | 離脱防止 |
| 検討 | 投稿が「シェアしやすい」フォーマット | グループLINEへの共有促進 |
| 体験中 | 映えスポット・ネームプレート設置 | UGC生成 |
| 退店後 | タグ付け特典 | ストーリーズ投稿率向上 |
| リピート | LINE配信・限定情報 | 再訪動機の生成 |
