# リピーター施策 — BiPシーシャ

> 初回来店から常連化までの仕組みづくり。大学生コミュニティの特性を活かしたリテンション戦略。

---

## 1. リピーターの重要性

### 数字で見るリピーターの価値

| 指標 | 数値目安 |
|-----|---------|
| 新規顧客獲得コスト vs リピーター維持コスト | 5:1（新規は5倍コストがかかる） |
| 目標リピート率 | 初回→2回目: 30%以上 / 2回目→3回目: 50%以上 |
| リピーターの売上貢献 | 全体売上の60-70%が理想 |
| 口コミ経由の新規 | リピーターからの紹介が最もCVR（成約率）が高い |

### 大学生リピーターの特徴

- **コミュニティ単位で動く**: 1人がハマると友人グループ全体に波及
- **習慣化しやすい**: 「金曜の夜はシーシャ」のようなルーティンになる可能性
- **在学期間が限られる**: 4年で卒業するため、後輩への文化継承が重要
- **予算に制約がある**: 「また行きたいけど高い」が最大の離脱要因

## 2. 初回→2回目のハードルを下げる

2回目の来店が最も難しい。ここを突破すれば常連化の確率が大幅に上がる。

### 退店時の仕掛け

| 施策 | 内容 | 期待効果 |
|-----|------|---------|
| 次回クーポン | 退店時に「次回500円OFF」クーポンを手渡し | 再訪の金銭的動機 |
| 有効期限の設定 | クーポンの有効期限は**2週間以内**。短いほど行動を促す | 先延ばし防止 |
| 未体験フレーバーの紹介 | 「次来たら○○も試してみて！」と具体的なフレーバーを伝える | 体験の続き感 |
| LINE登録 | 同行者全員にLINE登録を案内。2回目予約のハードルが下がる | 直接リーチ可能に |

### 来店後のフォロー

| タイミング | アクション |
|-----------|----------|
| 当日夜 | ストーリーズをリポスト+お礼コメント |
| 翌日 | LINE「昨日はありがとう！クーポン期限○日までだから使ってね」 |
| 1週間後 | まだ予約がなければ「新フレーバー入ったよ」とLINE（押しすぎ注意） |

### 2回目来店を促す心理テクニック

- **ツァイガルニク効果**: 「まだ試してないフレーバーがある」→未完了タスクが気になる心理
- **サンクコスト**: LINE登録・クーポン取得で「もう関係がある」状態にする
- **社会的証明**: 「今月○人が2回目来てくれた」とストーリーズで発信

## 3. スタンプカード / ポイント制度

### 設計の基本方針

大学生向けなので**シンプル・わかりやすい・特典が魅力的**であること。

### おすすめ: LINEショップカード

紙のスタンプカードではなく、**LINE公式アカウントのショップカード機能**を活用。

| 項目 | 設計 |
|-----|------|
| スタンプ付与条件 | 来店1回につき1スタンプ |
| 必要スタンプ数 | 5個でコンプリート（多すぎると脱落する） |
| 中間特典（3個） | フレーバー1種追加無料 or ドリンク1杯無料 |
| コンプリート特典（5個） | 1回無料招待 or 半額チケット |
| リセット後 | 新しいカードが自動発行。継続利用を促進 |

### LINEショップカードのメリット

- 紙と違って忘れない・なくさない
- プッシュ通知で「あと○個」をリマインドできる
- 運営側でデータが見える（リピート率の計測）
- 導入コストゼロ（LINE公式アカウントの標準機能）

### 運用上の注意

- スタンプ付与は**会計時にその場で**行う（後からだと忘れる）
- 「あと1個でコンプリート！」の人にはLINEで積極的にリマインド
- 特典の原価管理を忘れない（無料招待のコストを事前に計算）

## 4. 紹介制度（友達割）

大学生の最強の集客チャネルは「友達の口コミ」。これを仕組み化する。

### 紹介プログラム設計

| 項目 | 内容 |
|-----|------|
| 名称 | 「ダチ割」（友達割引のカジュアル版） |
| 仕組み | 既存客が新規の友達を連れてきたら、**両者に特典** |
| 紹介者特典 | 次回500円OFF or フレーバー追加無料 |
| 被紹介者特典 | 初回500円OFF |
| 上限 | 紹介者の特典に上限なし（紹介するほどお得に） |

### 紹介を促進する仕組み

| 施策 | 具体的な方法 |
|-----|------------|
| 紹介カード | LINEで共有できる紹介用URL or QRコード |
| 紹介しやすいトーク | 「友達連れてきたら2人とも500円OFFだよ」と退店時に伝える |
| SNS紹介 | ストーリーズで店のアカウントをメンションしたら次回特典 |
| グループ割 | 5人以上のグループは1人あたり○○円OFF（新規を含むグループ向け） |

### 紹介連鎖の設計

```
Aさん（初回来店）
  ├─ 紹介 → Bさん（Aに特典、Bに初回割引）
  │         ├─ 紹介 → Cさん（Bに特典、Cに初回割引）
  │         └─ 紹介 → Dさん
  └─ 紹介 → Eさん

※ 大学の友人グループは5-10人規模。
   1人がハマると芋づる式に広がる。
```

## 5. 季節イベント・限定メニュー

「いつ行っても同じ」だと飽きる。季節感と限定性で再訪動機を作る。

### 年間イベントカレンダー

| 月 | イベント | 施策 |
|---|--------|------|
| 4月 | 新歓シーズン | 新歓グループプラン（5人以上で1人無料）。新入生向け初回割引 |
| 5月 | GW | GW限定フレーバー。長期休み中の昼営業 |
| 6月 | 梅雨 | 雨の日割引（来店ハードルが上がる時期の対策） |
| 7月 | 前期試験後 | 「打ち上げプラン」。テスト終わりの解放感に訴求 |
| 8月 | 夏休み | サマーフレーバー祭り。トロピカル系限定メニュー |
| 9月 | 後期開始 | 「おかえりプラン」。久しぶりの来店者にウェルカム特典 |
| 10月 | 学祭 | 学祭コラボ企画。学祭後のアフターパーティープラン |
| 11月 | 紅葉 | オータムフレーバー（アップル・シナモン系） |
| 12月 | クリスマス・忘年会 | クリスマス限定空間演出。忘年会プラン |
| 1月 | 新年 | 新年一発目割引。おみくじフレーバー（ランダム選択） |
| 2月 | バレンタイン | チョコ系フレーバー限定。カップルプラン |
| 3月 | 卒業・追いコン | 卒業お祝いプラン。4年生向けラストセッション割引 |

### 限定フレーバーの運用

| ポイント | 内容 |
|---------|------|
| 頻度 | 月1回のペースで新しい限定フレーバーを導入 |
| 告知 | LINEとInstagramで1週間前から予告。「限定」の文字を強調 |
| ネーミング | 遊び心のある名前をつける（例: 「盛岡の夏休み」「チルの極み」） |
| アーカイブ | 人気だった限定フレーバーは定番化を検討 |

## 6. コミュニティ形成（常連グループ化）

### 目指す状態

「BiPシーシャに行く」が岩手大学生の間で一つのカルチャーになること。

### コミュニティ施策

| 施策 | 内容 | 効果 |
|-----|------|------|
| 常連ニックネーム | 3回以上来店した人には「BiPメンバー」的な呼称 | 帰属意識 |
| 常連限定特典 | 5回以上来店で「推しフレーバー」を常備 | 特別感 |
| 常連同士の交流 | 「シーシャ好き交流会」を不定期開催 | コミュニティの横のつながり |
| 運営参加 | 常連にフレーバーの仕入れ選定に参加してもらう | 当事者意識 |
| SNSコミュニティ | Instagram非公開アカウントで常連限定の情報を共有 | 内輪感 |

### 文化継承の仕組み

大学生は4年で卒業するため、**先輩→後輩への文化継承**が長期存続の鍵。

| 施策 | 内容 |
|-----|------|
| 先輩×後輩プラン | 先輩が後輩を連れてきたら先輩は無料 |
| サークル連携 | サークルの打ち上げ会場として定着させる |
| 新歓利用 | 毎年4月に「新入生連れてきたら割引」キャンペーン |
| 卒業生ネットワーク | 卒業生が帰省時に来店できる「OB/OG割」 |

## 7. LINE公式アカウントの活用

リピーター施策の中心をLINE公式アカウントに据える。

### リッチメニュー設計

```
┌──────────┬──────────┬──────────┐
│  📅 予約  │ 💳 カード │ 🎁 クーポン │
├──────────┼──────────┼──────────┤
│  📋 メニュー │ 📍 アクセス│ 💬 お問合せ │
└──────────┴──────────┴──────────┘
```

### 配信スケジュール

| 頻度 | 内容 | 注意点 |
|-----|------|-------|
| 週1回 | 空き状況・新フレーバー情報 | 短く、画像中心 |
| 月1回 | 限定イベント告知 | 1週間前に予告 |
| 不定期 | 雨の日割引・急な空き枠告知 | リアルタイム性が価値 |

### 配信のNG

- 週2回以上の配信（ブロック率が上がる）
- テキストだけの長文（読まれない）
- 営業感の強い文面（「来てください！」は逆効果）

## 8. 料金設計とリピーターの関係

### リピーターが増える料金設計のポイント

| 原則 | 理由 |
|-----|------|
| 1人あたりの金額を明示 | 大学生は割り勘前提。「4人で来たら1人○○円」が最重要情報 |
| 回数券・プリペイドの導入 | 「5回分まとめて買うと1回無料」→ 先払いで来店確約 |
| 平日割引 | 平日の稼働率を上げつつ、予算に厳しい学生の味方に |
| 延長料金を明確に | 「30分○○円で延長できるよ」と事前に伝えて安心感 |

### 回数券の設計例

| プラン | 価格（例） | 1回あたり | お得率 |
|-------|----------|----------|-------|
| 単発 | 3,000円/人 | 3,000円 | - |
| 3回券 | 8,000円/人 | 2,667円 | 11% OFF |
| 5回券 | 12,000円/人 | 2,400円 | 20% OFF |

※ 回数券はLINE上でデジタル管理すると紛失リスクなし

## 9. 効果測定

### 追跡すべきKPI

| KPI | 計測方法 | 目標値 |
|-----|---------|-------|
| 2回目来店率 | LINE登録者の再予約率 | 30%以上 |
| 紹介経由の新規率 | 予約時に「紹介」を確認 | 新規の20%以上 |
| LINEブロック率 | LINE管理画面 | 10%以下 |
| 月間リピーター比率 | 予約台帳で計測 | 50%以上 |
| 客単価 | 売上/来店グループ数 | 維持 or 微増 |
| NPS（推奨度） | 退店時に「友達に勧める？10点満点で」 | 8点以上 |

### 月次レビュー

毎月末に以下を振り返る:
1. 新規 vs リピーターの比率
2. どの施策経由のリピートが多いか（クーポン / 紹介 / LINE配信）
3. 離脱しているのはどの段階か（1回で終わり？3回目で離脱？）
4. フレーバーの人気ランキング（仕入れの最適化）
5. クレーム・改善要望のまとめ

## 参考情報

- 飲食店のリピーター施策では「2回目来店で○○円引き」「3回目で○○サービス」が初期段階で効果的
- デジタルスタンプカード（LINE等）は紙に比べて利用率が高く、運営コストも低い
- 大学生の集客はSNSでの友人の投稿（リア充写真）が匿名口コミより強く響く
- LINE公式アカウントの配信頻度は週1回が最適。それ以上はブロック率が急増する
