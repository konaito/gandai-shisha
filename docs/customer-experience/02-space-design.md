# 空間演出ガイド — BiPシーシャ

> 貸切シーシャスペースの空間コンセプト設計。低コストで非日常感を演出し、SNS投稿を自然に誘発する空間づくり。

---

## 1. 空間コンセプト

### コンセプトの方向性

貸切シーシャスペースは「友達の秘密基地」がコアコンセプト。大学生にとって、居酒屋でもカラオケでもない「第三の居場所」を提供する。

| 要素 | 方向性 | 避けるべき方向 |
|-----|-------|-------------|
| 雰囲気 | カジュアルだけど特別感がある | 高級すぎて緊張する / 安っぽくてガッカリ |
| テイスト | モロッカン / ボヘミアン / チル系 | 無機質なオフィス風 / 生活感のある自宅風 |
| ターゲット感覚 | 「ここ知ってる自分、センスいい」 | 「誰でも知ってる」チェーン店感 |

### コンセプト案: 「チルの隠れ家」

- 外からは分かりにくい（隠れ家感）
- 中に入ると非日常空間が広がる（ギャップ）
- 友達だけの貸切空間で気兼ねなくリラックス
- 「ここ、めっちゃ良くない？」とストーリーズを撮りたくなる

## 2. 照明設計

照明は空間演出の8割を決める最重要要素。低コストで最も効果が大きい。

### 基本原則

- **天井の蛍光灯は絶対に使わない**（一瞬で生活感が出る）
- **暖色系（2700K以下）** で統一
- **間接照明中心** で陰影をつくる
- **調光可能** にして時間帯や雰囲気に合わせる

### 具体的な照明アイテム（低コスト順）

| アイテム | 予算目安 | 効果 | 備考 |
|---------|---------|-----|------|
| LEDテープライト（暖色） | 1,000-2,000円 | ★★★★★ | 棚裏・天井際に設置。コスパ最強 |
| フェアリーライト（ワイヤーライト） | 500-1,500円 | ★★★★ | 壁に這わせる・瓶に入れる |
| キャンドル型LEDライト | 1,000-2,000円 | ★★★★ | テーブルに置くだけで雰囲気UP |
| フロアランプ（間接照明） | 3,000-5,000円 | ★★★★ | コーナーに1台で空間が締まる |
| ネオンサイン（小型） | 2,000-5,000円 | ★★★★★ | 写真映え最強。店名やロゴで |
| プロジェクター投影 | 10,000-20,000円 | ★★★ | 壁に映像を映す。映画上映にも使える |

## 3. 音楽・BGM設計

### 基本原則

- **会話の邪魔にならない音量**（貸切なので友達との会話が主目的）
- **ジャンルはLofi / チルホップ / アンビエント** が鉄板
- Bluetooth接続可能にして**お客様が自分の音楽を流せる**ようにするのも差別化ポイント

### プレイリスト例

| シーン | おすすめジャンル | Spotifyキーワード |
|-------|--------------|----------------|
| 通常営業 | Lofi Hip Hop / Chillhop | 「lofi chill」「cafe jazz」 |
| パーティー寄り | R&B / City Pop | 「japanese city pop」 |
| 落ち着いた雰囲気 | アンビエント / ジャズ | 「ambient chill」「jazz lounge」 |
| 季節感（冬） | 暖炉の音 + ジャズ | 「winter jazz cozy」 |

### 音響設備（低コスト）

| 機材 | 予算目安 | 備考 |
|-----|---------|-----|
| Bluetoothスピーカー（JBL Flip等） | 5,000-10,000円 | 1台で十分。防水だと安心 |
| Echo Dot（Alexa） | 3,000-5,000円 | 「アレクサ、Lofi流して」で簡単操作 |

## 4. 香り・嗅覚の演出

### シーシャスペース特有の考慮点

シーシャ自体がフレーバーの香りを出すため、**ベースの空間の香りはシーシャと喧嘩しないもの**を選ぶ。

| タイミング | 香りの方針 | おすすめ |
|-----------|----------|---------|
| 来店前（入口） | 清潔感のある香り | 柑橘系アロマ / お香（控えめ） |
| シーシャ準備中 | ほのかなベース香 | ホワイトムスク系ディフューザー |
| シーシャ中 | フレーバーに任せる | 追加の香りは不要 |
| 退店後の清掃 | 換気+消臭 | 重曹スプレー / 空気清浄機 |

### 換気設備（必須）

シーシャは煙が出るため、換気は空間設計の最重要インフラ。

- **換気扇**: 必須。天井埋込型 or 壁付型
- **空気清浄機**: 大型1台（シャープ等）で常時稼働
- **窓の開閉**: 可能であれば窓を確保。自然換気が最も効果的
- **煙の流れ**: シーシャ台の近くに排気口を配置

## 5. 写真映えする空間づくり（SNS投稿誘発）

### 「撮りたくなる」仕掛け

大学生は「撮って」と言われなくても良い空間は自然に撮影する。重要なのは**撮影したくなるポイントを意図的に配置する**こと。

| 仕掛け | コスト | 効果 | 具体例 |
|-------|------|-----|-------|
| ネオンサイン | 2,000-5,000円 | ★★★★★ | 店名 or キャッチーなフレーズ |
| アクセントウォール | 1,000-3,000円 | ★★★★ | 1面だけ色を変える / タペストリー |
| シーシャ本体のデザイン | - | ★★★★ | ガラス製のカラフルなボトルは映える |
| 小物・ドライフラワー | 1,000-3,000円 | ★★★ | テーブル周りの装飾 |
| ミラーボール（小型） | 1,000-2,000円 | ★★★ | 光の反射で幻想的に |
| 手書きメニューボード | 500-1,000円 | ★★★ | 黒板 + チョークアート |

### 撮影アングルの設計

- **「座った状態で撮れる」** アングルを意識する（立ち上がって撮影は面倒）
- テーブル上のシーシャ + ドリンク + 照明が1枚に収まる配置
- 背景にネオンサインやアクセントウォールが映り込む角度

### SNS投稿を促す仕組み

- Instagramのネームプレート（QRコード）をテーブルに設置
- 「ストーリーズにタグ付けで次回フレーバー1種追加無料」等の特典
- 店側が写真を撮ってあげる → AirDropで送る（自分では撮りにくい全体写真）

## 6. 低コスト内装のコツ

### 予算別の優先順位

物件の状態にもよるが、限られた予算で最大効果を出す優先順位。

| 優先度 | 項目 | 予算目安 | 理由 |
|-------|------|---------|-----|
| 1位 | 照明の変更 | 5,000-15,000円 | 最もコスパが高い。蛍光灯→間接照明で別空間に |
| 2位 | 布・ファブリック | 5,000-10,000円 | クッション・ラグ・カーテンで質感UP |
| 3位 | 壁面の装飾 | 3,000-8,000円 | タペストリー・ポスター・ネオンサインで個性を出す |
| 4位 | テーブル・座席 | 5,000-20,000円 | ローテーブル+クッション or ソファ。中古活用 |
| 5位 | 小物・植物 | 2,000-5,000円 | フェイクグリーン、キャンドル、本棚等 |

### コストを抑えるテクニック

- **中古・リサイクル活用**: メルカリ、ジモティーで家具を調達
- **DIY**: 棚板やテーブルは2x4材+塗装で自作可能
- **100均活用**: フェアリーライト、キャンドル、フェイクグリーンは100均で十分
- **布で隠す**: 生活感のある部分（配線、収納）は布で隠すだけで見違える
- **壁紙シート**: 賃貸でも貼って剥がせる壁紙シートで1面だけ変える
- **植物**: フェイクグリーンを要所に。水やり不要でコスパ良し

### 座席レイアウト

貸切スペースなので、**全員が輪になれる配置**が基本。

```
推奨レイアウト（4-6人想定）:

       クッション    クッション
          ┌──────────┐
          │ ロー       │
クッション │ テーブル    │ クッション
          │  🔵シーシャ │
          └──────────┘
       クッション    クッション

※ 床座り（ラグ+クッション）が最もコスパが良く、
  シーシャとの相性も良い（低い姿勢でリラックス）
```

## 8. 空間づくりチェックリスト

開業前・改善時に確認するリスト。

- [ ] 蛍光灯を使っていないか
- [ ] 暖色系の照明で統一されているか
- [ ] 写真映えポイントが2箇所以上あるか
- [ ] BGMが流れているか（音量は会話の邪魔にならないか）
- [ ] 換気設備は十分か
- [ ] 清潔感があるか（最も基本かつ最重要）
- [ ] 座り心地は良いか（2時間以上座ることを想定）
- [ ] 荷物を置く場所があるか
- [ ] コンセント・充電スポットがあるか（大学生には重要）
- [ ] 室温は適切か（シーシャ+人数で暑くなりやすい）
- [ ] Instagramネームプレートが見える位置にあるか
