# 日常衛生チェックリスト

> BiPシーシャ 衛生管理ドキュメント
> 最終更新: 2026-03-21

---

## 概要

本ドキュメントは、BiPシーシャの1人運営における日常的な衛生管理チェックリストである。
保健所の検査にも対応できるよう、チェック項目を明確化し、記録を残す運用を行う。

関連ドキュメント:
- `01-mouthpiece-hygiene.md` — マウスピース衛生管理
- `02-equipment-cleaning.md` — 機材清掃マニュアル
- `03-ventilation-co.md` — 換気と一酸化炭素対策

## 1. 営業前チェック項目

予約のお客様を迎える前に、以下を確認する。

### 空間・環境

| # | チェック項目 | 判定 |
|---|-------------|------|
| 1 | 室内の換気を開始した（換気扇ON / 窓開け） | |
| 2 | CO検知器が正常に動作している（数値表示を確認） | |
| 3 | 室内CO濃度が10ppm以下であることを確認 | |
| 4 | 室温・湿度が適切である（エアコン・除湿器を稼働） | |
| 5 | テーブル・椅子・ソファを消毒用エタノールで拭き取り済み | |
| 6 | 床にゴミ・灰が落ちていないことを確認 | |
| 7 | トイレが清潔であることを確認 | |
| 8 | ゴミ箱を空にした / 十分な容量がある | |

### シーシャ機材

| # | チェック項目 | 判定 |
|---|-------------|------|
| 9 | シーシャ本体が洗浄・乾燥済みであることを確認 | |
| 10 | ベースに清潔な水を入れた（適切な水位） | |
| 11 | ホースが洗浄・乾燥済みであることを確認 | |
| 12 | 使い捨てマウスピースの在庫が十分にある（最低10個以上） | |
| 13 | ボウル・HMDが洗浄済みであることを確認 | |
| 14 | グロメット（パッキン）の状態を確認（ひび割れ・劣化なし） | |
| 15 | 炭の在庫が十分にある | |

### 衛生用品

| # | チェック項目 | 判定 |
|---|-------------|------|
| 16 | 消毒用エタノールのスプレーボトルが補充されている | |
| 17 | ペーパータオル / ウェットティッシュの在庫確認 | |
| 18 | ゴム手袋の在庫確認 | |
| 19 | 手洗い用石鹸の残量確認 | |

## 2. 営業中チェック項目

セッション中に継続的に確認する事項。

### 安全管理

| # | チェック項目 | 頻度 |
|---|-------------|------|
| 1 | CO検知器の数値を確認（10ppm以下を維持） | **15分ごと** |
| 2 | 換気扇が稼働していることを確認 | セッション開始時 |
| 3 | 炭の状態を確認（適切に燃焼しているか） | 15〜20分ごと |
| 4 | お客様の体調に異変がないか確認 | 随時 |
| 5 | 自分自身の体調（頭痛・めまい等の有無） | 随時 |

### 衛生管理

| # | チェック項目 | 頻度 |
|---|-------------|------|
| 6 | お客様ごとに新しいマウスピースを提供した | お客様到着時 |
| 7 | マウスピースをお客様の目の前で開封した | お客様到着時 |
| 8 | 灰や炭のカスが散らばっていないか | 炭交換時 |
| 9 | ドリンク提供時の手指衛生 | ドリンク提供前 |

### セッション間（お客様入れ替え時）

| # | チェック項目 | 判定 |
|---|-------------|------|
| 10 | 使用済みマウスピースを廃棄した | |
| 11 | ボウルの残渣を除去し水洗いした | |
| 12 | ベースの水を交換した | |
| 13 | ホースに水を通した | |
| 14 | テーブル・灰皿を清掃した | |
| 15 | 室内を換気した（CO濃度が10ppm以下に低下するまで） | |
| 16 | 新しいお客様用のマウスピースを準備した | |

## 3. 営業後チェック項目

最後のお客様が退店した後に実施する。

### シーシャ機材の清掃

| # | チェック項目 | 判定 |
|---|-------------|------|
| 1 | ボウルを洗剤で洗浄し乾燥させた | |
| 2 | HMDの灰・焦げを除去し洗浄した | |
| 3 | ステムをブラシで洗浄した | |
| 4 | ベースの水を捨て、洗剤で洗浄し乾燥させた | |
| 5 | ホースを水通し・洗剤洗浄し、吊るして乾燥 | |
| 6 | 使用済みの炭を完全に消火した（炭消し缶に入れた） | |
| 7 | 灰を適切に処分した | |

### 空間の清掃

| # | チェック項目 | 判定 |
|---|-------------|------|
| 8 | テーブル・椅子をエタノールで拭き取り消毒 | |
| 9 | 床を掃除した（掃除機 or ほうき） | |
| 10 | ゴミを回収・分別した | |
| 11 | トイレを清掃した | |
| 12 | 手洗い場を清掃した | |

### 設備の確認

| # | チェック項目 | 判定 |
|---|-------------|------|
| 13 | 換気扇を停止した（または常時運転の場合は稼働確認） | |
| 14 | エアコン・除湿器を適切な設定にした | |
| 15 | CO検知器の最終数値を記録した | |
| 16 | 炭バーナー・コンロの電源をOFFにした | |
| 17 | 消火器の位置を確認（移動していないか） | |

### 在庫確認

| # | チェック項目 | 判定 |
|---|-------------|------|
| 18 | マウスピースの残数を記録（不足なら発注） | |
| 19 | フレーバーの残量を確認 | |
| 20 | 炭の残量を確認 | |
| 21 | 消毒液・洗剤の残量を確認 | |

## 4. 週次・月次の定期チェック

### 週次チェック（毎週月曜日推奨）

| # | チェック項目 | 判定 |
|---|-------------|------|
| 1 | ベースのハイター漬け置き消毒を実施 | |
| 2 | HMDのクエン酸浸漬を実施 | |
| 3 | ホースの状態を詳細に確認（臭い・変色・硬化） | |
| 4 | グロメット（パッキン）の劣化チェック | |
| 5 | CO検知器のテストボタンで動作確認 | |
| 6 | 換気扇フィルターの汚れを確認（必要に応じて清掃） | |
| 7 | 消火器の外観確認（圧力ゲージ・ピン） | |
| 8 | 衛生チェック記録簿のレビュー（記入漏れがないか） | |

### 月次チェック（毎月1日推奨）

| # | チェック項目 | 判定 |
|---|-------------|------|
| 1 | ホースの交換判断（使用開始からの経過日数を確認） | |
| 2 | ボウルの煮沸消毒（シリコン製） | |
| 3 | ステムの分解洗浄 | |
| 4 | CO検知器の電池残量確認 | |
| 5 | 換気扇フィルターの交換判断 | |
| 6 | 衛生用品の月間使用量を記録（コスト管理） | |
| 7 | 消耗品（マウスピース・炭・フレーバー等）の月間発注計画 | |
| 8 | 保健所提出用の衛生管理記録の整理 | |

### 半年次チェック（3月・9月推奨）

| # | チェック項目 | 判定 |
|---|-------------|------|
| 1 | CO検知器の電池交換 | |
| 2 | 換気扇の清掃・メンテナンス | |
| 3 | シーシャ本体全パーツの総点検 | |
| 4 | 衛生管理マニュアルの見直し・更新 | |
| 5 | 消火器の有効期限確認 | |

### 年次チェック（開業月推奨）

| # | チェック項目 | 判定 |
|---|-------------|------|
| 1 | CO検知器のセンサー寿命確認（5〜7年で交換） | |
| 2 | 換気設備の専門業者点検 | |
| 3 | 消火器の交換判断（製造年から5年で詰替えまたは交換） | |
| 4 | 保健所への定期報告（必要に応じて） | |
| 5 | 衛生管理体制全体のレビュー | |

## 6. 記録の保管ルール

| 項目 | ルール |
|------|--------|
| **日次チェックシート** | 最低1年間保管 |
| **週次・月次チェック記録** | 最低2年間保管 |
| **CO値の異常記録** | 永年保管（インシデントレポートとして） |
| **保管形式** | 紙（ファイリング）+ デジタル（写真撮影してクラウド保存）の二重保管推奨 |
| **保健所提出用** | 直近1年分をいつでも提示できる状態にしておく |

## 7. 保健所対応メモ

保健所の検査・立ち入り時に提示するもの:

- [ ] 本チェックリストの運用実績（直近1年分の日次チェックシート）
- [ ] CO検知器の設置状況と点検記録
- [ ] 換気設備の仕様書と設置図面
- [ ] 機材清掃マニュアル（`02-equipment-cleaning.md`）
- [ ] マウスピースの衛生管理方針（`01-mouthpiece-hygiene.md`）
- [ ] 使用する洗剤・消毒液の一覧とSDS
- [ ] 食品衛生責任者の資格証
- [ ] 緊急対応フロー（`03-ventilation-co.md`）
