## 1. 使い捨てマウスピース vs 洗浄再利用

| 項目 | 使い捨て（ディスポーザブル） | 洗浄再利用 |
|------|----------------------------|------------|
| **衛生面** | 毎回新品。感染リスクほぼゼロ | 洗浄・消毒の精度に依存 |
| **運用負荷** | 交換するだけ（数秒） | 洗浄→消毒→乾燥で10〜15分/本 |
| **コスト** | 約9〜15円/個（後述） | 初期費用1,000〜3,000円/本 + 消毒液代 |
| **ゴミ** | プラスチック廃棄が出る | 少ない |
| **お客様の印象** | 個包装を目の前で開封 → 安心感が高い | 「本当に清潔？」という不安が残りやすい |

### 推奨方針

**貸切シーシャスペースでは「使い捨てマウスピース」を標準運用とする。**

理由:
- 1人運営のため、洗浄再利用のオペレーション負荷が大きい
- 保健所の衛生検査において「使い捨て＋個包装」は最も説明しやすい
- お客様（大学生）が衛生面で安心できることが集客上も重要

## 3. 消毒方法（予備・再利用マウスピース用）

万が一、再利用タイプのマウスピース（シリコン製・金属製など）を使う場合の消毒手順。

### 消毒方法の比較

| 方法 | 手順 | 接触時間 | メリット | デメリット |
|------|------|----------|----------|------------|
| **次亜塩素酸ナトリウム（ハイター等）** | 0.05%溶液に浸漬 | 5分以上 | 殺菌力が高い、安価 | 金属を腐食する、残留臭がある |
| **次亜塩素酸水（電解水）** | 原液〜200ppmでスプレーまたは浸漬 | 1〜3分 | 残留性が低い、口に触れるものに適する | 有効期限が短い、保管に注意 |
| **消毒用エタノール（70〜80%）** | スプレー後拭き取りまたは浸漬 | 30秒〜1分 | 速乾性、入手しやすい | プラスチックを劣化させる場合がある |
| **煮沸消毒** | 100℃の沸騰水に投入 | 5分以上 | 薬品不要 | 耐熱素材限定、手間がかかる |

### 推奨手順（再利用マウスピースの場合）

1. 使用直後に流水で予洗い（残留フレーバー・唾液の除去）
2. 中性洗剤とブラシで物理的に洗浄
3. 次亜塩素酸水（200ppm）に3分間浸漬
4. 流水で十分にすすぎ（30秒以上）
5. 清潔なペーパータオルで水気を拭き取り
6. 自然乾燥（通気性のよいラック等で保管）

## 4. 1セッションごとの交換ルール

### 基本ルール

| ルール | 内容 |
|--------|------|
| **1人1個の原則** | 来店されたお客様1人につき、使い捨てマウスピースを1個提供 |
| **セッション中の追加交換** | お客様が落としたり、汚れた場合は無料で新品を提供 |
| **複数人利用時** | 全員に個別のマウスピースを配布。共有は禁止 |
| **開封タイミング** | お客様の目の前で個包装を開封する（衛生アピール） |
| **使用済みの処理** | セッション終了後、専用のゴミ箱に廃棄 |

### セッション中の交換フロー

```
お客様来店
  ↓
個包装マウスピースを目の前で開封・提供
  ↓
セッション中（必要に応じて追加提供）
  ↓
セッション終了
  ↓
使用済みマウスピースを回収・廃棄
  ↓
次のお客様用に新品を準備
```

## 7. 保健所対応メモ

保健所の検査・立ち入り時に説明できるよう、以下を準備しておく。

- [ ] 使い捨てマウスピースの商品パッケージ（素材・滅菌方法が記載されたもの）を保管
- [ ] 仕入れ伝票・納品書を整理（トレーサビリティ）
- [ ] 「1人1個・個包装・目の前で開封」のオペレーションルールを文書化
- [ ] 廃棄マウスピースは一般廃棄物として適切に処分していることの記録
