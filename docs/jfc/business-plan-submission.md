## 1. 創業の動機

### 事業着想の経緯

私は岩手大学理工学部に在籍する大学3年生です。東京で約2年間、AIエンジニアとして実務経験を積んだ後、2026年3月に復学しました。東京在住時にシーシャ（水たばこ）の文化に触れ、友人との交流手段として日常的に利用していました。

復学後、岩手大学周辺にシーシャを楽しめる場所が存在しないことに気づきました。盛岡市内の既存シーシャ店（6店舗）はいずれも中央通・内丸・菜園・大通の繁華街エリアに集中しており、岩手大学から徒歩で25分以上かかります。一方、岩手大学周辺の上田・本町通エリアにはシーシャ店が1店舗もありません。

この「地理的空白」に着目し、岩手大学徒歩圏内に完全貸切制のシーシャスペースを開業することで、大学生の交流の場を提供できると考えました。

### なぜ今この事業か

AIの急速な普及により、リモートワークやオンラインコミュニケーションが当たり前になりつつあります。その一方で、「同じ場所に集まり、同じ時間を過ごす」物理的な体験の価値は相対的に高まっています。シーシャは1セッション60〜120分かけてゆっくり楽しむものであり、スマートフォンやPCでは代替できない「場と空気の共有体験」を提供できます。

また、飲食の提供を行わず、シーシャ機材と空間のみを貸し出す「持ち込みスタイル」を採用することで、初期投資と運営コストを大幅に抑えた低リスクの事業設計が可能です。

## 3. 取扱商品・サービスの内容

### 事業概要

完全予約制・貸切制のシーシャ（水たばこ）スペースの運営。飲食の提供は行わず、お客様の持ち込みとします。

### サービス内容

| 区分 | 内容 | 売上構成比（見込み） |
|------|------|---------------------|
| グループ貸切（主力） | 3〜5名 / 120分 / シーシャ1台込み 4,000円〜 | 60% |
| ソロ・デュオ | 1〜2名 / 90分 / 2,200円〜 | 20% |
| パーティ貸切 | 6名以上 / 180分 / シーシャ2台込み 6,000円〜 | 10% |
| 月額サブスクリプション | シーシャパス 月3,000円（月4回スペース利用料無料）/ フリーパス 月5,000円（月4回シーシャ1台込み） | 10% |

### 料金体系

**基本プラン（主力: グループ貸切）**

- 3〜5名 / 120分: 4,000円（シーシャ1台込み）
- 追加シーシャ: 1,000円/台
- 例: 3名 + シーシャ2台 = 5,000円（1人あたり約1,667円）

**オプション**

| 項目 | 料金 |
|------|------|
| 延長 | 500円/30分 |
| 追加シーシャ | 1,000円/台 |
| フレーバーMIX | 200円 |
| トップ替え（フレーバー交換） | 800円 |

### サービスの特徴

1. **完全貸切制**: 他のお客様と空間を共有しません。友人グループだけのプライベート空間を提供します。
2. **飲食持ち込み自由**: コンビニ等で購入した飲食物をお持ち込みいただけます。店舗側で飲食物の提供は行いません。
3. **低価格**: 既存店の平均客単価（約2,500円/人）に対し、グループ利用で1人あたり約1,667円と、30〜50%程度安い価格設定です。
4. **岩手大学徒歩圏内**: 既存のシーシャ店は全て繁華街に集中しており、大学周辺には競合がありません。

### 対象顧客

- **メインターゲット**: 岩手大学の学生（20歳以上）。岩手大学の学生数は約5,500名。
- **サブターゲット**: 岩手県立大学、盛岡大学等の周辺大学の学生、及び近隣の社会人。
- ※ 20歳未満の方の入場はお断りします（たばこ事業法準拠）。

## 9. 収益計画の根拠と裏付け

### 売上の根拠

**1. 価格設定の妥当性**

盛岡市内の既存シーシャ店の客単価は1人あたり約2,000〜3,000円です。BiPシーシャはグループ利用で1人あたり約1,667円と、既存店より30〜50%安い価格設定としています。飲食の提供を行わないことで原価を抑え、低価格を実現します。

**2. 客数の根拠**

- 岩手大学の学生数: 約5,500名
- うち20歳以上（3・4年生 + 大学院生）: 推定約2,500〜3,000名
- 喫煙経験率（20代）: 約15〜20%（厚生労働省データ）
- シーシャに関心のある層: 推定300〜600名
- 月間60セッション（稼働率60%）は、上記の潜在顧客の10〜20%が月1回利用する水準であり、達成可能と考えます。

**3. 原価率の根拠**

シーシャ1セッションの材料原価は約345円です。

| 材料 | 使用量 | 単価 | 小計 |
|------|--------|------|------|
| フレーバー（たばこ葉） | 10g | 25円/g | 250円 |
| ココナッツ炭 | 6個 | 12円/個 | 72円 |
| マウスピース | 1〜2個 | 12円/個 | 18円 |
| その他消耗品 | — | — | 5円 |
| **合計** | | | **345円** |

セッション売上4,000〜5,000円に対し原価345円であり、原価率は7〜9%です。これはシーシャ業界の一般的な原価率（10〜25%）の範囲内であり、飲食業全体（原価率28〜35%）と比較して非常に優位な構造です。

## 11. リスクと対策

### 主要リスクの一覧

| リスク | 影響度 | 発生確率 | 対策 |
|--------|--------|---------|------|
| **許認可が取得できない** | 極大 | 中 | 物件契約前に保健所・財務局へ確認を完了する。不許可の場合は事業計画を根本から見直す |
| **CO中毒事故** | 極大 | 低 | CO検知器2台を常設。排気量800〜1,200m3/h以上の機械換気を設置。50ppmでアラームが鳴る検知器を採用。セッション中は代表者が常駐 |
| **火災** | 極大 | 低 | 消火器設置、防炎素材の家具使用、耐火床材の採用、火災保険（借家人賠償特約付き）への加入 |
| **稼働率が想定を下回る** | 大 | 中 | 間借りテスト（Phase 1）で3ヶ月間の需要検証を実施。リピート率30%未満の場合はPhase 2に進まず撤退する |
| **長期休暇中の売上減少** | 中 | 高（確実） | サブスク収入で固定費の一部をカバー。運転資金バッファ（固定費6ヶ月分）を事前に確保 |
| **近隣トラブル（騒音・臭い）** | 中 | 中 | 防音対策、排気の屋外排出、営業時間の制限、近隣への事前挨拶 |
| **泥酔客によるトラブル** | 中 | 中 | 予約時に利用規約への同意を義務付け。損害に対する賠償条項を明記 |
| **代表者の体調不良** | 中 | 低 | 完全予約制のため、体調不良時は予約を受け付けない運用で対応可能 |

### 撤退基準

以下のいずれかに該当した場合、事業を縮小または撤退します。

- Phase 1（間借りテスト）で3ヶ月間のリピート率が30%未満、かつ月間売上が10万円に満たない場合 → Phase 2に進まず撤退（損失: 15〜25万円）
- Phase 2（独立物件）で6ヶ月連続の赤字が続き、運転資金バッファが底をつく見込みとなった場合 → 物件を解約し撤退
- 許認可が取得できない、または取り消された場合 → 即時撤退

## 14. 自己PR・補足事項

### Build in Public（事業プロセスの全公開）

本事業は「Build in Public」というコンセプトのもと、物件探し・料金設計・原価計算・競合調査・法務確認など事業構築の全プロセスを、Webサイト（shisha.konaito.com）にて公開しています。これにより以下の効果を見込んでいます。

1. **開業前からの認知獲得**: サイト公開時点で岩手大学の学生を中心にSNSで認知が広がり始めており、開業時の初期集客コストを抑制できます。
2. **透明性による信頼構築**: 数字やリスクを隠さず公開することで、顧客や関係者からの信頼を得ています。
3. **事業計画の精度向上**: 公開することで外部からのフィードバックを受け、計画の弱点を事前に修正できています。

### 副業としての堅実な設計

本事業は「大学生活と両立可能な副業」として設計しています。完全予約制のため営業日・営業時間を柔軟に調整でき、学業に支障をきたしません。月間利益6.7〜8.8万円は本業（学業）を持つ大学生の副収入として十分であり、無理な拡大を行わない堅実な運営を心がけます。

### フェーズ制による段階的な投資

一度に大きな投資を行うのではなく、Phase 0（調査: 約3,000円）→ Phase 1（間借りテスト: 15〜25万円）→ Phase 2（独立物件: 約150万円）と段階的に投資額を増やす計画です。各フェーズで得られたデータに基づいて次の投資判断を行うため、失敗した場合の損失を最小限に抑えられます。
