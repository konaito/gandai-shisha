### 学生起業で融資を受けるための前提条件

日本政策金融公庫は学生でも融資を受けられる。利用できる制度は以下の通り。

- **新規開業・スタートアップ支援資金**: 融資限度額7,200万円（うち運転資金4,800万円）
- **女性・若者/シニア起業家支援資金**: 35歳未満の若者が対象で、低金利で借りられる

ただし学生の場合、以下が特に重要:

- **自己資金の準備**: アルバイト等でコツコツ貯めた自己資金があることが信頼につながる
- **経済的自立の証拠**: 携帯料金・家賃等を自分名義で支払っている実績
- **数字の根拠**: 学生だからこそ、市場調査やデータに基づく計画が求められる

## 2. 各項目の解説と記入例

### 項目1: 創業の動機

#### 解説

「なぜこの事業を始めたいのか」を具体的に書く。担当者が見ているのは以下の3点:

1. **課題認識**: どんな問題を解決したいのか
2. **準備の具体性**: 思いつきではなく、調査・準備をしてきた証拠
3. **実現可能性**: 自分がやる必然性（経験・スキル・人脈）

#### BiPシーシャの記入例

```
岩手大学に通う中で、学生が気軽にリラックスできる場所が極端に少ないこと
を課題に感じていました。居酒屋やカラオケ以外の選択肢として、都市部で
人気が拡大しているシーシャ（水たばこ）に着目しました。

盛岡市内にはシーシャ専門店がほぼなく、市場調査の結果、大学生を中心とした
需要があることを確認しました。完全貸切制にすることで、初期投資を最小限に
抑えつつ、プライベート空間という付加価値を提供できると考えています。

事前準備として、Webサイト（shisha.konaito.com）で事業構築の全プロセスを
公開する「Build in Public」手法を採用し、開業前から潜在顧客との接点を
構築しています。SNS（Instagram）での反応から、一定の需要があることも
確認済みです。
```

> **ポイント**: 「好きだから」「流行っているから」だけでは弱い。課題→解決策→準備実績の流れで書く。

### 項目2: 経営者の略歴等

#### 解説

担当者が最も重視するのは「創業する事業に関連する経験があるか」。直接的な業界経験がなくても、関連するスキルや実績を結びつける。

#### BiPシーシャの記入例

```
【学歴】
岩手大学 理工学部 数理・物理コース 在学中（3年次）

【関連する経験・スキル】
・Web開発・マーケティング: 個人でWebサイトやアプリを開発・運営。
  SNSマーケティング（Instagram運用）の実務経験あり
・事業企画: Build in Publicサイト（shisha.konaito.com）で物件調査、
  料金設計、原価計算、競合分析など事業計画の全工程を実施・公開中
・シーシャの知識: 自身がシーシャ愛好家として機材・フレーバーの知識を蓄積。
  複数のシーシャ店舗を訪問し、オペレーション・接客を研究
・IT活用: 予約管理、会計管理、顧客管理をITツールで効率化する計画を策定済み

【取得資格・免許】※取得予定も含む
・たばこ小売販売業許可（申請予定）
・食品衛生責任者（取得予定 ※飲食提供がある場合）
```

> **ポイント**: 業界未経験でも、「自分で調査・研究してきた」「IT活用で効率化できる」といった強みを具体的に示す。

### 項目3: 取扱商品・サービス

#### 解説

商品・サービスの内容、セールスポイント、販売ターゲット、競合との差別化を記載する。

#### BiPシーシャの記入例

```
【取扱商品・サービスの内容】
① シーシャ（水たばこ）提供サービス
  ・完全貸切制のプライベートシーシャスペース
  ・1セッション（約2時間）：グループ貸切 5,000円（3人利用時 約1,667円/人）
  ・多種フレーバーから選択可能（フルーツ系、ミント系、スパイス系等）
  ・飲食持ち込み制（飲食店営業許可不要でコスト削減）

② サブスクリプションプラン
  ・月額3,000円で月2回利用可能（1回あたり1,500円）
  ・安定的なリピーター確保と収益の平準化

【セールスポイント】
  ・盛岡市初の完全貸切制シーシャスペース
  ・1人あたり約1,667円と大学生に手の届く価格設定
  ・プライベート空間で周囲を気にせずリラックスできる
  ・Build in Publicで開業前からファンコミュニティを構築

【販売ターゲット・商圏】
  ・主要ターゲット: 岩手大学生（約5,500名）、近隣大学生
  ・副次ターゲット: 盛岡市内の20〜30代社会人
  ・商圏: 岩手大学から徒歩・自転車圏内（半径2km程度）

【競合との差別化】
  ・盛岡市内にシーシャ専門店はほぼ皆無（競合不在の市場）
  ・完全貸切制は仙台以北では希少
  ・飲食持ち込み制により原価率11〜17%と低コスト運営を実現
```

## 3. 収支計画の書き方

### 基本の算出式

シーシャスペースの売上は以下の式で算出する:

```
月間売上 = セッション単価 × 1日あたりセッション数 × 月間営業日数
```

### 売上の算出根拠を明示する

担当者に対して、以下のように根拠を説明する:

```
【売上高の算出根拠】

■ 貸切セッション
  ・単価: 5,000円/セッション（3名利用で1人あたり約1,667円）
  ・創業当初: 週4セッション × 4週 = 月16セッション = 80,000円
    → SNSフォロワーおよびBuild in Publicサイト経由の認知から算出
  ・軌道後: 週6〜7セッション × 4週 = 月26セッション = 130,000円
    → 口コミ拡大・リピーター定着を想定

■ サブスクリプション
  ・単価: 3,000円/月
  ・創業当初: 4名 → 12,000円/月
  ・軌道後: 10名 → 30,000円/月
    → 岩手大学生 約5,500名の0.2%がサブスク加入と仮定

■ 原価率の根拠
  ・フレーバー: 1セッションあたり約15g使用、50g 1,500円 → 1回あたり約450円
  ・炭: 1セッションあたり約150円
  ・原価率 = (450+150) / 5,000 = 12%
  ・シーシャ業界の一般的な原価率: 10〜20%（当事業はこの範囲内）
```

## 4. 「事業の見通し」の書き方のコツ

### 3つのシナリオを用意する

創業計画書のフォーマットには「創業当初」と「軌道に乗った後」の2列しかないが、面談時に3シナリオを口頭で説明できるように準備しておく。

#### 楽観シナリオ（目標値）

| | 月間売上 | 月間利益 | 備考 |
|---|---------|---------|------|
| 6ヶ月後 | 200,000円 | 120,000円 | 毎日セッション + サブスク15名 |
| 1年後 | 250,000円 | 160,000円 | 2部制導入、単価見直し |

#### 現実シナリオ（計画値）← 創業計画書にはこれを書く

| | 月間売上 | 月間利益 | 備考 |
|---|---------|---------|------|
| 創業当初 | 100,000円 | 40,000円 | 週4セッション |
| 6ヶ月後 | 170,000円 | 98,000円 | 週6〜7セッション + サブスク10名 |

#### 悲観シナリオ（最悪ケース）

| | 月間売上 | 月間利益 | 備考 |
|---|---------|---------|------|
| 創業当初 | 60,000円 | -5,000円 | 週2〜3セッションのみ |
| 6ヶ月後 | 100,000円 | 30,000円 | 認知拡大に時間がかかるケース |

### 「事業の見通し」記述のコツ

1. **計画書には現実シナリオを書く**: 楽観的すぎると信頼を失う
2. **売上は控えめ、経費は多めに**: 堅実な姿勢が評価される
3. **悲観シナリオでも破綻しないことを示す**: 赤字でも生活費（アルバイト収入等）でカバーでき、事業継続可能であることを説明
4. **数字には必ず算出式をつける**: 「客単価 × 回数 × 営業日数」の形式
5. **根拠は客観的データを使う**: 競合調査、業界平均、SNSの反応数など
6. **黒字化の時期を明示する**: 「3ヶ月目で単月黒字化、6ヶ月で投資回収」のように

### BiPシーシャ特有の強み（見通しに織り込む）

- **低い損益分岐点**: 月間経費が約4.5万円と低いため、月9セッション（週2回強）で損益分岐
- **在庫リスクが小さい**: フレーバー・炭は少量仕入れが可能で、腐敗しにくい
- **固定費が小さい**: 人件費ゼロ（1人運営）、飲食提供なしで食材廃棄リスクなし
- **撤退コストが低い**: 大型設備投資がなく、最悪の場合でも機材をフリマ等で売却可能

## 5. 公庫の担当者に刺さるポイント

### 1. 「なぜあなたがやるのか」の説得力

- 自分自身がシーシャ愛好家で、機材・フレーバーの知識がある
- IT/Webスキルがあり、集客・運営の効率化ができる
- Build in Publicで既に事業計画の全プロセスを公開・実行している（口だけではない証拠）

### 2. 数字の一貫性と根拠

- 売上の算出式が具体的（単価 × 回数 × 日数）
- 原価率が業界水準（10〜20%）の範囲内
- 必要資金と調達方法の合計が一致
- 収支計画の各数字が他の項目と矛盾しない

### 3. リスクへの対策を示す

| リスク | 対策 |
|--------|------|
| 集客が伸びない | Build in Public + SNSで開業前から認知獲得。大学内の口コミ活用 |
| 季節変動（夏休み等） | サブスクで安定収入を確保。長期休暇は営業時間調整 |
| たばこ販売許可が取れない | 事前に管轄の財務局に相談済み。距離規制のクリアを確認 |
| 法規制の変更 | たばこ事業法・消防法を継続的にモニタリング |
| 近隣トラブル | 換気設備の設置、営業時間の制限、防音対策 |

### 4. 自己資金の準備と返済能力

- アルバイト収入から計画的に自己資金を貯めてきたことを通帳で証明
- 融資額が少額（15〜30万円程度）であれば、事業利益で十分に返済可能
- 学生の間はアルバイト収入もあるため、最悪の場合でも返済に支障がない

### 5. 添付資料で差をつける

創業計画書はA3用紙1枚のフォーマットだが、以下の補足資料を別紙で添付すると効果的:

- **事業計画の詳細資料**: Build in Publicサイトの印刷版
- **競合調査レポート**: 盛岡市内の類似店舗の調査結果
- **価格調査**: 他都市のシーシャ店の料金比較表
- **SNSの実績データ**: フォロワー数、エンゲージメント、反応のスクリーンショット
- **設備の見積書**: シーシャ機材の見積もり
- **物件情報**: 候補物件の情報（家賃、広さ、立地）

### 6. 面談対策

公庫の融資は面談が必須。以下を準備しておく:

- 創業計画書の各数字をすべて自分の言葉で説明できるようにする
- 「なぜシーシャなのか」「なぜ盛岡なのか」「なぜ今なのか」に即答できるようにする
- 悲観シナリオでも事業が継続できる理由を説明できるようにする
- たばこ事業法の許可取得プロセスを把握していることを示す
- 服装はビジネスカジュアル（学生でもきちんとした印象を与える）

## 6. NG例・よくある失敗パターン

### NG 1: 売上予測に根拠がない

```
❌ NG例:
「月に20万円くらいは売り上げられると思います。シーシャは流行っているので。」

✅ 改善例:
「セッション単価5,000円 × 月26回 = 130,000円。サブスク3,000円 × 10名 = 30,000円。
合計170,000円。セッション数は、SNSフォロワー300名のうち転換率3%で約9名/月の
新規獲得を見込み、リピート率50%を加味して算出。」
```

### NG 2: 創業の動機が抽象的

```
❌ NG例:
「シーシャが好きで、みんなに広めたいと思いました。
世の中に癒しを届けたいです。」

✅ 改善例:
「盛岡市にシーシャ専門店がほぼ存在せず、需要があるにもかかわらず供給が
不足していることを市場調査で確認しました。自身の機材知識とIT・マーケティング
スキルを活かし、低コスト・完全貸切モデルで参入します。」
```

### NG 3: 自己資金がゼロ

```
❌ NG例:
自己資金0円、全額借入を希望

✅ 改善:
少額でもよいので自己資金を用意する。最低でも必要資金の3分の1。
15万円の事業なら5万円、30万円なら10万円。
コツコツ貯めた通帳の履歴が重要（一括入金は「見せ金」と疑われる）。
```

### NG 4: 必要な許認可に触れていない

```
❌ NG例:
シーシャ事業に必要な許可について一切触れていない

✅ 改善:
以下の許認可が必要であることを計画書または別紙で明記する:
・たばこ小売販売業許可（財務局に申請）
・開業届（税務署）
・消防署への届出（換気・防火設備）
※飲食持ち込み制のため飲食店営業許可は不要
```

### NG 5: 経費の見積もりが甘い

```
❌ NG例:
経費は家賃のみ記載で「他はほとんどかからない」

✅ 改善:
水道光熱費、通信費、消耗品、広告宣伝費、雑費まで漏れなく記載する。
予備費も含めて、実際よりやや多めに見積もる。
```

### NG 6: 返済計画が不明確

```
❌ NG例:
「儲かったら返します」

✅ 改善:
「月間営業利益40,000〜98,000円の見通し。借入額150,000円に対し、
月額返済10,000円 × 15ヶ月で完済予定。悲観シナリオでも月間利益30,000円
を見込めるため、返済に支障なし。」
```

### NG 7: 競合分析がない

```
❌ NG例:
「盛岡にはシーシャ屋がないので競合はいません」

✅ 改善:
「直接競合（シーシャ専門店）は盛岡市内にほぼ存在しない。
間接競合として居酒屋、カラオケ、カフェがあるが、差別化要因は
①シーシャという独自体験 ②完全貸切のプライバシー ③1人1,667円の
手頃な価格。仙台・東京のシーシャ店との比較でも価格競争力あり。」
```
