# 創業の動機

## 事業着想の経緯

- 2024〜2025年、東京でAIエンジニアとして勤務中にシーシャ文化に触れる
- 2026年3月に岩手大学に復学 → **大学周辺にシーシャ店がゼロ**であることに着目
- 盛岡市内の既存6店舗はすべて**繁華街（中央通・大通・菜園）に集中**
- 岩手大学から最寄りのシーシャ店まで**徒歩25分以上**

## なぜ今この事業か

- AI時代に「同じ場所に集まり、同じ時間を過ごす」**物理体験の価値が向上**
- シーシャは60〜120分かけて楽しむ → **スマホでは代替できない体験**
- 飲食提供なし（持ち込み制）で**低コスト・低リスクの事業設計**が可能

# 07
## リスク管理

---

# 主要リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| **許認可が取得できない** | 極大 | 物件契約前に保健所・財務局へ確認完了。不許可なら計画見直し |
| **CO中毒事故** | 極大 | CO検知器2台常設、排気800〜1,200m³/h、代表者常駐 |
| **火災** | 極大 | 消火器設置、防炎家具、耐火床材、借家人賠償特約付き火災保険 |
| **稼働率低迷** | 大 | Phase 1で3ヶ月間の需要検証。リピート率30%未満なら撤退 |
| **長期休暇の売上減** | 中 | サブスク収入＋運転資金バッファ（固定費6ヶ月分） |
| **近隣トラブル** | 中 | 防音対策、排気の屋外排出、営業時間制限、事前挨拶 |

## 撤退基準（明確に設定済み）

- Phase 1: リピート率30%未満 & 月売上10万円未満 → **即撤退**（損失: 15〜25万円）
- Phase 2: 6ヶ月連続赤字で運転資金枯渇見込み → **物件解約・撤退**
- 許認可取得不可・取消 → **即時撤退**
