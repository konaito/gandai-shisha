## 1. 利用可能な融資制度一覧

### 1-1. 新規開業・スタートアップ支援資金（旧：新規開業資金）

**最も有力な制度。2025年3月に「新規開業資金」から改名。**

| 項目 | 内容 |
|------|------|
| 対象者 | 新たに事業を始める方、または事業開始後おおむね7年以内の方 |
| 融資限度額 | 7,200万円（うち運転資金4,800万円） |
| 返済期間 | 設備資金：20年以内 / 運転資金：10年以内 |
| 据置期間 | 各5年以内 |
| 担保・保証人 | 創業期は原則として無担保・無保証人 |
| 自己資金要件 | **撤廃済み**（2024年3月〜。ただし実務上は用意が望ましい） |
| 金利優遇 | 基準利率から0.65%引下げ（雇用拡大時は0.9%引下げ） |

**備考**: 旧「新創業融資制度」は2024年3月に廃止されたが、無担保・無保証人の仕組みは本制度に引き継がれている。

### 1-2. 女性・若者・シニア起業家支援（金利優遇枠）

独立した融資制度ではなく、新規開業・スタートアップ支援資金の中で **特別利率** が適用される優遇措置。

| 項目 | 内容 |
|------|------|
| 対象者 | 女性 / **35歳未満の方** / 55歳以上の方で新たに事業を始める方 |
| 優遇内容 | 特別利率A〜Cが適用され、基準利率より0.4〜0.9%程度低い金利 |
| 適用条件 | 新規開業・スタートアップ支援資金と併用 |

**大学生（21〜22歳）は「35歳未満」に該当するため、金利優遇の対象になる。**

### 1-3. 創業支援貸付利率特例制度

| 項目 | 内容 |
|------|------|
| 対象者 | 新たに事業を始める方・事業開始後税務申告を2期終えていない方 |
| 優遇内容 | 新規開業・スタートアップ支援資金の利率からさらに0.65%引下げ |
| 期間 | 融資後3年間 |

### 1-4. その他関連制度

| 制度名 | 概要 |
|--------|------|
| 経営者保証免除特例制度 | 法人代表者の連帯保証を免除 |
| マル経融資（小規模事業者経営改善資金） | 商工会議所の推薦が必要。特別利率F（2.40%）が適用 |

## 2. 融資条件の詳細

### 2-1. 金利一覧（令和8年3月2日現在）

#### 無担保融資（税務申告2期未了＝創業者が該当）

| 利率区分 | 金利範囲 |
|----------|----------|
| 基準利率 | 3.25〜4.65% |
| 特別利率A | 2.85〜4.25% |
| 特別利率B | 2.60〜4.00% |
| 特別利率C | 2.35〜3.75% |
| 特別利率E | 1.85〜3.25% |

#### 有担保融資

| 利率区分 | 金利範囲 |
|----------|----------|
| 基準利率 | 2.30〜4.30% |
| 特別利率A | 1.90〜3.90% |
| 特別利率B | 1.70〜3.65% |
| 特別利率C | 1.65〜3.40% |

> **想定ケース**: 35歳未満の若者枠で特別利率Aが適用された場合、無担保で **2.85〜4.25%** 程度。さらに創業支援貸付利率特例で0.65%引下げされると **2.20〜3.60%** 程度が目安。

### 2-2. 返済シミュレーション（参考）

50万円を金利3.0%・5年返済で借りた場合：

- 月々返済額：約8,985円
- 総返済額：約539,100円（利息約39,100円）

### 2-3. 担保・保証人

- 創業期は **原則として無担保・無保証人**
- 個人事業主の場合、法人のような連帯保証の問題は発生しない
- 担保を提供すると金利が下がる（基準利率で約1%差）

### 2-4. 自己資金要件

- 制度上の要件は **2024年3月に撤廃**
- ただし実務上は **融資希望額の1/3程度** あると審査が通りやすい
- 公庫の調査では創業資金総額に占める自己資金の割合は **平均2割程度**
- 自己資金が少ない場合、融資額が制限される傾向あり

## 3. 申請の流れ

### ステップ1: 事前相談（融資申込の1〜2ヶ月前）

| 方法 | 詳細 |
|------|------|
| 電話相談 | 事業資金相談ダイヤル **0120-154-505**（平日9:00〜19:00） |
| 窓口相談 | 最寄りの日本政策金融公庫支店で予約相談 |
| 商工会議所 | 地域の商工会議所・商工会でも相談可能 |

**岩手県の管轄支店**: 盛岡支店（盛岡市中央通1-7-35）

> この段階で「どの制度が使えるか」「自己資金はどの程度必要か」を確認する。

### ステップ2: 申し込み

- **インターネット申込**: 24時間365日受付（公庫Webサイトから）
- **郵送申込**: 最寄りの支店に書類を送付
- 創業計画書と必要書類を提出

### ステップ3: 面談（申込から1〜2週間後）

- 担当者と30〜60分の面談
- 事業計画の詳細、資金使途、返済計画を確認
- 店舗・事業所予定地の訪問調査あり
- **オンライン面談も対応可能**

### ステップ4: 融資決定・実行（面談から1〜2週間後）

- 審査結果の通知
- 電子契約手続き
- 指定銀行口座への送金

### タイムライン全体

```
事前相談 → 申込 → 面談 → 融資実行
  ↓        ↓      ↓       ↓
 随時   書類準備  1-2週間  1-2週間
        1-2週間

合計: 申込から融資実行まで約3週間（平均）
```

## 4. 必要書類リスト

### 4-1. 必須書類

| 書類 | 備考 |
|------|------|
| **借入申込書** | 公庫所定の書式（Webまたは窓口で入手） |
| **創業計画書** | 公庫所定の書式。事業概要・資金計画・収支見通しを記入 |
| **本人確認書類** | 運転免許証・マイナンバーカード・パスポートのいずれか |
| **許認可証** | たばこ小売販売業許可証、飲食店営業許可証など（取得済みの場合） |

### 4-2. 状況に応じて必要な書類

| 書類 | 該当するケース |
|------|--------------|
| 設備の見積書 | 設備資金を申し込む場合（シーシャ機材、内装工事等） |
| 物件の賃貸借契約書（案） | 店舗物件が決まっている場合 |
| 預金通帳（直近6ヶ月分以上） | 自己資金の証明・資金の出入りの確認 |
| 法人の履歴事項全部証明書 | 法人で申し込む場合 |
| 不動産登記簿謄本 | 担保を提供する場合 |
| 収支計画書・資金繰り表 | 審査過程で追加提出を求められることがある |
| 源泉徴収票・確定申告書 | 現在の収入を証明する書類 |

### 4-3. 面談時に持参すべきもの

- 本人確認書類（原本）
- 預金通帳（原本）
- 創業計画書の控え
- 事業に関する補足資料（メニュー表、価格表、仕入先の見積書、物件の写真等）
- 商品サンプル（シーシャのフレーバー等があれば）

## 5. 審査で見られるポイント

### 5-1. 最重要ポイント

#### (1) 自己資金の質と量
- 制度上の要件は撤廃されたが、**審査上は極めて重要**
- コツコツ貯めた形跡のある預金が高評価（通帳で確認される）
- 突然の大金入金は「見せ金」と疑われるリスクあり
- **融資希望額の1/3程度が目安**（50万円借りたいなら自己資金15〜20万円）

#### (2) 事業計画の具体性・実現可能性
- 売上の根拠が具体的な数字で示されているか
- 「なんとなく」「たぶん」ではなく計算根拠があるか
- ターゲット顧客、集客方法、差別化ポイントが明確か

#### (3) 代表者の信用情報
- CIC（信用情報機関）の記録が確認される
- クレジットカードの延滞、携帯料金の未払い等があると厳しい
- **学生であっても信用情報は存在する**（携帯分割払い、クレカ等）

#### (4) 返済能力
- 毎月の返済額を事業収入で賄えるか
- 副業の場合、本業（アルバイト等）の収入も考慮される

### 5-2. プラス評価になる要素

- 事業に関連する経験・スキル（シーシャ店でのアルバイト経験等）
- 既に具体的な取引先・仕入先が決まっている
- 物件が確定している
- 許認可の取得（または取得見込み）
- SNSフォロワー等の集客基盤
- Build in Publicサイトでの情報発信実績

### 5-3. マイナス評価になる要素

- 自己資金がゼロまたは極端に少ない
- 信用情報に傷がある（延滞、債務整理等）
- 事業計画が曖昧・非現実的
- 資金使途が不明確
- 借入金を自己資金と偽る（見せ金は即アウト）

## 6. 面談対策

### 6-1. よく聞かれる質問と模範回答の方向性

#### Q1: 創業の動機は？
- **求められる回答**: 単なる「儲けたい」ではなく、社会的意義や地域への貢献を含める
- **シーシャの場合**: 「岩手にシーシャ文化がまだ浸透していない → 大学生が気軽にリラックスできる空間を提供したい → 地域の若者コミュニティの活性化」

#### Q2: なぜこの事業ですか？経験はありますか？
- シーシャに関する知識・経験（利用経験、調査実績等）
- Build in Publicでの調査プロセスを説明できると強い

#### Q3: 事業の強み・セールスポイントは？
- 競合との差別化（貸切、価格、立地等）
- 専門用語を避け、わかりやすく説明する

#### Q4: 売上の見通しは？根拠は？
- **数字で説明**: 「1日◯組 × 単価◯円 × 稼働日数◯日 = 月商◯円」
- 類似店舗のデータ、市場調査の結果を引用

#### Q5: 自己資金はどうやって貯めましたか？
- 通帳の入金履歴と整合する説明を用意
- アルバイト収入からコツコツ貯めた等

#### Q6: 他に借入はありますか？
- 正直に申告（嘘は信用情報で即バレる）
- 奨学金がある場合は延滞なく返済していることを強調

#### Q7: リスクにはどう対応しますか？
- 「リスクはない」は絶対NG
- 想定されるリスクと対策をセットで説明

#### Q8: 返済計画は？
- 具体的な月次収支から返済可能であることを示す

### 6-2. 絶対に避けるべきNGワード・行動

| NG | 理由 |
|----|------|
| 「いくらまで借りられますか？」 | 資金使途が不明確と判断される |
| 「リスクはありません」 | リスク認識能力の欠如と見なされる |
| 「それは◯◯に聞かないとわかりません」 | 主体性・経営者としての覚悟がないと判断される |
| 「たぶん」「だと思います」 | 計画の曖昧さを露呈する |
| 専門家（税理士等）の同席 | 本人の能力・覚悟を評価する場なので逆効果 |
| 代理人による出席 | そもそも認められない |
| 「同業他社もこれくらいだから」 | 自分の事業の根拠にはならない |
| Excel形式での書類提出 | 改ざんリスクがあるためPDFで提出する |

### 6-3. 面談成功のコツ

1. **創業計画書を完全に理解する** — 自分が書いた数字をすべて説明できるようにする
2. **視覚資料を用意する** — メニュー表、店舗写真、競合調査表、Webサイトのスクリーンショット等
3. **身だしなみ** — スーツまたはオフィスカジュアル。清潔感重視
4. **誠実な態度** — わからないことは正直に「確認して回答します」と答える
5. **数字で語る** — 感覚的な表現を避け、具体的な数値と計算根拠を示す

## 7. 大学生・若者が融資を受ける際の注意点

### 7-1. 大学生特有のハードル

| ハードル | 対策 |
|----------|------|
| 社会的信用が低い | 事業計画の精度で補う。Build in Publicの実績も信用材料になりうる |
| 事業経験がない | シーシャ店でのアルバイト経験、調査実績、関連知識の習得をアピール |
| 安定収入がない | 副業として開始し、本業（アルバイト等）の収入もあることを示す |
| 自己資金が少ない | 少額でもコツコツ貯めた実績を通帳で示す。15〜50万円規模なら自己資金10〜20万円でも可能性あり |
| 返済能力の不安 | 月々の返済額を低く設定し、確実に返済できる計画を立てる |

### 7-2. 大学生のアドバンテージ

- **35歳未満の若者枠** で金利優遇を受けられる
- 公庫は「政策金融」であり、民間金融機関が融資しにくい層（若者・女性・シニア）を支援する使命がある
- **小規模・低リスクの事業** は公庫が最も得意とする領域
- 公庫の統計では自己資金割合は平均2割 → 初期投資50万円なら自己資金10万円からでも理論上は申込可能

### 7-3. 学生ならではの準備ポイント

1. **開業届の提出** — 個人事業主として開業届を税務署に提出してから申し込む
2. **事業用口座の開設** — 個人口座とは別に事業用の通帳を作っておく
3. **創業予定地の確定** — 物件が決まっていないと申し込みができない
4. **許認可の取得（または取得見込み）** — シーシャの場合はたばこ関連許可が必須
5. **奨学金の状況整理** — 延滞がないことを確認。融資面談で聞かれる可能性あり

## 8. シーシャ・飲食業での融資申請時の特有の注意点

### 8-1. シーシャ開業に必要な許認可一覧

融資申請前に、以下の許認可の取得（または取得見込み）を確認しておくこと。

| 許認可 | 管轄 | 備考 |
|--------|------|------|
| **飲食店営業許可** | 保健所 | ドリンク・フードを提供する場合は必須 |
| **たばこ小売販売業許可**（一般小売販売業） | 財務局 | 自らたばこを仕入れて提供する場合。距離制限あり |
| **たばこ出張販売許可** | 財務局 | 仕入先業者が取得する方式。小売販売許可より取得しやすい |
| **深夜酒類提供飲食店営業届出** | 警察署 | 深夜0時以降に酒類を提供する場合 |
| **喫煙目的室の設置届出** | 保健所 | 2020年4月施行の改正健康増進法への対応 |

### 8-2. 健康増進法との関係（重要）

2020年4月施行の改正健康増進法により、新規開業の飲食店は **原則屋内禁煙**。シーシャを提供するには以下の要件を満たす必要がある：

- **たばこの対面販売** を行っていること
- **喫煙を主たる目的** とする施設であること（「喫煙目的店」の認定）
- 法令・条例による **設備基準** を満たしていること
- 20歳未満の客の入場を禁止すること

> **融資申請への影響**: これらの許認可を取得済み（または取得見込みであることを説明できる状態）にしておかないと、事業の実現可能性が疑問視される。

### 8-3. 飲食業の融資申請で気をつけること

| ポイント | 説明 |
|----------|------|
| 許認可の事前確認 | たばこ関連許可の取得可否を事前に財務局に確認する |
| 距離制限の調査 | たばこ小売販売業許可には既存販売店との距離制限がある。物件選びの段階で確認 |
| 保健所への事前相談 | 飲食店営業許可の要件（設備基準）を物件契約前に確認 |
| 資金使途の明確化 | シーシャ機材、フレーバー仕入れ、内装工事等の見積書を用意 |
| 原価率の説明 | シーシャの原価率（フレーバー、炭等）を具体的に計算して示す |
| 衛生管理計画 | HACCP対応の衛生管理計画を策定しておく |
| 「シーシャ」の説明準備 | 審査担当者がシーシャを知らない可能性が高い。わかりやすい説明資料を用意する |

### 8-4. 貸切スペース型の場合の追加ポイント

- レンタルスペースとしての運営形態を明確にする
- 予約システム・集客方法を具体的に示す
- 1回あたりの売上単価 × 稼働率で現実的な収支計画を立てる
- 火気使用に関する消防法の確認（炭を使用するため）

## 付録: 初期投資15〜50万円の場合の融資戦略

### パターンA: 自己資金のみで開業（融資なし）

- 初期投資15〜30万円程度なら、アルバイト収入からの貯蓄で賄える可能性あり
- メリット: 借入リスクゼロ、手続き不要、すぐに開業可能
- デメリット: 設備投資に制限あり

### パターンB: 少額融資を活用

- 自己資金10〜20万円 + 融資30〜50万円 = 初期投資40〜70万円
- 公庫に相談する価値あり。ただし少額融資は公庫側のコストパフォーマンスが低いため、審査がより慎重になる可能性
- **融資希望額は自己資金の3〜4倍以内が現実的な上限**

### パターンC: 成長を見据えた融資

- 開業後の運転資金（3〜6ヶ月分）も含めて100〜200万円の融資を申し込む
- より本格的な事業計画が必要
- 事業の拡大計画（2店舗目、メニュー拡充等）を示せると説得力が増す

### 推奨アプローチ

1. まず **公庫の事前相談（無料）** を利用して、自分のケースで融資が可能かを確認
2. 創業計画書を作成して持参し、フィードバックをもらう
3. 指摘事項を修正してから正式に申し込む
4. 電話相談ダイヤル: **0120-154-505**
