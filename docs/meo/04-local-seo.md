## 1. サイテーション（NAP情報）の構築先一覧

サイテーションとは、Web上で店舗名・住所・電話番号（NAP）が言及されている状態のこと。Googleはサイテーション数をローカルランキングの評価要素として使用している。

### 最重要: NAP統一ルール

全ての掲載先で以下の情報を **完全に一致** させる:

| 項目 | 統一フォーマット（例） |
|------|---------------------|
| ビジネス名 | BiPシーシャ |
| 住所 | 岩手県盛岡市〇〇町X丁目X番X号 |
| 電話番号 | 019-XXX-XXXX |
| WebサイトURL | https://shisha.konaito.com |

表記揺れ（全角/半角、丁目の有無、ハイフンの有無等）を一切作らないこと。

## 2. ポータルサイト活用戦略

### 食べログ

- **無料掲載**: 店舗基本情報の掲載は無料
- **有料プラン**: 予約機能やページカスタマイズは有料
- **シーシャ店の場合**: 「バー」「カフェ」カテゴリで登録
- **効果**: サイテーションとしての効果 + 食べログ自体のSEOが強いため被リンク効果も

### ホットペッパーグルメ

- 有料掲載が基本だが、無料プランがある場合も
- クーポン機能が強力（学生向けクーポンは効果的）
- 予約システム連携が可能

### シーシャ専門ポータル

盛岡のシーシャ店は既にこれらのポータルに掲載されている。BiPシーシャも開業時に必ず登録する。

- **SuKu**: 盛岡市のシーシャ店一覧ページがある（suku-shisha.com/shopcat/moriokashi/）
- **SHISHA-NAVI**: 盛岡市内の店舗一覧がある
- **シーシャスイタイ**: 口コミ機能つき

### 地域メディアへの掲載

盛岡の地域メディアへの掲載はサイテーション効果だけでなく、実際の集客にも直結する。

- **もりおか通信**: hookapookaのオープン記事等、シーシャ店の掲載実績あり
- **いわてFUN**: popoの取材記事あり
- アプローチ方法: 「岩手大学生が開業する貸切シーシャスペース」というストーリー性でプレスリリース

## 4. 構造化データの実装

### LocalBusiness構造化データ（JSON-LD）

公式サイト（shisha.konaito.com）の `<head>` タグ内に以下のJSON-LDを実装する。

```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "BiPシーシャ",
  "description": "盛岡市の岩手大学近くにある貸切シーシャスペース。少人数でゆったりとシーシャを楽しめるプライベート空間。",
  "url": "https://shisha.konaito.com",
  "telephone": "019-XXX-XXXX",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "〇〇町X丁目X番X号",
    "addressLocality": "盛岡市",
    "addressRegion": "岩手県",
    "postalCode": "020-XXXX",
    "addressCountry": "JP"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": "XX.XXXXX",
    "longitude": "XXX.XXXXX"
  },
  "openingHoursSpecification": [
    {
      "@type": "OpeningHoursSpecification",
      "dayOfWeek": ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday"],
      "opens": "XX:00",
      "closes": "XX:00"
    },
    {
      "@type": "OpeningHoursSpecification",
      "dayOfWeek": ["Saturday", "Sunday"],
      "opens": "XX:00",
      "closes": "XX:00"
    }
  ],
  "priceRange": "$$",
  "image": "https://shisha.konaito.com/img/store-photo.jpg",
  "sameAs": [
    "https://www.instagram.com/konaito5",
    "https://github.com/konaito"
  ]
}
```

### 実装方法（Next.js）

`frontend/app/layout.tsx` の `<head>` 内にスクリプトタグとして挿入する。

```tsx
<script
  type="application/ld+json"
  dangerouslySetInnerHTML={{
    __html: JSON.stringify({
      "@context": "https://schema.org",
      "@type": "LocalBusiness",
      // ... 上記のJSON-LDをここに
    }),
  }}
/>
```

または、`frontend/app/page.tsx` に `metadata` として設定する方法もある。

## 5. その他のローカルSEO施策

### 公式サイトのSEO基本設定

| 項目 | 現状 | 推奨アクション |
|------|------|---------------|
| title タグ | 要確認 | 「BiPシーシャ｜盛岡の貸切シーシャスペース」に最適化 |
| meta description | 要確認 | 120〜160文字でキーワードを含む説明文 |
| OGP設定 | 要確認 | SNSシェア時の表示を最適化 |
| モバイル対応 | 対応済み（レスポンシブ） | 引き続き維持 |
| ページ速度 | 要確認 | Core Web Vitalsの基準をクリア |
| SSL | 対応済み（https） | 引き続き維持 |

### 被リンク獲得

| 方法 | 具体的なアクション |
|------|-------------------|
| 地域メディア | もりおか通信、いわてFUN等に取材・掲載依頼 |
| 大学メディア | 岩手大学の学生メディア・サークルに紹介記事を依頼 |
| シーシャメディア | シーシャマガジン等に新店情報を提供 |
| プレスリリース | PR TIMES等で開業プレスリリースを配信 |
| ブログ | Build in Public のストーリーをブログ記事として発信 |

### コンテンツマーケティング

公式サイトにブログ/記事セクションを追加し、以下のようなコンテンツを発信:

- 「盛岡でシーシャを楽しむならBiPシーシャ」（ローカルSEO狙い）
- 「シーシャ初心者ガイド」（初心者向けキーワード狙い）
- 「おすすめフレーバー紹介」（定期更新コンテンツ）
- 「BiP日記（Build in Public）」（ユニークなコンテンツとしてファン獲得）
