# 口コミ獲得戦略

> BiPシーシャ（岩手大学生向け貸切シーシャスペース / 盛岡市）向け

---

## 1. 口コミを自然に集める方法

### 大原則

- 金銭・割引・特典と引き換えに口コミを依頼するのは **Googleガイドライン違反**
- 「良い口コミを書いてください」ではなく「体験の感想をGoogleに投稿してもらえると嬉しいです」という形で依頼する
- 口コミの内容を指定・指示してはいけない

### 具体的な獲得施策

#### A. 店内での声かけ

| タイミング | 方法 |
|-----------|------|
| 退店時 | 「本日はありがとうございました。よろしければGoogleに感想を投稿いただけると嬉しいです」と口頭で依頼 |
| 会計時 | QRコード付きカードを渡す（口コミ投稿ページへの直リンク） |
| テーブル | 口コミ依頼のPOPをさりげなく設置 |

#### B. 口コミ投稿リンクの活用

Googleビジネスプロフィールの管理画面から「口コミ投稿リンク」を取得できる。

- **QRコード化** して店内に掲示
- **LINE公式アカウント** の来店後メッセージにリンクを添付
- **Instagram ストーリー** でリンクをシェア
- **レシートやカード** にQRコードを印刷

#### C. SNS連携

| 施策 | 詳細 |
|------|------|
| Instagram投稿 | 来店客のストーリーズ投稿をリポストし、Google口コミも促す |
| LINE公式 | 来店翌日にサンクスメッセージ + 口コミリンクを送信 |
| 大学のSNSコミュニティ | 岩手大学関連のグループで体験をシェアしてもらう |

#### D. 初期口コミの獲得（開業直後）

- プレオープン・友人招待時に体験してもらい、率直な感想を口コミ投稿依頼
- 開業イベント参加者への口コミ依頼
- 大学のサークル・ゼミ単位での貸切利用後に依頼
- **目標: 開業3ヶ月で20件以上**

#### E. 口コミが増えやすい体験設計

口コミは「感動体験」から生まれる。以下の体験を意識的に設計する:

- 初めてのシーシャ体験（丁寧な説明 → 感動）
- フォトジェニックな空間（SNS投稿 → Google口コミへの導線）
- サプライズ要素（誕生日利用への小さなプレゼント等）
- 期待を超えるフレーバー提案（「お好みに合わせてブレンドします」等）

## 2. 口コミへの返信テンプレート

全ての口コミに **24〜48時間以内** に返信することを目標とする。機械的な定型文は避け、口コミの内容に触れたパーソナルな返信を心がける。

### ポジティブな口コミ（星4〜5）への返信

#### テンプレートA: 初来店のお客様

```
〇〇様、ご来店いただきありがとうございます！
初めてのシーシャを楽しんでいただけたようでとても嬉しいです。
[口コミで触れられた具体的な内容]は、私たちもこだわっているポイントなので、
気に入っていただけて光栄です。
またお友達とご一緒にお越しください。新しいフレーバーもご用意してお待ちしています。
```

#### テンプレートB: リピーターのお客様

```
〇〇様、いつもありがとうございます！
リピートしていただけること、何よりも嬉しいです。
[口コミの具体的内容に触れる]
次回は新入荷の〇〇フレーバーもぜひお試しください。
またのお越しをお待ちしています。
```

#### テンプレートC: グループ利用のお客様

```
〇〇様、グループでのご利用ありがとうございます！
皆さんで楽しい時間を過ごしていただけたようで嬉しいです。
貸切スペースならではのプライベート感を気に入っていただけて何よりです。
次回のグループ利用もお気軽にご予約ください。お待ちしています！
```

### ネガティブな口コミ（星1〜2）への返信

#### テンプレートD: サービスへの不満

```
〇〇様、ご来店いただきありがとうございます。
この度はご期待に沿えず、大変申し訳ございませんでした。
[具体的な不満点]について、スタッフ一同で改善に取り組んでまいります。
もしよろしければ、詳しい状況をお聞かせいただけると幸いです。
[メールアドレスまたは電話番号]までご連絡いただければ、
直接お話しさせていただきたく存じます。
貴重なご意見をいただき、ありがとうございました。
```

#### テンプレートE: 設備・環境への不満

```
〇〇様、ご来店ありがとうございます。
[具体的な指摘内容]についてご不便をおかけし、申し訳ございませんでした。
いただいたご指摘を踏まえ、[具体的な改善予定]を進めております。
改善後にまたお越しいただけましたら嬉しく思います。
ご意見ありがとうございました。
```

#### テンプレートF: 待ち時間・予約への不満

```
〇〇様、ご来店いただきありがとうございます。
お待たせしてしまい、大変申し訳ございませんでした。
予約管理の見直しを行い、同様のことが起きないよう改善いたします。
今後ともBiPシーシャをよろしくお願いいたします。
```

### 返信の基本原則

| ルール | 説明 |
|--------|------|
| 迅速に | 24〜48時間以内に返信 |
| 感謝を | ポジティブ・ネガティブ問わず最初に感謝を伝える |
| 具体的に | 口コミの内容に触れ、パーソナルな返信にする |
| 謙虚に | ネガティブな口コミには言い訳せず、改善姿勢を示す |
| 短すぎない | 「ありがとうございます。」だけの返信は逆効果 |
| 長すぎない | 3〜5文程度が適切 |
| キーワード | 返信に「シーシャ」「盛岡」等を自然に含めるとMEOにプラス |

## 3. Googleのガイドライン（やってはいけないこと）

### 絶対にやってはいけないこと

| 禁止行為 | リスク |
|---------|--------|
| 金銭・割引・無償提供と引き換えの口コミ依頼 | 口コミの一時非表示、新規投稿ブロック、警告表示 |
| 自作自演の口コミ（自分や知人による偽レビュー） | アカウント停止リスク |
| 口コミの内容を指定・誘導（「星5をつけてください」等） | ガイドライン違反 |
| 競合店へのネガティブ口コミ投稿 | 発覚時に重大なレピュテーションリスク |
| 口コミ代行業者の利用 | Googleが検出し、ペナルティ対象 |
| 否定的な口コミの削除を条件にした交渉 | ガイドライン違反 |

### グレーゾーンに注意

| 行為 | 判定 | 備考 |
|------|------|------|
| 「口コミを書いてください」と口頭で依頼 | OK | 内容の指定をしなければ問題なし |
| QRコードで口コミページに誘導 | OK | 推奨される方法 |
| 「口コミを書いたらドリンク1杯サービス」 | NG | 対価との引き換えは禁止 |
| 「口コミを書いてくれたら次回10%オフ」 | NG | 同上 |
| 友人に来店してもらい口コミを依頼 | OK | 実際に来店・体験した上での投稿であれば問題なし |
| 星の数を指定して依頼 | NG | 「率直な感想をお願いします」はOK |

### 不当な口コミへの対処

虚偽や悪意のある口コミを受けた場合:

1. **冷静に返信** する（感情的にならない）
2. Googleに **不適切な口コミとして報告** する（管理画面から）
3. 明らかなポリシー違反（来店していない、誹謗中傷等）の場合は削除申請
4. 削除されない場合でも、丁寧な返信で第三者からの印象を守る

## 4. 口コミ数と評価の目標設定

### フェーズ別目標

| フェーズ | 期間 | 口コミ数目標 | 平均評価目標 |
|---------|------|------------|------------|
| 立ち上げ | 開業〜1ヶ月 | 10件 | 4.5以上 |
| 初期成長 | 1〜3ヶ月 | 20件 | 4.3以上 |
| 安定期 | 3〜6ヶ月 | 40件 | 4.2以上 |
| 成熟期 | 6ヶ月〜1年 | 70件 | 4.0以上 |
| 長期 | 1年以上 | 月3〜5件ずつ増加 | 4.0以上を維持 |

### 盛岡の競合との比較目安

盛岡市内のシーシャ店は口コミ件数が数十件程度のところが多い。**50件を超えれば地域トップクラス** の可能性が高い。

### モニタリング項目

| 指標 | チェック頻度 | ツール |
|------|------------|--------|
| 口コミ件数 | 週1回 | Googleビジネスプロフィール管理画面 |
| 平均評価 | 週1回 | 同上 |
| 新着口コミの内容 | 毎日 | 通知設定をONにする |
| 返信率 | 月1回 | 全件返信を維持 |
| 競合店の口コミ動向 | 月1回 | Googleマップで確認 |

### 口コミから得られるインサイトの活用

口コミは「無料のユーザーリサーチ」でもある:

- 多く言及されるポジティブな点 → 強みとして打ち出す
- 繰り返し指摘されるネガティブな点 → 優先的に改善
- ユーザーが使うキーワード → GBPの説明文や投稿に反映
- 来店動機（「友達に勧められて」等） → マーケティング施策の参考に
