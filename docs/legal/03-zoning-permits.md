# 用途地域と営業許可

> 最終更新: 2026年3月
> 対象: BiPシーシャ（岩手県盛岡市・個人事業主運営）

---

## 1. 盛岡市の用途地域と出店可能エリア

### 1-1. 用途地域とは

都市計画法に基づき、土地の利用目的を定めたゾーニング。飲食店（シーシャカフェ含む）の出店可否は用途地域によって決まる。

### 1-2. 用途地域別の出店可否

| 用途地域 | 飲食店の出店 | 深夜酒類提供 | 備考 |
|----------|:------------:|:------------:|------|
| 第一種低層住居専用地域 | **不可** | 不可 | 住宅専用エリア |
| 第二種低層住居専用地域 | △（50m2以下） | 不可 | 小規模のみ |
| 第一種中高層住居専用地域 | △（500m2以下） | 不可 | 制限付き |
| 第二種中高層住居専用地域 | △（1,500m2以下） | 不可 | 制限付き |
| 第一種住居地域 | △（3,000m2以下） | 不可 | 制限付き |
| 第二種住居地域 | **可** | △（条例による） | |
| 準住居地域 | **可** | △（条例による） | |
| 田園住居地域 | **不可** | 不可 | |
| **近隣商業地域** | **可** | **可** | **出店に適している** |
| **商業地域** | **可** | **可** | **最も出店に適している** |
| 準工業地域 | **可** | **可** | 出店可能 |
| 工業地域 | **可** | △ | 飲食需要は少ない |
| 工業専用地域 | **不可** | 不可 | |

### 1-3. 盛岡市で狙うべきエリア

BiPシーシャのターゲット（岩手大学生）を考慮すると、以下のエリアが有力候補。

| エリア | 用途地域（想定） | 大学からのアクセス | 特徴 |
|--------|------------------|---------------------|------|
| 盛岡駅周辺 | 商業地域 | バス15分程度 | 集客力高いが賃料も高い |
| 大通り・菜園エリア | 商業地域・近隣商業地域 | バス20分程度 | 繁華街。飲食店多数 |
| 上田・岩大周辺 | 住居系地域が多い | 徒歩圏内 | 用途地域の確認が必須 |
| 本町・中央通り | 商業地域・近隣商業地域 | バス10分程度 | バランスが良い |

**重要**: 岩手大学周辺（上田地区）は住居系の用途地域が多いため、出店前に必ず用途地域を確認すること。

### 1-4. 用途地域の確認方法

1. **盛岡市都市計画課に問い合わせ**
   - 電話: 019-626-7524（盛岡市都市整備部都市計画課）
   - 窓口で住所を伝えれば用途地域を教えてもらえる

2. **盛岡市の都市計画情報サービス（Web）**
   - 盛岡市のWebサイトで都市計画図を閲覧できる場合がある
   - 「盛岡市 都市計画図」で検索

3. **物件の不動産業者に確認**
   - 物件の重要事項説明書に用途地域が記載されている

## 2. 飲食店営業許可の取得手順（保健所）

### 2-1. 概要

シーシャカフェで飲み物や軽食を提供する場合、食品衛生法第55条に基づく**飲食店営業許可**が必要。

| 項目 | 内容 |
|------|------|
| **申請先** | 盛岡市保健所 生活衛生課 食品衛生担当 |
| **所在地** | 〒020-0884 盛岡市神明町3-29 盛岡市保健所5階 |
| **電話** | 019-603-8311 |
| **手数料** | 約16,000円（自治体により異なる。盛岡市に要確認） |
| **許可の有効期間** | 原則5〜8年（施設の構造等による） |

### 2-2. 取得の流れ

```
1. 事前相談（保健所に店舗の図面を持参して相談）
   ↓
2. 施設の工事・設備の整備
   ↓
3. 営業許可申請書の提出
   ↓
4. 施設検査（保健所職員が店舗を検査）
   ↓  チェック項目:
   ↓  - 手洗い設備（専用の流し）
   ↓  - 調理場と客席の区画
   ↓  - 冷蔵・冷凍設備
   ↓  - 食器洗浄設備
   ↓  - 換気設備
   ↓  - 防虫・防鼠設備
   ↓  - 床・壁・天井の材質（清掃しやすい材質）
   ↓  - トイレの設置・手洗い
   ↓
5. 許可証交付
   ↓
6. 営業開始
```

### 2-3. 必要書類

1. 営業許可申請書
2. 施設の構造及び設備を示す図面（平面図）
3. 食品衛生責任者の資格を証する書類
4. 水質検査成績書（水道水以外を使用する場合）
5. 登記事項証明書（法人の場合。個人事業主は不要）
6. 手数料

### 2-4. 施設基準の主なポイント

| 設備 | 基準 |
|------|------|
| **手洗い設備** | 調理場に専用の手洗い器（自動式・レバー式の蛇口が望ましい）。ハンドソープ・消毒液・ペーパータオルを常備 |
| **調理場の床・壁** | 耐水性のある材質（タイル・ステンレス等）。排水設備あり |
| **食器の収納** | 戸付きの収納棚 |
| **冷蔵庫** | 温度計付き |
| **トイレ** | 手洗い設備付き。調理場から離れた場所に設置 |

**ポイント**: シーシャカフェは軽食・ドリンクのみの提供であっても、飲食店営業許可が必要。ドリンクのみ（チャイ・ジュース等）でも同様。

## 3. 食品衛生責任者の取得方法

### 3-1. 概要

飲食店営業許可を取得するには、**食品衛生責任者**を1名以上設置する必要がある。店舗運営者自身が取得するのが一般的。

### 3-2. 資格の取得方法

| 方法 | 内容 | 費用 | 期間 |
|------|------|------|------|
| **養成講習会を受講** | 1日（約6時間）の講習を受講して修了 | 約10,000円 | 1日 |
| **eラーニング** | オンラインで受講可能（岩手県対応済み） | 約10,000円 | 自分のペースで |
| **資格免除** | 調理師・栄養士・製菓衛生師等の有資格者は免除 | - | - |

### 3-3. 岩手県での受講先

- **主催**: 一般社団法人 岩手県食品衛生協会
- **県央支会**（盛岡市管轄）が毎月講習会を開催
- **Webサイト**: https://iwasyoku.jp/kousyukai
- **eラーニング**: 日本食品衛生協会のeラーニングシステムでも受講可能
  - https://www.n-shokuei.jp/topics/elearning/e_yousei.html

### 3-4. 講習内容

| 科目 | 時間 |
|------|------|
| 衛生法規 | 2時間 |
| 公衆衛生学 | 1時間 |
| 食品衛生学 | 3時間 |
| **合計** | **6時間** |

講習会の最後に確認テストがあるが、落ちることはほぼない。

## 4. 深夜営業の届出（風営法）

### 4-1. 深夜酒類提供飲食店営業届

**深夜0時以降に酒類を提供する**場合は、風俗営業等の規制及び業務の適正化等に関する法律（風営法）に基づく届出が必要。

| 項目 | 内容 |
|------|------|
| **届出名** | 深夜における酒類提供飲食店営業営業開始届出書 |
| **届出先** | 岩手県警察本部 生活安全課（管轄警察署経由） |
| **届出時期** | 営業開始の10日前まで |
| **届出費用** | 無料 |

### 4-2. BiPシーシャでの判断基準

| 営業形態 | 届出の要否 |
|----------|:----------:|
| シーシャのみ提供（酒なし）、深夜営業あり | **不要** |
| ドリンク（ノンアルコールのみ）、深夜営業あり | **不要** |
| 酒類を提供、24時前に閉店 | **不要** |
| **酒類を提供、24時以降も営業** | **必要** |

**ポイント**: シーシャカフェとして酒類を提供しない場合は、深夜営業の届出は不要。酒類を提供する場合でも24時前に閉店すれば不要。

### 4-3. 深夜営業届出が必要な場合の注意点

- **用途地域の制限**: 住居系地域では深夜酒類提供飲食店の営業が禁止されている場合が多い
- **客室の面積**: 客室の面積が9.5m2以上であること
- **客室内の見通し**: 客室内の見通しを妨げる設備を設けないこと（高さ1m以上の仕切り等）
- **騒音・振動**: 条例で定める基準を超えないこと

### 4-4. 風俗営業許可との違い

シーシャカフェは通常、風俗営業許可は不要。ただし、以下に該当する場合は風営法の1号営業（社交飲食店）の許可が必要になる可能性がある。

- 接待行為を行う（特定の客の近くに座って会話する等）
- 照明が10ルクス以下の暗い店内

シーシャカフェは「客がシーシャを吸いながらくつろぐ」業態であり、接待行為は通常行わないため、風俗営業許可は不要と考えてよい。

## 5. 改正健康増進法と喫煙目的施設

### 5-1. シーシャカフェの法的位置づけ

2020年4月施行の改正健康増進法により、飲食店は原則屋内禁煙。ただし、シーシャカフェは**「喫煙目的施設」**として例外的に屋内喫煙が認められる。

### 5-2. 喫煙目的施設の要件

| 要件 | 内容 |
|------|------|
| **主たる目的** | たばこの対面販売（シーシャの提供）が主たる業務であること |
| **たばこの販売許可** | 製造たばこ小売販売業許可または出張販売許可があること |
| **設備基準** | 出入口の気流0.2m/秒以上、壁・天井で区画、屋外排気 |
| **標識の掲示** | 「喫煙目的室」の標識を出入口に掲示すること |
| **従業員への配慮** | 受動喫煙を望まない従業員に配慮すること |
| **20歳未満立入禁止** | 20歳未満の者を立ち入らせてはならない |

### 5-3. 重要な注意点

**20歳未満の者は入店不可**。岩手大学1・2年生の一部は20歳未満であるため、年齢確認を徹底すること。身分証の提示を必須とする運用が必要。

## 6. 実務上のアクションリスト

| # | やること | 時期 | 備考 |
|---|---------|------|------|
| 1 | 出店候補エリアの用途地域を確認 | 物件探し開始時 | 盛岡市都市計画課に確認 |
