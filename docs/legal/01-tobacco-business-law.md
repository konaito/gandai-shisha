# たばこ事業法とシーシャの関係

> 最終更新: 2026年3月
> 対象: BiPシーシャ（岩手県盛岡市・個人事業主運営）

---

## 1. シーシャの法的分類

シーシャ（水たばこ）のフレーバーは、日本の法律上「**製造たばこ**」に分類される。具体的には「**パイプたばこ**」のカテゴリに該当し、たばこ事業法の規制対象となる。

したがって、シーシャカフェでフレーバーを客に提供する行為は「**たばこの小売販売**」に該当し、無許可で行うと**たばこ事業法違反（罰則あり）**となる。

---

## 2. 許可の種類と比較

シーシャカフェを開業するには、以下の2つのルートがある。

| 項目 | 製造たばこ小売販売業許可 | たばこ出張販売許可 |
|------|--------------------------|---------------------|
| **根拠法** | たばこ事業法 第22条・第23条 | たばこ事業法 第26条 |
| **申請者** | 店舗運営者（自分） | たばこ小売販売業者（仕入先） |
| **距離制限** | あり（繁華街50m / 市街地100m / 住宅地200m） | **なし** |
| **審査期間** | 約2ヶ月 | 約2ヶ月 |
| **登録免許税** | 15,000円 | 3,000円（1件あたり） |
| **申請先** | 管轄の財務局（東北財務局） | JT支社経由で財務局 |
| **メリット** | 自分で仕入先を自由に選べる | 距離制限なし、手続きが比較的シンプル |
| **デメリット** | 距離制限で不許可になるリスクがある | 仕入先（出張元）との契約関係が必要 |

### どちらを選ぶべきか

**実務上、多くのシーシャカフェは「たばこ出張販売許可」を利用している。** 理由は以下の通り。

- 距離制限がないため、立地を選ばず許可が取れる
- シーシャフレーバーの卸業者が出張販売許可の取得をサポートしてくれることが多い
- 登録免許税も安い（3,000円 vs 15,000円）

ただし、出張販売許可はあくまで仕入先（たばこ小売販売業者）が申請するものであり、自分自身が許可を取得するわけではない点に注意。仕入先との関係が途切れると営業できなくなるリスクがある。

---

## 3. 製造たばこ小売販売業許可の申請手順

自分で小売販売業許可を取得する場合の手順。

### 3-1. 申請先

- **東北財務局**（岩手県を管轄）
- JT（日本たばこ産業株式会社）の東北支社が窓口となる

### 3-2. 必要書類

1. 製造たばこ小売販売業許可申請書
2. 店舗の位置図（周辺地図）
3. 店舗の平面図
4. 住民票の写し（申請者）
5. 身分証明書（市区町村長発行の破産者でないことの証明）
6. 店舗の賃貸借契約書の写し（賃貸の場合）
7. 登録免許税 15,000円

### 3-3. 許可基準（主なもの）

- **距離基準**: 最寄りのたばこ販売店から所定の距離が離れていること
  - 繁華街: 50m以上
  - 市街地: 100m以上
  - 住宅地: 200m以上
- **取扱予定高基準**: 月間の販売予定数量が一定基準を満たすこと
- **申請者の適格性**: 未成年者でないこと、破産者でないこと等

### 3-4. 審査の流れ

```
申請書提出 → JT支社が現地調査 → 財務局が審査 → 許可/不許可の通知
                                （約2ヶ月間）
```

---

## 4. たばこ出張販売許可の取得手順

多くのシーシャカフェが選択するルート。

### 4-1. 前提条件

- シーシャフレーバーの仕入先が「製造たばこ小売販売業」の許可を持っていること
- その仕入先が、自店舗を「出張販売先」として申請してくれること

### 4-2. 流れ

```
1. シーシャフレーバー卸業者と取引契約
2. 卸業者がJT支社に出張販売許可を申請（出張先＝自店舗の住所）
3. JT支社が現地調査
4. 財務局が審査（約2ヶ月）
5. 許可通知
6. 営業開始
```

### 4-3. 注意点

- 出張販売許可はあくまで仕入先の許可であり、自分の許可ではない
- 仕入先を変更する場合は、新たな仕入先が再度出張販売許可を取得する必要がある
- 許可が下りるまでの約2ヶ月間はシーシャの提供ができない

## 5. たばこ税の仕組み

### 5-1. シーシャフレーバーにかかる税

シーシャフレーバーは「パイプたばこ」に分類され、**重量ベース**で課税される。

| 税目 | 課税単位 | 税率 |
|------|----------|------|
| 国たばこ税 | 1gあたり | 紙巻たばこ1本に換算して課税 |
| 地方たばこ税（道府県） | 1gあたり | 同上 |
| 地方たばこ税（市町村） | 1gあたり | 同上 |
| たばこ特別税 | 1gあたり | 同上 |
| 消費税 | 販売価格 | 10% |

**パイプたばこの換算方法**: パイプたばこ1g ＝ 紙巻たばこ1本として換算される。

### 5-2. 価格への影響

シーシャフレーバーの小売価格には、すでにたばこ税が含まれている。仕入れ時に税込み価格で購入するため、店舗側が別途納税手続きをする必要は基本的にない。ただし、仕入原価に税が含まれているため、原価率の計算時に留意が必要。

## 6. 2024-2026年の法改正動向

### 加熱式たばこの課税方式見直し

- **令和8年（2026年）4月**: 加熱式たばこの新換算方式が5割適用開始
- **令和8年（2026年）10月**: 新換算方式が10割適用
- **令和9年〜11年（2027〜2029年）**: 国たばこ税が毎年0.5円/本ずつ段階的に引き上げ

**シーシャへの影響**: 上記改正は主に加熱式たばこ（iQOS等）を対象としており、パイプたばこ（シーシャフレーバー）への直接的な税率変更は現時点では予定されていない。ただし、たばこ税全体の引き上げトレンドの中で、将来的にパイプたばこの税率が見直される可能性は否定できない。

### 受動喫煙防止関連

健康増進法の強化に伴い、喫煙規制が年々強まっている。シーシャカフェは「喫煙目的施設」として営業する必要があるが、自治体ごとの条例による追加規制の可能性がある。岩手県の動向を定期的にチェックすること。

## 7. 実務上のアクションリスト

| # | やること | 時期 | 備考 |
|---|---------|------|------|
| 1 | シーシャフレーバー卸業者を選定・契約 | 物件契約前〜契約直後 | 出張販売許可のサポート体制を確認 |
| 2 | 出張販売許可の申請（卸業者側） | 物件契約後すぐ | 審査に約2ヶ月かかるため早めに |
| 3 | 許可通知の受領 | 申請から約2ヶ月後 | 許可前にシーシャ提供は不可 |
| 4 | 店舗に許可証を掲示 | 営業開始時 | 法律上の義務 |
