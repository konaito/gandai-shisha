# 賃貸交渉テクニック

## 1. 家賃交渉のポイント

### 交渉の前提知識

盛岡市のテナント市場は東京や仙台と比べて圧倒的に供給過多。空室率が高いエリアでは、オーナー側も「埋まらないよりは安くても入ってほしい」というマインドになりやすい。これは交渉の追い風。

### 具体的な交渉材料

| 交渉材料 | 使い方 |
|---------|--------|
| **空室期間** | 「この物件、○ヶ月空いてますよね」→ 長期間空いているほど交渉力UP |
| **周辺相場との比較** | 「近くの同条件物件が○万円なんですが」→ 具体的な物件名を出す |
| **長期契約の約束** | 「3年以上借りる予定です」→ 安定収入をアピール |
| **即決の意思** | 「条件が合えば今月中に契約します」→ オーナーの意思決定を促す |
| **内装投資の提示** | 「○万円の内装投資をするので、簡単には出ていきません」→ 長期入居の信頼性 |

### 交渉の進め方

1. **最初の内見では交渉しない**: まず物件を見て、気に入ったら「前向きに検討します」で帰る
2. **2回目の接触で条件提示**: 「この条件なら契約します」と具体的に提示
3. **一度に全部交渉しない**: 家賃→フリーレント→敷金の順で、一つずつ交渉
4. **仲介業者を味方につける**: 仲介業者は契約が成立しないと手数料が入らない。「家賃が下がれば契約する」と伝えれば、業者がオーナーを説得してくれる

### 盛岡の家賃交渉の相場感

- **家賃値引き**: 表示価格の5〜15%が現実的なライン（5.5万円→5万円程度）
- **ただし**: 元々安い物件（5万円台）はこれ以上の値引きは厳しい。フリーレントで交渉するほうが効果的

## 2. フリーレント（無料期間）の交渉

### フリーレントとは

契約後、一定期間の家賃が免除される仕組み。内装工事期間中は営業できないため、この期間の家賃をゼロにしてもらう交渉。

### 交渉の根拠

「内装工事に○週間かかるため、工事期間中は営業できません。この期間のフリーレントをお願いできませんか」

- **1ヶ月のフリーレント**: 比較的通りやすい。工事期間として合理的な説明が可能
- **2ヶ月のフリーレント**: 工事＋プレオープン準備として交渉可能
- **3ヶ月以上**: 空室期間が長い物件なら可能性あり

### 交渉のコツ

- **工事計画書を見せる**: 「換気ダクト工事に2週間、内装仕上げに1週間かかります」と具体的に
- **オーナーのメリットを強調**: 「工事で物件の価値が上がります」「退去後もダクトは残るので次のテナントも使えます」
- **譲歩と組み合わせる**: 「家賃は提示通りでOKなので、代わりにフリーレント1ヶ月をお願いします」

### フリーレントの経済効果

家賃5.5万円の物件でフリーレント1ヶ月を獲得 = **5.5万円の節約**。これは実質的に1ヶ月分の家賃値引きと同じ効果。

## 3. 敷金・礼金・保証金の相場

### 盛岡市テナントの初期費用

| 項目 | 相場 | 備考 |
|------|------|------|
| **敷金** | 家賃の1〜3ヶ月分 | テナントは住居より高め。退去時に原状回復費と相殺 |
| **礼金** | 0〜1ヶ月分 | 最近は0の物件も多い |
| **保証金** | 家賃の3〜6ヶ月分 | 敷金の代わりに保証金を設定する場合あり |
| **仲介手数料** | 家賃の1ヶ月分 | 法定上限。交渉余地は少ない |
| **前家賃** | 1ヶ月分 | 契約月の翌月分 |

### 初期費用のシミュレーション（家賃5.5万円の場合）

| 項目 | 金額 |
|------|------|
| 敷金（2ヶ月） | 11万円 |
| 礼金（0） | 0円 |
| 仲介手数料（1ヶ月） | 5.5万円 |
| 前家賃（1ヶ月） | 5.5万円 |
| 火災保険 | 1〜2万円 |
| **合計** | **23〜24万円** |

> サニーフラットは敷金0・礼金0のため、初期費用が大幅に低い。これは「撤退のしやすさ」にも直結する。

### 敷金・保証金の交渉

- **保証金を自ら多めに提示する**: 「シーシャ営業で不安があると思うので、保証金を通常の1.5倍お預けします」→ オーナーの不安を先回りして解消。信頼感が大幅にUP
- **分割払いの交渉**: 「保証金は2回に分けてお支払いできませんか」→ 初期のキャッシュフローを改善

## 4. シーシャ営業の許可を得るための交渉術

### オーナーが抱える不安と対策

シーシャ営業は一般的なテナント利用に比べて、オーナーに特有の不安を与える。これを先回りして解消することが契約成立の鍵。

| オーナーの不安 | 対策 |
|--------------|------|
| **火災リスク** | 防火シート＋消火器＋炭火対応火災保険の証書を見せる |
| **煙・臭い汚損** | 換気設備の設計図を提示＋退去時クロス全面張替えを自己負担と提案 |
| **近隣苦情** | 排気口の位置計画＋開業前の近隣挨拶を約束 |
| **深夜の騒音** | 営業時間の制限を自ら提案（例: 〜24時） |
| **学生が信用できない** | 事業計画書を持参。親の連帯保証も提示 |

### 事業計画書に含めるべき内容

オーナーや不動産業者に見せる簡易版の事業計画書:

1. **事業概要**: 完全予約制の貸切シーシャスペース
2. **ターゲット**: 岩手大学生（20歳以上限定）
3. **営業時間**: 例）13:00〜24:00
4. **安全対策**: 換気設備の設計、CO検知器、消火器、火災保険
5. **原状回復の約束**: 退去時のクロス張替え・クリーニングを自己負担
6. **収支計画**: 月の売上見込みと家賃支払い能力の根拠

### 「シーシャ」の説明の仕方

盛岡のオーナー世代（50〜70代）はシーシャを知らない可能性が高い。

- **NG**: 「シーシャカフェをやりたいんです」（？？？となる）
- **OK**: 「水タバコの喫煙スペースを運営したいと考えています。タバコの煙を水で冷やして吸う中東由来の喫煙具で、最近若者の間で人気です。完全予約制で、換気設備もしっかり整えます」

### 契約形態の選択

| 形態 | メリット | デメリット |
|------|---------|-----------|
| **個人契約** | 手続きが簡単 | 信用度が低い |
| **法人契約** | 信用度UP。経費処理が明確 | 法人設立コスト（合同会社で約6万円） |

## 5. 契約書チェックポイント

### 必ず確認すべき条項

#### 用途制限

- [ ] 「喫煙を伴う営業」が許可されているか
- [ ] 「飲食店」または「サービス業」として利用可能か
- [ ] 用途変更に関する条項がないか

#### 火気使用

- [ ] 炭火の使用が明示的にOKか
- [ ] 「火気厳禁」条項がないか（あったら交渉して削除 or 例外追加）
- [ ] IHヒーター（炭起こし用）の使用は問題ないか

#### 工事関連

- [ ] ダクト工事（壁の穴あけ）の許可
- [ ] 電気工事の許可
- [ ] 内装変更の範囲と条件
- [ ] 工事前のオーナー承認プロセス

#### 原状回復

- [ ] 原状回復の範囲（どこまで元に戻すか）
- [ ] 費用上限の設定があるか
- [ ] 換気ダクトは残置可能か（残したほうがオーナーにもメリットがある場合）
- [ ] 原状回復の査定方法（第三者見積もりを取れるか）

#### 契約期間・解約

- [ ] 契約期間（2年が一般的）
- [ ] 中途解約の条件（何ヶ月前通知か）
- [ ] 解約違約金の有無
- [ ] 更新条件（家賃改定の可能性）

#### その他

- [ ] 転貸（サブリース）の可否
- [ ] 営業時間の制限
- [ ] 看板・外装変更の可否
- [ ] 共用部分（トイレ・駐車場等）の利用条件

### 契約前の最終チェック

1. 契約書は必ず持ち帰って確認する（その場でサインしない）
2. 不明な条項があれば、盛岡市の無料法律相談を利用（岩手弁護士会: 019-651-5095）
3. 口頭の約束は必ず書面化する（「シーシャ営業OK」を契約書に明記）
4. 特約事項として「喫煙を伴う営業を許可する」旨を追記

## 交渉の心構え

- **学生だからと卑屈にならない**: テナントを借りてくれる人は、オーナーにとっても貴重
- **準備が9割**: 事業計画書、安全対策資料、収支計画を持参するだけで印象が変わる
- **「No」は交渉の始まり**: 最初に断られても、条件を変えて再提案すればOKが出ることも多い
- **複数の候補を持つ**: 「ここしかない」と思うと交渉力が下がる。常に複数の候補を持つこと
