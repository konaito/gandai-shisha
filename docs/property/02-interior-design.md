# 内装工事ガイド

## 1. 居抜き vs スケルトンの比較

### 居抜き物件

前のテナントの内装・設備がそのまま残っている物件。

| 項目 | 内容 |
|------|------|
| **メリット** | 工事費が大幅に安い。ダクトが残っていれば換気工事費50〜100万円で済む。即営業開始できる可能性 |
| **デメリット** | レイアウトの自由度が低い。前テナントの汚れ・匂いが残る場合がある。設備の老朽化リスク |
| **向いているケース** | 飲食店・バー・スナックの居抜き。換気設備が既存なら最高 |

### スケルトン物件

コンクリート打ちっぱなしの何もない状態。

| 項目 | 内容 |
|------|------|
| **メリット** | 完全に自由な設計が可能。清潔な状態からスタート |
| **デメリット** | 内装工事費が高い（換気だけで80〜150万円）。工事期間中も家賃が発生 |
| **向いているケース** | 予算に余裕がある場合。こだわりの空間を作りたい場合 |

### BiPシーシャの結論

**居抜き物件を最優先で探す。** 初期投資を最小化し、「撤退のしやすさ」を確保することが大学生起業の生命線。特に飲食店・バーの居抜きはダクトが残っている可能性が高く、最大のコスト削減ポイント。

## 2. シーシャスペースに必要な設備工事

### 換気設備（最重要・最優先）

シーシャ炭はCO（一酸化炭素）を大量に発生させる。5年半で全国64件のCO中毒事故が報告されている。換気は安全上も営業継続上も最優先事項。

#### 必要スペック（30〜40m²の場合）

| 項目 | 必要スペック |
|------|------------|
| 必要排気量 | 800〜1,200 m³/h 以上 |
| 換気回数 | 10〜15回/時間 |
| 入口風速 | 0.2m/s以上（法的基準） |

#### 換気設計の基本

```
┌──────────────────────────────────────┐
│  【天井付近】                          │
│  排気ファン ×2〜3台 → ダクト → 屋外排気  │
│                                        │
│     ソファ    ソファ    ソファ           │
│     シーシャ台  シーシャ台  シーシャ台    │
│                                        │
│  【壁面下部】                          │
│  ←── 自然給気口（入口付近）             │
│                                        │
│  CO検知器 ×2台                         │
└──────────────────────────────────────┘
  ※ 排気口と給気口は対角線上に配置
```

- 排気は天井付近（煙は上に溜まる）
- 給気は壁面下部（入口付近）
- 排気口と給気口は対角線上に配置して空気の流れを作る
- **CO検知器は必ず2台以上設置**（1台は天井付近、1台はソファ高さ）

#### 換気工事の費用

| パターン | 費用 | 備考 |
|---------|------|------|
| **居抜き（ダクト既存）** | **50〜100万円** | これを狙う |
| スケルトン（壁出し排気） | 80〜150万円 | 2026年の資材高騰込み |
| 地下物件 | 200万円〜 | 避けるべき |

### 電気工事

| 項目 | 内容 | 費用目安 |
|------|------|---------|
| 分電盤増設 | 排気ファン・エアコン・照明の電力確保。30A以上推奨 | 5〜15万円 |
| コンセント増設 | シーシャ炭用IHヒーター（炭起こし用）、照明、音響 | 3〜8万円 |
| 照明設置 | 間接照明・ダウンライト。雰囲気づくりの要 | 5〜20万円 |

### 給排水設備

飲食店営業許可を取得する場合は、手洗い場（従業員用）とシンクが必須。

| 項目 | 内容 | 費用目安 |
|------|------|---------|
| 手洗い場設置 | 保健所の基準を満たす手洗い設備 | 10〜20万円 |
| シンク設置 | シーシャのボウル・ホース洗浄用 | 5〜15万円 |
| 既存設備の改修 | 居抜きの場合、既存の水回りを活用 | 5〜10万円 |

> **注意**: 飲食店営業許可が不要と確認できた場合、大掛かりな給排水工事は不要になる可能性がある。保健所への確認を先にすること。

### 防音工事

完全貸切のため、グループ利用時の会話・音楽が隣接テナントや上階に漏れないよう対策が必要。

| 対策 | 費用目安 | 効果 |
|------|---------|------|
| 吸音パネル（壁面） | 3〜10万円 | 中（DIY可能） |
| 防音カーテン | 1〜3万円 | 低〜中 |
| 床材（防音マット） | 2〜5万円 | 中（1Fなら不要） |
| 防音ドア | 10〜30万円 | 高（予算があれば） |

## 4. DIYでコストを下げる方法

### DIYに向いている工事

| 工事内容 | 難易度 | 削減額目安 | ポイント |
|---------|-------|-----------|---------|
| **壁の塗装** | 低 | 5〜10万円 | ペンキ代のみ。養生をしっかり |
| **間接照明の設置** | 低 | 3〜8万円 | LEDテープライト。電気工事資格不要の範囲で |
| **家具の組立・配置** | 低 | 0円（工賃節約） | IKEAやニトリの家具を自分で搬入・組立 |
| **吸音パネルの取付** | 低〜中 | 3〜5万円 | 壁にビスで固定するだけ |
| **クッションフロア貼り** | 中 | 3〜5万円 | 接着剤で貼るだけだが仕上がりに差が出る |

### DIYに向かない工事（プロに任せるべき）

| 工事内容 | 理由 |
|---------|------|
| **換気ダクト工事** | 安全に直結。設計ミスはCO中毒事故につながる |
| **電気工事（分電盤・コンセント増設）** | 電気工事士の資格が必要 |
| **給排水工事** | 水漏れリスク。建物への損害賠償問題 |
| **ガス工事** | 法律上、有資格者のみ |

### 盛岡でDIY資材を調達できる場所

- **コメリ（盛岡西バイパス店等）**: 塗料・工具・建材が揃う
- **DCMホーマック**: 同上。品揃えが広い
- **ジモティー**: 中古家具・設備。盛岡エリアは出品が少ないが、仙台まで広げれば見つかる
- **メルカリ/ラクマ**: ソファ・照明などの中古品。送料に注意
- **IKEA（仙台）**: 盛岡から車で約2時間。ソファ・収納・照明を一括購入

## 5. 工事業者の選び方

### 盛岡で内装工事を頼む場合の選択肢

| 業者タイプ | メリット | デメリット | 費用感 |
|-----------|---------|-----------|-------|
| **地元の内装業者** | 地域の事情に詳しい。アフターフォローが手厚い | 見つけるのが大変 | 相場通り |
| **プロワイズ等の開業サポート会社** | 内装・許認可・融資をワンストップ | 手数料が乗る | やや高め |
| **リフォーム会社** | 住宅リフォームの延長で対応可能 | 店舗の専門知識が薄い場合あり | 相場通り |
| **飲食店の内装専門業者（仙台）** | シーシャ・バーの実績がある可能性 | 出張費がかかる | 高め |

### 業者選定のポイント

1. **相見積もりは必ず3社以上**: 盛岡は業者数が限られるが、仙台の業者も含めて比較
2. **換気設備の実績を確認**: シーシャ店またはバー・焼肉店の換気工事実績があるか
3. **見積書の明細を確認**: 「一式○万円」ではなく、項目ごとの内訳がある業者を選ぶ
4. **工期を確認**: 工事中もテナント家賃は発生する。工期が長いほど無駄な固定費が増える
5. **保証・アフターサポート**: 換気設備の不具合は命に関わる。保証期間を確認

### 業者への依頼手順

1. 物件が決まったら、内見に業者を同行させる（現場確認）
2. 現場確認後、見積もりを依頼
3. 3社以上の見積もりを比較
4. 工事内容・工期・支払い条件を確定
5. 着工 → 中間確認 → 完了検査
6. 保健所の立入検査（飲食店営業許可が必要な場合）

## 内装工事のタイムライン

| フェーズ | 期間 | 内容 |
|---------|------|------|
| 設計・見積もり | 2〜3週間 | 業者選定、相見積もり、設計確定 |
| 換気工事 | 1〜2週間 | ダクト設置、排気ファン設置 |
| 電気工事 | 3〜5日 | 分電盤、コンセント、照明 |
| 給排水工事 | 3〜5日 | 手洗い、シンク（必要な場合） |
| 内装仕上げ | 1週間 | 壁面、床、防音（一部DIY可） |
| 家具搬入・設置 | 2〜3日 | ソファ、テーブル、シーシャ台 |
| 検査・調整 | 3〜5日 | CO検知器テスト、換気量確認、保健所検査 |
| **合計** | **約1〜1.5ヶ月** | 居抜きなら2〜3週間に短縮可能 |

> **重要**: 工事期間中もテナント家賃は発生する。フリーレント（工事期間中の家賃免除）を交渉すること。詳しくは `03-lease-negotiation.md` を参照。
