# LINE公式アカウント設定ガイド

BiPシーシャのLINE公式アカウントを開設し、集客・予約の中核チャネルとして機能させるための設定ガイド。

## 1. アカウント作成手順

### 1-1. 事前準備

| 項目 | 内容 |
|------|------|
| 必要なもの | LINEアカウント（個人）、メールアドレス |
| 管理画面URL | https://manager.line.biz/ |
| スマホアプリ | 「LINE公式アカウント」アプリ（iOS/Android） |
| 所要時間 | 約5分 |

### 1-2. 開設ステップ

1. [LINE公式アカウント作成ページ](https://www.lycbiz.com/jp/service/line-official-account/)にアクセス
2. 「LINE公式アカウントをはじめる」をクリック
3. LINEアカウントまたはメールアドレスでログイン
4. アカウント情報を入力:
   - **アカウント名**: `BiPシーシャ`（後から変更可能）
   - **業種**: 大業種「グルメ・飲食」→ 小業種「バー」または「カフェ」
   - **会社/事業者名**: 任意
5. 入力内容を確認し「アカウントを作成」

### 1-3. 認証済みアカウント申請（推奨）

認証済みアカウント（青バッジ）を取得すると以下のメリットがある:

- LINE内検索結果に表示される
- 友だち追加広告が利用可能
- 請求書払いに対応
- 信頼性が向上

> **注意**: 認証審査には営業許可証などが必要。物件確定・営業許可取得後に申請する。

## 2. プロフィール最適化

### 2-1. 基本設定

| 設定項目 | 推奨内容 |
|----------|----------|
| アカウント名 | `BiPシーシャ｜岩手大学前の貸切シーシャ` |
| アイコン画像 | ロゴまたはシーシャの写真（640×640px推奨） |
| カバー画像 | 店内の雰囲気が伝わる写真（1080×878px推奨） |
| ステータスメッセージ | `🎓岩大生向け貸切シーシャ｜完全予約制` |

### 2-2. プロフィールページの充実

プロフィールページはLINE内のミニWebサイトとして機能する。以下を設定:

- **基本情報**
  - 営業時間（例: 14:00〜23:00）
  - 住所（Googleマップ連動）
  - 電話番号
  - Webサイト: `https://shisha.konaito.com`
- **プラグイン（パーツ）**
  - 「ショップカード」パーツ
  - 「クーポン」パーツ
  - 「テキスト」パーツでコンセプト説明
  - SNSリンク（Instagram: @konaito5）

## 3. リッチメニューの設計

リッチメニューはトーク画面下部に常時表示されるメニューで、最もタップされるUI要素。

### 3-1. メニュー構成案

**6分割テンプレート（大）推奨**:

```
┌──────────┬──────────┬──────────┐
│  📅 予約  │  💰 料金  │  📍 場所  │
├──────────┼──────────┼──────────┤
│ 🎫 クーポン│ 📱 SNS   │ ❓ よくある│
│          │          │    質問   │
└──────────┴──────────┴──────────┘
```

| ボタン | アクション | リンク先 |
|--------|-----------|---------|
| 予約 | URL | 予約フォームURL or トーク内で予約メッセージ送信 |
| 料金 | URL | `https://shisha.konaito.com#pricing` |
| 場所 | URL | Googleマップリンク |
| クーポン | クーポン | LINE公式のクーポン機能 |
| SNS | URL | Instagramプロフィール |
| よくある質問 | テキスト送信 | 自動応答で回答 |

### 3-2. デザイン作成

- **Canva**の無料テンプレートを活用（「LINEリッチメニュー」で検索）
- 画像サイズ: 2500×1686px（大）or 2500×843px（小）
- ブランドカラーを統一し、ボタンだとわかるデザインにする
- テキストは大きめ・読みやすいフォントで

### 3-3. 設定手順

1. LINE Official Account Manager → トークルーム管理 → リッチメニュー
2. 「作成」をクリック
3. タイトル・表示期間を設定
4. テンプレート選択 → 画像アップロード → アクション設定
5. 「デフォルトで表示」を**オン**にする

## 4. 自動応答メッセージの設定

### 4-1. あいさつメッセージ（友だち追加時）

友だち追加直後に自動送信されるメッセージ。第一印象を決める重要なタッチポイント。

**テンプレート例**:

```
友だち追加ありがとう！🙌
BiPシーシャは岩手大学近くの貸切シーシャスペースです。

📅 予約はこのトークから！
💰 学割料金あり
🎫 友だち追加特典クーポンはこちら↓

下のメニューから気になる項目をタップしてね！
```

> **ポイント**: 大学生向けなのでフランクな口調でOK。メッセージは3吹き出し以内に収める（課金対象になるため）。

### 4-2. キーワード応答

特定のキーワードに対して自動返信する設定:

| キーワード | 応答内容 |
|-----------|---------|
| `予約` `予約したい` | 予約フォームURLまたは予約手順の案内 |
| `料金` `値段` `いくら` | 料金表テキスト + Webサイトリンク |
| `場所` `住所` `アクセス` | 住所 + Googleマップリンク |
| `営業時間` `何時` | 営業時間の案内 |
| `メニュー` `フレーバー` | フレーバーラインナップの案内 |
| `キャンセル` | キャンセルポリシーの案内 |

### 4-3. 応答モード設定

| モード | 用途 |
|--------|------|
| **チャット + 自動応答** | 推奨。普段は自動応答、必要に応じて手動返信 |
| Bot（自動応答のみ） | 手が離せないとき用 |
| チャット（手動のみ） | 自動応答を一時停止したいとき |

> **1人運営のコツ**: 基本は「スマートチャット」モード（自動応答 + 手動チャット併用）にしておき、営業時間外は自動応答のみで対応する。

## 5. 友だち追加導線の設計

### 5-1. オンライン導線

| 施策 | 具体的なアクション |
|------|-------------------|
| Webサイト | `shisha.konaito.com` にLINE友だち追加ボタン設置 |
| Instagram | プロフィールリンク・ストーリーズにLINEリンク |
| X (Twitter) | 固定ツイートにLINEリンク |
| QRコードシェア | SNS投稿用のQRコード画像を作成 |

### 5-2. オフライン導線

| 施策 | 具体的なアクション |
|------|-------------------|
| 店内POP | QRコード付きカードをテーブルに設置 |
| 名刺・チラシ | QRコード印刷 |
| 大学構内 | 学食・掲示板にQRコード付きチラシ（許可取得要） |
| 友だち追加特典 | 「LINE登録で初回○○円OFF」で追加動機を作る |

### 5-3. 友だち追加URL・QRコードの取得

1. LINE Official Account Manager → 友だちを増やす → 友だち追加ガイド
2. URL・QRコードをコピー/ダウンロード
3. URL形式: `https://lin.ee/xxxxxxx`

### 5-4. 友だち追加特典（重要）

大学生が「追加する理由」を作ることが最重要。おすすめ特典:

- **初回限定クーポン**（例: 初回500円OFF）
- **ドリンク1杯無料**
- **友だち紹介で両者に特典**（バイラル効果）

## 6. 料金プラン選択

BiPシーシャの規模感では以下のプラン選択が現実的:

| プラン | 月額 | 無料メッセージ数 | 推奨タイミング |
|--------|------|-----------------|---------------|
| **コミュニケーション（無料）** | 0円 | 200通/月 | 開業初期（友だち〜50人） |
| **ライト** | 5,000円 | 5,000通/月 | 友だち100人以上 |
| **スタンダード** | 15,000円 | 30,000通/月 | 友だち500人以上 |

> **メッセージ通数の計算**: 配信対象人数 × 吹き出し数 = 通数。3吹き出しのメッセージを100人に送ると300通消費。セグメント配信で対象を絞ることで節約可能。
