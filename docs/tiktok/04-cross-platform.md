## TikTok / Instagram Reels / YouTube Shorts の使い分け

### プラットフォーム特性比較

| 項目 | TikTok | Instagram Reels | YouTube Shorts |
|---|---|---|---|
| **月間ユーザー（日本）** | 約4,200万人 | 約6,600万人（Instagram全体） | 約7,000万人（YouTube全体） |
| **主要年齢層** | 10-20代が中心 | 20-30代が中心 | 全年齢層 |
| **リーチ力** | 最も高い（非フォロワーへの拡散力） | 中程度 | 高い（検索からの流入） |
| **コンテンツの寿命** | 短い（1-3日がピーク） | 中程度（1-2週間） | 長い（数ヶ月〜数年） |
| **収益化** | TikTok Shop（2026年〜日本対応） | ショッピング連携 | 広告収益（条件あり） |
| **検索性** | 高い（TikTok SEO） | 中程度 | 最も高い（Google検索にも表示） |
| **コミュニティ** | トレンド・エンタメ中心 | ライフスタイル・美意識 | 教育・情報・レビュー |

### BiPシーシャにおける各プラットフォームの役割

| プラットフォーム | 役割 | コンテンツ例 |
|---|---|---|
| **TikTok** | **メイン拠点**。バズ狙い・新規獲得 | 煙トリック、あるある、Build in Public |
| **Instagram Reels** | **ブランディング**。フィードとの連携で世界観を構築 | 空間紹介、おしゃれな煙動画、ストーリーズとの連動 |
| **YouTube Shorts** | **資産化**。長期的な検索流入 | シーシャ解説、開業ガイド、フレーバーレビュー |

## 1つの素材から3つのプラットフォームに展開する方法

### 基本戦略: TikTok ファーストで制作

```
Step 1: TikTok 向けに撮影・編集
         ↓
Step 2: TikTok に投稿（最優先）
         ↓
Step 3: 同じ動画を Instagram Reels に投稿（微調整あり）
         ↓
Step 4: 同じ動画を YouTube Shorts に投稿（微調整あり）
```

### 各プラットフォームへの最適化ポイント

#### TikTok → Instagram Reels へ変換

| 変更点 | 理由 |
|---|---|
| **TikTok のウォーターマークを除去** | ウォーターマーク付きは Reels で配信制限される |
| **ハッシュタグを5-10個に増やす** | Reels はハッシュタグが多めでも問題ない |
| **キャプションをやや丁寧に** | Instagram ユーザーはキャプションを読む傾向 |
| **ストーリーズでも告知** | 「新しい Reels 投稿したよ」とストーリーズで誘導 |
| **フィード投稿としても表示** | Reels をプロフィールグリッドにも表示させて世界観統一 |

#### TikTok → YouTube Shorts へ変換

| 変更点 | 理由 |
|---|---|
| **TikTok のウォーターマークを除去** | 他プラットフォームのロゴは NG |
| **タイトルにキーワードを入れる** | YouTube は検索エンジン。SEO が重要 |
| **説明欄にフルテキストを記載** | 説明欄にキーワード、ハッシュタグ、リンクを入れる |
| **サムネイルは自動選択で OK** | Shorts はサムネイルの重要度が低い |
| **長尺動画への導線を入れる** | 将来的に通常動画を出す場合のチャンネル育成 |

### ウォーターマーク除去の方法

TikTok で投稿した動画を他プラットフォームに転載する際、TikTok のロゴが入るとペナルティを受ける。

**最も確実な方法**: CapCut で編集した動画を書き出す際に、TikTok にアップロードする前のファイルを保存しておく。

```
CapCut で編集
  ├── 書き出し → 保存（ウォーターマークなし原本）
  ├── TikTok にアップロード
  ├── Instagram Reels にアップロード（原本を使用）
  └── YouTube Shorts にアップロード（原本を使用）
```

### 投稿タイミングの分散

同じ動画を同時に全プラットフォームに投稿するのではなく、時間差をつける。

| 順序 | プラットフォーム | タイミング | 理由 |
|---|---|---|---|
| 1 | **TikTok** | 当日 20:00 | メイン拠点。最速でバズを狙う |
| 2 | **Instagram Reels** | 翌日 12:00 | 昼休みの閲覧を狙う |
| 3 | **YouTube Shorts** | 翌々日 or 週末 | 検索流入がメインなのでタイミングの影響は小さい |

## 各プラットフォームのアルゴリズムの違い

### TikTok のアルゴリズム

| 特徴 | 詳細 |
|---|---|
| **配信ロジック** | フォロワー → 少数の非フォロワーでテスト → 反応が良ければ大量配信 |
| **最重要指標** | 視聴完了率、保存、シェア |
| **コンテンツの寿命** | 短い（投稿後24-72時間がピーク） |
| **検索** | TikTok SEO が急成長。キャプション・テロップ・音声がキーワード対象 |
| **トレンド依存度** | 高い。トレンド音源・チャレンジに乗ると加速 |
| **新規アカウントの優遇** | あり。初期の数本は多めに配信される傾向 |

### Instagram Reels のアルゴリズム

| 特徴 | 詳細 |
|---|---|
| **配信ロジック** | 既存フォロワーへの配信がベース。Reels タブで非フォロワーにも |
| **最重要指標** | いいね、コメント、シェア、保存 |
| **コンテンツの寿命** | 中程度（1-2週間は伸び続ける可能性） |
| **検索** | ハッシュタグ検索、キーワード検索に対応 |
| **トレンド依存度** | 中程度。トレンドより「美しさ」「世界観」が重視される |
| **フィードとの連携** | Reels がフィード投稿にも表示され、プロフィール全体の統一感が重要 |

### YouTube Shorts のアルゴリズム

| 特徴 | 詳細 |
|---|---|
| **配信ロジック** | 視聴履歴ベース。ユーザーの興味に合致する動画を配信 |
| **最重要指標** | クリック率（CTR）、視聴時間、チャンネル登録率 |
| **コンテンツの寿命** | 最も長い（数ヶ月〜数年にわたり検索流入） |
| **検索** | Google 検索にも表示される。SEO が最も重要 |
| **トレンド依存度** | 低い。エバーグリーン（時期に関係なく検索される）コンテンツが強い |
| **長尺動画との連携** | Shorts → 通常動画への誘導で、チャンネル全体が伸びる |

### アルゴリズム対策の比較まとめ

| 対策 | TikTok | Instagram Reels | YouTube Shorts |
|---|---|---|---|
| 視聴完了率の最適化 | ★★★★★ | ★★★★ | ★★★★ |
| トレンド音源の活用 | ★★★★★ | ★★★★ | ★★ |
| ハッシュタグ最適化 | ★★★★ | ★★★★★ | ★★★ |
| SEO/キーワード対策 | ★★★★ | ★★★ | ★★★★★ |
| 投稿頻度の維持 | ★★★★★ | ★★★★ | ★★★ |
| ビジュアルの美しさ | ★★★ | ★★★★★ | ★★★ |
| オリジナリティ | ★★★★★ | ★★★★ | ★★★★ |

## BiPシーシャの優先順位と運用計画

### プラットフォーム優先順位

```
1位: TikTok     — メイン拠点。バズ・新規獲得の起点
2位: Instagram  — ブランディング・既存フォロワーとの関係維持
3位: YouTube    — 長期的な検索流入・資産化
```

### フェーズ別の運用計画

#### フェーズ1: 立ち上げ期（開業前〜開業1ヶ月）

| プラットフォーム | 運用 | 目標 |
|---|---|---|
| TikTok | 週5投稿 | フォロワー500人 |
| Instagram | TikTokの転載 + ストーリーズ | フォロワー300人 |
| YouTube | TikTokの転載のみ | チャンネル開設・動画蓄積 |

#### フェーズ2: 成長期（開業2-6ヶ月）

| プラットフォーム | 運用 | 目標 |
|---|---|---|
| TikTok | 週5投稿 | フォロワー3,000人 |
| Instagram | 週3投稿 + ストーリーズ毎日 | フォロワー1,000人 |
| YouTube | 週2投稿（Shorts） | チャンネル登録100人 |

#### フェーズ3: 安定期（開業6ヶ月〜）

| プラットフォーム | 運用 | 目標 |
|---|---|---|
| TikTok | 週3-5投稿 | フォロワー10,000人 |
| Instagram | 週3投稿 + Reels + ストーリーズ | フォロワー3,000人 |
| YouTube | 週1-2投稿（Shorts + 長尺検討） | チャンネル登録500人 |

### コンテンツ × プラットフォーム マトリクス

どのコンテンツをどのプラットフォームに出すべきかの指針。

| コンテンツの型 | TikTok | Instagram | YouTube |
|---|---|---|---|
| 煙トリック・映え | ○ | ◎（映え重視） | ○ |
| ASMR | ◎ | ○ | ◎ |
| Build in Public | ◎（メイン） | ○（ストーリーズ連動） | ○ |
| シーシャ解説 | ○ | △ | ◎（検索向き） |
| フレーバーレビュー | ○ | ○ | ◎（検索向き） |
| あるある・ネタ | ◎（バズ狙い） | △ | △ |
| ルームツアー | ○ | ◎ | ○ |
| 開業ガイド | ○ | △ | ◎（エバーグリーン） |

◎ = 最適  ○ = 適している  △ = 優先度低
