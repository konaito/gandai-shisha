## For You ページのランキング要因

TikTok の For You ページ（FYP）は、ユーザーごとにパーソナライズされたフィードであり、フォローしていないアカウントの動画も表示される。ここに載ることがバズの入口。

### 3つの主要シグナル

| シグナル | 内容 | 重み（推定） |
|---|---|---|
| **ユーザーインタラクション** | 視聴時間、いいね、コメント、シェア、保存、スキップ | 最大（40-50%） |
| **動画情報** | キャプション、ハッシュタグ、使用音源、テキストオーバーレイ、エフェクト | 中（25-30%） |
| **アカウント情報** | 言語、国、デバイス種別、過去の視聴パターン | 小（15-20%） |

### インタラクションの優先順位（2026年版）

2026年現在、エンゲージメント指標の重みは以下の順で高い:

```
保存 > シェア > コメント > いいね > 視聴完了
```

**保存とシェアが「いいね」を大幅に上回る**のが2026年の最大の特徴。保存=「後で見返したい価値」、シェア=「人に教えたい価値」を意味し、TikTok はこれらを「高品質コンテンツ」の証拠として重視している。

### コンテンツ理解の高度化

TikTok は動画内の以下を AI で解析してカテゴリ分類している:

- **音声認識**: 話している内容をテキスト化して意味を理解
- **テキストオーバーレイ**: テロップ・字幕の内容を読み取り
- **映像認識**: 映っているモノ・場所・行動を分析
- **音楽・効果音**: 使用サウンドのカテゴリ・ムード

つまり、**動画の中身がアルゴリズムに直接影響する**。シーシャの映像、煙の演出、店内の雰囲気は全て分類対象。

## 視聴完了率・ループ再生の重要性

### 視聴完了率が最重要指標

視聴完了率（動画を最後まで見た割合）は、アルゴリズムの中で最も影響力が大きい単一指標。

| 指標 | 目安 | 効果 |
|---|---|---|
| 視聴完了率 70%以上 | バイラル配信のしきい値（2024年は50%だった） | For You で大量配信される |
| 視聴完了率 50-70% | 中程度の拡散 | 段階的にリーチが広がる |
| 視聴完了率 50%未満 | 配信制限のリスク | ほぼフォロワーにしか届かない |

### ループ再生の仕組み

- 動画が短いほどループ再生（繰り返し視聴）が起きやすい
- ループ再生は「視聴時間の累積」としてカウントされる
- **15-30秒の動画でループを狙う**のが最も効率的

### 視聴完了率を上げるテクニック

| テクニック | 説明 |
|---|---|
| **冒頭1秒のフック** | 「え、これ本当？」「知らないと損する」系の引き |
| **15-30秒に収める** | 短いほど完了率が上がる。情報は絞る |
| **結末への期待感** | 「最後にオチがある」構成で離脱を防ぐ |
| **テンポの良い編集** | 2-3秒ごとにカット・テロップ変更 |
| **ループ構造** | 終わりと始まりが自然に繋がる編集 |

## 2025-2026年のアルゴリズム変更点

### 1. フォロワーファースト配信（2025年後半〜）

**最大の変更点**: 動画はまずフォロワーに配信され、フォロワーの反応が良ければ非フォロワーへ拡散される仕組みに変更。

```
従来: 動画投稿 → ランダムな少数ユーザーにテスト → 反応良ければ拡散
2026: 動画投稿 → フォロワーに優先配信 → フォロワーの反応で拡散判定
```

**BiPシーシャへの影響**: フォロワーの質が重要。岩大生を中心とした「反応してくれるフォロワー」を初期から集めることが不可欠。

### 2. 検索アルゴリズムの強化（TikTok SEO）

TikTok が「検索エンジン」として使われる時代。特に若年層は Google より TikTok で検索する。

- キャプション・テロップ内のキーワードが検索結果に影響
- 「盛岡 シーシャ」「岩大 おすすめ」などのローカル検索に対応
- 動画の音声内容も自動文字起こしされ検索対象になる

### 3. オリジナルコンテンツ優遇

- 他プラットフォームからの転載動画は配信を大幅制限
- ウォーターマーク（他アプリのロゴ）が入った動画はペナルティ
- **TikTok ネイティブで撮影・編集した動画が優遇**

### 4. クリエイター信頼スコアの導入

- 継続的に投稿し、一貫したテーマで発信するアカウントが優遇
- 「一発屋」より「信頼できる専門家」が評価される
- シリーズ化した投稿が特に有利

### 5. 制作品質の評価

- 照明・音質・編集品質がアルゴリズムの評価対象に
- 暗い映像、ノイズの多い音声は配信制限される可能性
- プロ品質は不要だが、「最低限きれいな映像と音声」は必要

## ハッシュタグ戦略の最新トレンド

### 基本ルール: 3-5個に絞る

2026年の最適解は**1投稿あたり3-5個のハッシュタグ**。詰め込みすぎるとアルゴリズムが混乱し、むしろ逆効果。

### ハッシュタグの3層構造

| レイヤー | 種類 | 例（BiPシーシャ） | 用途 |
|---|---|---|---|
| **広域** | トレンド・大カテゴリ | `#シーシャ` `#大学生の日常` | 発見されやすさ |
| **中域** | ニッチカテゴリ | `#シーシャ好きと繋がりたい` `#起業大学生` | ターゲット層へのリーチ |
| **狭域** | ブランド・ローカル | `#BiPシーシャ` `#盛岡カフェ` `#岩大` | コミュニティ形成 |

### おすすめハッシュタグセット

**通常投稿用**:
```
#シーシャ #盛岡 #大学生の日常 #BiPシーシャ
```

**Build in Public 系投稿用**:
```
#起業 #大学生起業 #ビルドインパブリック #BiPシーシャ
```

**煙・映え系投稿用**:
```
#シーシャ #煙アート #チルタイム #盛岡カフェ #BiPシーシャ
```

### ハッシュタグ運用のポイント

- **月1回は見直す**: TikTok Creative Center でトレンドハッシュタグを確認
- **A/Bテスト**: 同じ系統の動画で異なるハッシュタグセットを試す
- **#FYP は不要**: 2026年現在、`#fyp` `#foryou` は効果がないとされる
- **キャプションにキーワードを含める**: ハッシュタグだけでなく、キャプション本文にも「盛岡」「シーシャ」「大学生」などを自然に入れる

## BiPシーシャへの実践アクション

### アルゴリズム攻略チェックリスト

- [ ] 動画は15-30秒を基本にし、視聴完了率70%以上を目指す
- [ ] 冒頭1秒で視聴者を引きつけるフックを入れる
- [ ] 保存・シェアされるような「有益 or 感動 or 驚き」のある内容にする
- [ ] キャプションとテロップに検索キーワードを含める（盛岡、シーシャ、岩大 等）
- [ ] ハッシュタグは3-5個、広域+中域+狭域の組み合わせ
- [ ] TikTok アプリ内で撮影・編集するか、ウォーターマークを入れない
- [ ] 照明と音質は最低限確保する（自然光 + 静かな場所）
- [ ] シリーズ化して一貫したテーマで投稿する
- [ ] フォロワーが反応しやすい時間帯（19-22時）に投稿する
- [ ] 投稿頻度は最低週3回を維持する
