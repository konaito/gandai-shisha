# 料金設計

BiPシーシャの料金設計に関するリサーチと戦略。大学生がメインターゲットであること、貸切予約制であることを前提に、適正価格を検討する。

## 2. 貸切制の料金モデル

BiPシーシャは「貸切予約制」のため、通常のシーシャカフェとは料金体系が異なる。以下の2つのモデルを比較する。

### モデルA: 時間制

| プラン | 料金 | 含まれるもの |
|--------|------|-------------|
| 60分プラン | 2,000円/人 | シーシャ1台（2名までシェア）、ドリンク1杯 |
| 90分プラン | 2,500円/人 | シーシャ1台（2名までシェア）、ドリンク1杯 |
| 120分プラン | 3,000円/人 | シーシャ1台（3名までシェア）、ドリンク飲み放題 |
| 延長 | +500円/30分/人 | — |

**メリット**: 回転率を管理しやすい、予約スケジュールが組みやすい
**デメリット**: 時間を気にしてリラックスできない、追加料金への不満

### モデルB: セット料金（グループ定額）

| プラン | 料金 | 含まれるもの |
|--------|------|-------------|
| 2名セット | 4,000円 | シーシャ1台、ドリンク2杯、90分 |
| 3名セット | 5,500円 | シーシャ1台、ドリンク3杯、90分 |
| 4名セット | 7,000円 | シーシャ2台、ドリンク4杯、120分 |
| パーティープラン（5〜8名） | 12,000円 | シーシャ3台、ドリンク飲み放題、180分 |

**メリット**: わかりやすい、グループで割り勘しやすい、客単価が安定
**デメリット**: 人数変動への対応が必要

### 推奨: ハイブリッド型

```
基本料金: 2,000円/人（シーシャ1台シェア + ドリンク1杯 + 90分）
追加シーシャ: +1,500円/台
追加ドリンク: +300円/杯
延長: +500円/30分/人
```

この方式なら:
- 2名で来店 → 2,000円 x 2 = 4,000円（1人2,000円）
- 3名で来店 → 2,000円 x 3 = 6,000円（1人2,000円）
- 4名 + 追加シーシャ → 2,000円 x 4 + 1,500円 = 9,500円（1人約2,375円）

## 3. 大学生向け価格設定の考え方

### 岩手大学生のアルバイト・支出感覚

| 項目 | 目安 |
|------|------|
| 大学生の月間アルバイト収入 | 3〜6万円程度 |
| 1回の飲み会予算 | 2,000〜4,000円 |
| カフェ1回の支出 | 500〜1,000円 |
| シーシャへの許容支出（想定） | 1,500〜3,000円/人 |

### 価格設定の原則

1. **1人2,000円前後を死守** — 飲み会1回分以下に抑える
2. **割り勘前提で設計** — グループで来る前提。1グループ4,000〜8,000円が目安
3. **初回割引で敷居を下げる** — 初めての人は500円OFF等
4. **リピーター優遇** — 回数券やLINE会員割引
5. **曜日/時間帯で差をつける** — 平日昼は安く、週末夜は通常料金

### 割引・特典の例

| 施策 | 内容 |
|------|------|
| 初回割引 | 初来店時 500円OFF |
| 学割 | 学生証提示で常時10%OFF（他割引と併用不可） |
| リピーター回数券 | 5回分で1回無料（実質17%OFF） |
| 平日デイタイム割 | 月〜金 12:00-17:00 は 300円OFF/人 |
| 誕生日特典 | 本人無料（要3名以上のグループ予約） |
| SNS投稿割 | Instagram投稿で次回100円OFF |

## 4. 損益分岐点の計算方法

### 基本式

```
損益分岐点売上 = 固定費 ÷ (1 - 変動費率)
```

### BiPシーシャの想定値

| 項目 | 月額 |
|------|------|
| 家賃 | 50,000円（盛岡市の小規模テナント想定） |
| 光熱費・通信費 | 15,000円 |
| 保険・雑費 | 5,000円 |
| **月間固定費合計** | **70,000円** |

| 項目 | 率 |
|------|-----|
| 材料原価率 | 20% |
| 決済手数料等 | 3% |
| **変動費率合計** | **23%** |

### 損益分岐点の計算

```
損益分岐点売上 = 70,000 ÷ (1 - 0.23)
               = 70,000 ÷ 0.77
               = 約 90,909円/月
```

### 必要セッション数に換算

| 客単価（/人） | 平均人数 | 1組あたり売上 | 月間必要組数 | 週あたり |
|--------------|---------|-------------|------------|---------|
| 2,000円 | 2.5人 | 5,000円 | 18.2組 | 約4.6組 |
| 2,000円 | 3人 | 6,000円 | 15.2組 | 約3.8組 |
| 2,500円 | 3人 | 7,500円 | 12.1組 | 約3.0組 |

**つまり、週に4〜5組の予約があれば固定費を賄える**。ただしこれは自分の人件費を含まない数字。

### 自分の生活費を含めた目標

| 月間目標利益 | 必要売上（変動費率23%の場合） | 週あたり必要組数（@5,000円/組） |
|-------------|----------------------------|-------------------------------|
| 0円（固定費のみ） | 90,909円 | 4.6組 |
| 50,000円 | 155,844円 | 7.8組 |
| 100,000円 | 220,779円 | 11.1組 |

## 5. 価格改定のタイミング

- **開業後3ヶ月**: 初期データを元に価格の妥当性を検証
- **半年後**: リピート率・客単価を分析し、必要に応じて調整
- **仕入れ価格変動時**: フレーバー値上げ等があれば原価率を再計算
- **競合出店時**: 盛岡エリアに競合が出た場合の対応策を準備
