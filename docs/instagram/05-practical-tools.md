# Instagram運用の実践テクニック・ツール

## 撮影テクニック（スマホ）

### 構図の基本
- **真俯瞰（真上）**: 複数の皿を並べたテーブルフォトに最適
- **斜め45度**: 料理の立体感。最も「食べたい」と思わせる角度
- **三分割構図**: 交点に被写体を配置
- **余白の活用**: 画面いっぱいにせず余白を意識

### 照明
- **自然光が最強**: 午前中の窓際
- **逆光・半逆光**: ツヤと透明感が出る
- **レフ板代用**: 白いハンカチ・紙で影を飛ばす
- **蛍光灯は避ける**: 色味が青白くなる

### スマホのコツ
- グリッド線を表示
- HDR機能をON
- ズームは使わず自分が近づく
- ポートレートモードで背景ぼかし

## インサイトの見方

### 前提: プロアカウント（無料）への切り替えが必要

### チェックすべき主要指標

| 指標 | 意味 | 目安 |
|---|---|---|
| リーチ数 | 投稿を見たユニークユーザー数 | 多いほど良い |
| エンゲージメント率 | (いいね+コメント+保存+シェア)÷リーチ | 飲食店は3〜6%目標 |
| 保存率 | 保存数÷リーチ数 | 1.5%以上目標 |
| シェア率 | シェア数÷リーチ数 | 0.5%以上目標 |
| プロフィールアクセス | プロフィールを見た回数 | フォロー転換の指標 |

### 分析の実践手順
1. 毎週月曜にインサイト確認
2. 保存率・シェア率の高い投稿を特定
3. 共通点を分析
4. 次の投稿に反映（PDCAサイクル）

### インタラクティブ機能

| 機能 | 活用例 |
|---|---|
| アンケート | 「新フレーバーA vs B どっちが吸いたい？」 |

### ベストプラクティス
- 1日1〜3件
- 動画は15秒以内
- テキストは最小限、ビジュアルで伝える

## DM・コメント対応

### 基本方針
- 返信速度: できれば1時間以内、遅くとも24時間以内
- トーン: フレンドリーだが丁寧
- 定型文を準備（営業時間・予約方法・アクセス）

### 自動返信ツール
- **iステップ**: コメントに反応して自動DM送信
- **エルグラム**: DM・コメント自動返信（無料プランあり）
- **Meta公式**: よくある質問への自動返信

## Instagram広告（少額から）

### 最低予算: 1日500円から（推奨テスト: 月1〜3万円）

| 方式 | 単価目安 | 適した目的 |
|---|---|---|
| CPC | 40〜200円/クリック | 予約ページ誘導 |
| CPM | 500〜3,000円/1,000表示 | 認知拡大 |

### 初心者向けステップ
1. 反応の良い投稿を選んで「投稿を宣伝」
2. 店舗周辺の半径数km + 年齢層 + 興味関心
3. 1日500円 x 7日 = 3,500円でテスト

## コラボ・インフルエンサー

### 費用相場
- マイクロ（1万〜5万フォロワー）: 2〜20万円/案件
- 計算式: フォロワー数 x 2〜4円

### 少額で始める方法
1. 地元のマイクロインフルエンサーに無料招待
2. Instagramのコラボ投稿機能活用
3. 近隣の非競合店舗と相互紹介

### 法的注意: ステマ規制（2023年10月〜）
- 「#PR」「#広告」を必ず付ける
- タイアップ投稿ラベル機能を使用

## Googleマップ等との連携

### 飲食店探しの主要チャネル
- 食べログ 59.8%
- **Instagram 52.5%**
- Google検索 49.2%
- Googleマップ 48.2%

→ 「Instagramで"行きたい"を作り、Googleマップで"来店"に変える」

### 連携方法
- プロフィールにGoogleマップの店舗リンク設置
- 投稿・ストーリーズに位置情報タグを必ず付ける
- GBP（Googleビジネスプロフィール）にInstagram URLを掲載

## 1人運営の効率的な運用フロー

### 週間スケジュール

| 曜日 | 作業内容 | 所要時間 |
|---|---|---|
| 月曜 | 前週インサイト分析、今週テーマ決定 | 30分 |
| 火曜 | 撮影（営業前の仕込み時間活用） | 30分 |
| 水曜 | 編集・キャプション作成 | 45分 |
| 木曜 | 1週間分を予約投稿 | 30分 |
| 毎日 | ストーリーズ1件 | 5〜10分 |
| 毎日 | DM・コメント返信 | 10〜15分 |

### 時短テクニック
1. Canvaでテンプレート化（写真と文字を差し替えるだけ）
2. バッチ処理（撮影→編集→キャプションを1日でまとめる）
3. AIでキャプション・ハッシュタグ生成
4. UGC活用でコンテンツ作成の負担軽減
5. 料理の仕上げ10秒を定点撮影→CapCutで音楽→完成
