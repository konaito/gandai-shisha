# キャプションテンプレート

## 通常投稿

```
[1行目: フック / 目を引く一文]

[本文: 2〜4行]

---
[CTA: フォロー・保存・DMなどの呼びかけ]

[ハッシュタグ]
```

## リール投稿

```
[1行目: 動画の要約 / フック]

[補足: 1〜2行]

🎵 BGM:
📍 場所:

[ハッシュタグ]
```
