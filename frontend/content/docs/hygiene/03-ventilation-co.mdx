## 重要: シーシャとCOの関係

シーシャは炭を燃焼させて加熱するため、**一酸化炭素（CO）が必ず発生する**。
換気対策はシーシャ営業において**最も重要な安全管理項目**である。

### 数値で見るリスク

| 測定場所 | CO濃度 | リスク |
|----------|--------|--------|
| シーシャホースから直接採取した煙 | 10,000ppm以上 | 休みなく吸い続けると数分で致死レベル |
| 換気不十分な閉鎖空間での室内濃度 | 約300ppm | 1〜2時間で頭痛・吐き気 |
| 適切に換気された室内 | 10ppm以下 | 安全域 |

> **参考**: 筑波大学の調査（全国約1,400店対象）では、回答のあった191店のうち**約6割**が客や従業員にめまい・吐き気などの急性CO中毒症状が出た経験があると回答。

## 1. シーシャ営業に必要な換気基準

### 法令上の基準

| 法令・基準 | 対象 | CO基準 | CO2基準 |
|-----------|------|--------|---------|
| 建築物衛生法（ビル管法） | 延べ面積3,000m2以上の建物 | 10ppm以下 | 1,000ppm以下 |
| 労働安全衛生法（事務所衛生基準規則） | 事務所 | 50ppm以下 | 5,000ppm以下 |
| 日本産業衛生学会 許容濃度 | 一般作業環境 | 25ppm（8時間TWA） | - |

### シーシャ営業における推奨基準

| 項目 | 推奨値 | 根拠 |
|------|--------|------|
| **室内CO濃度** | **常時10ppm以下** | 建築物衛生法の基準に準拠 |
| **室内CO2濃度** | **1,000ppm以下** | 換気の適切さの指標 |
| **換気回数** | **1時間あたり6〜10回以上** | シーシャ使用時の燃焼ガス排出に必要 |

### 必要換気量の目安計算

貸切スペースの条件:
- 部屋面積: 15〜20m2（想定）
- 天井高: 2.4m
- 室内容積: 36〜48m3
- 同時使用シーシャ台数: 1〜2台

**必要換気量 = 室内容積 x 換気回数**

| 換気回数 | 15m2の場合 | 20m2の場合 |
|----------|-----------|-----------|
| 6回/h | 216 m3/h | 288 m3/h |
| 8回/h | 288 m3/h | 384 m3/h |
| 10回/h | 360 m3/h | 480 m3/h |

→ **最低でも300 m3/h以上の換気能力**を確保すること。

## 2. 換気設備の種類と選び方

### 換気方式の比較

| 方式 | 仕組み | シーシャ営業への適合度 |
|------|--------|---------------------|
| **第1種換気** | 機械給気 + 機械排気 | 最適。給排気量を制御可能 |
| **第2種換気** | 機械給気 + 自然排気 | 不適。室内が正圧になりCOが排出されにくい |
| **第3種換気** | 自然給気 + 機械排気 | 推奨。室内を負圧に保てるためCO排出に有効 |
| 自然換気のみ | 窓開け | 補助的には有効だが、主たる換気手段としては不十分 |

### 推奨構成（BiPシーシャ向け）

**第3種換気（機械排気 + 自然給気）を基本とする。**

| 設備 | 仕様目安 | 設置場所 |
|------|----------|----------|
| **排気ファン（換気扇）** | 風量300〜500 m3/h | 天井付近または壁上部（COは空気より軽いため上部に溜まりやすい） |
| **給気口** | 排気量に見合ったサイズ | 排気ファンの対角位置の壁下部 |
| **サーキュレーター**（補助） | - | 室内の空気を循環させ、CO滞留を防ぐ |

### 換気扇の選び方

| チェック項目 | 内容 |
|-------------|------|
| 風量 | 300 m3/h以上（部屋サイズに応じて） |
| 騒音 | 40dB以下推奨（シーシャは静かな空間が重要） |
| ダクト径 | 150mm以上推奨 |
| 逆流防止 | シャッター付きを選ぶ（外気逆流防止） |
| フィルター | ヤニ対策としてフィルター付きが望ましい |

### 盛岡の気候を考慮した換気

| 季節 | 注意点 | 対策 |
|------|--------|------|
| **冬季（11〜3月）** | 外気が-10℃以下になることも。換気すると室温が急低下 | 全熱交換型換気扇の導入を検討。最低限エアコン暖房と併用 |
| **梅雨（6〜7月）** | 湿度が高く、結露・カビの原因 | 除湿器の併用 |
| **夏季（7〜8月）** | 冷房効率と換気のバランス | 窓開け換気を組み合わせる |

## 3. CO検知器の設置

### 設置義務と推奨

| 区分 | 内容 |
|------|------|
| **法的義務** | 飲食店としてのCO検知器設置義務は現時点で明文化されていないが、消防法に基づくガス漏れ警報器の設置が求められる場合がある |
| **業界推奨** | シーシャ営業においてはCO検知器の設置が**強く推奨**される |
| **保健所対応** | CO検知器の設置は保健所の指導・検査時にプラス評価 |

### 設置基準

| 項目 | 基準 |
|------|------|
| **設置台数** | 最低2台（客席エリア + 運営者作業エリア） |
| **設置高さ** | 天井から30cm以内（COは空気より軽い → 上部に滞留） |
| **設置位置** | 換気扇から離れた位置（排気口の近くだと正確に測定できない） |
| **警報設定値** | 25ppm以上で警報（国際基準UL 2034に準拠） |
| **電源** | 電池式 + 常時表示タイプ推奨（リアルタイムでCO濃度を確認できるもの） |

### 推奨製品の選定基準

| 項目 | 推奨仕様 |
|------|----------|
| 表示 | デジタル数値表示（ppm単位でリアルタイム表示） |
| 警報段階 | 複数段階（注意: 25ppm / 警告: 50ppm / 危険: 100ppm） |
| 電源 | 電池式（停電時も動作） |
| センサー寿命 | 5〜7年（メーカー推奨の交換時期を守る） |
| 認証 | UL 2034 または EN 50291 認証品 |

### CO検知器のメンテナンス

| 頻度 | 作業 |
|------|------|
| **毎月** | テストボタンで動作確認 |
| **半年ごと** | 電池残量の確認・交換 |
| **年1回** | センサーの校正確認（可能であれば） |
| **5〜7年** | 本体ごと新品に交換 |

## 4. CO中毒の症状と緊急対応

### CO中毒の症状（COHb濃度別）

| COHb濃度 | 症状 |
|----------|------|
| 5〜10% | 軽い頭痛、視力低下 |
| 10〜20% | 頭痛、疲労感、判断力低下 |
| 20〜30% | 激しい頭痛、めまい、吐き気、嘔吐 |
| 30〜40% | 意識混濁、筋力低下 |
| 40〜50% | 意識障害、けいれん |
| 50%以上 | 昏睡、死亡の危険 |

> **実例**: 日本国内で25歳女性がシーシャバーで3時間喫煙後に意識障害で救急搬送。COHb 30.7%の急性CO中毒と診断（高濃度酸素投与で回復、後遺症なし）。

### 室内CO濃度と症状の目安

| 室内CO濃度 | 曝露時間 | 症状 |
|-----------|----------|------|
| 50ppm | 8時間 | 頭痛 |
| 100ppm | 2〜3時間 | 頭痛、めまい |
| 200ppm | 1〜2時間 | 頭痛、判断力低下 |
| 400ppm | 1時間以内 | 生命の危険 |
| 800ppm以上 | 数分〜数十分 | 意識消失、死亡 |

### 緊急対応フロー

```
CO警報が鳴った / お客様・自分に症状が出た
  ↓
1. 直ちに炭を消火する（水をかける or 炭消し缶に入れる）
  ↓
2. すべての窓・ドアを開放する
  ↓
3. お客様を屋外に誘導する
  ↓
4. 症状の確認
  ├─ 軽症（頭痛・めまいのみ） → 屋外で安静、新鮮な空気を吸わせる
  └─ 中等症以上（意識障害・嘔吐等） → 119番通報
  ↓
5. CO濃度が10ppm以下に低下するまで営業を再開しない
  ↓
6. 原因の特定と再発防止策の実施
```

### 119番通報時の伝達事項

以下を簡潔に伝える:

1. 「シーシャ（水タバコ）使用中に一酸化炭素中毒の疑いがあります」
2. 場所の住所
3. 症状のある人数と状態
4. 意識の有無
5. 発症からの経過時間

### 店舗に常備すべきもの

| 品目 | 用途 |
|------|------|
| CO検知器（2台以上） | 常時モニタリング |
| 炭消し缶（火消し壺） | 緊急時の炭消火 |
| 救急キット | 基本的な応急処置 |
| 緊急連絡先一覧 | 119番、最寄りの救急病院、保健所の番号を掲示 |
| 盛岡市の救急病院リスト | 搬送先の参考 |

## 5. 換気不足のリスクと事例

### 日本国内の事例

| 時期 | 概要 |
|------|------|
| 2018〜2023年 | 東京消防庁管内でシーシャ関連のCO中毒が**64件発生、41件が救急搬送**。被害者の8割以上が20代、女性が56% |
| 事例（医学論文） | 25歳女性、シーシャバーで3時間喫煙後にCOHb 30.7%で救急搬送 |
| 2025年3月 | 東京都がシーシャ（水たばこ）によるCO中毒リスクについて注意喚起を発表 |

### リスクが高まる条件

| 条件 | 理由 |
|------|------|
| **密閉空間での使用** | CO濃度が急激に上昇する |
| **長時間セッション（2時間以上）** | COの蓄積とCOHbの上昇 |
| **複数台同時使用** | CO発生量が倍増 |
| **冬季の換気不足** | 寒さで換気を怠りがちになる |
| **炭の不完全燃焼** | 着火不十分な炭はCO発生量が多い |
| **飲酒との併用** | 症状に気づきにくくなる |

### BiPシーシャにおけるリスク軽減策

| 対策 | 実施方法 |
|------|----------|
| **セッション時間の制限** | 最大2時間を推奨。超過する場合は換気休憩を入れる |
| **炭の完全着火の徹底** | 炭全体が赤く均一に着火してから使用開始 |
| **CO濃度のリアルタイム監視** | デジタル表示のCO検知器を客席から見える位置に設置 |
| **換気ルールの明文化** | 冬季でも換気を止めない。最低限の機械換気を常時稼働 |
| **お客様への事前説明** | 「気分が悪くなったら遠慮なく申し出てください」と案内 |
| **1人運営者の自己防衛** | 自分自身がCO曝露のリスクが最も高いことを自覚。体調の変化に敏感になる |

## 6. 保健所・消防署対応メモ

- [ ] CO検知器の設置と動作確認記録を保管
- [ ] 換気設備の仕様書（風量・設置位置）を保管
- [ ] CO検知器の点検記録を定期的に記録
- [ ] 緊急対応フローを店内に掲示
- [ ] 盛岡市消防本部への「防火対象物使用開始届出書」を提出（開業7日前まで）
- [ ] 救急搬送事例が発生した場合の報告体制を整備
