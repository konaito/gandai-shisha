## 1. シーシャ本体の構造と清掃対象

```
  ┌─────────┐
  │  ボウル   │ ← フレーバーを載せるパーツ
  ├─────────┤
  │  HMD     │ ← ヒートマネジメントデバイス
  ├─────────┤
  │  ステム   │ ← 煙が通る金属パイプ
  ├─────────┤
  │  ベース   │ ← 水を入れるガラスボトル
  ├─────────┤
  │  ホース   │ ← 煙を吸引するチューブ
  └─────────┘
```

## 2. パーツ別洗浄手順

### 2-1. ボウル（クレイボウル / シリコンボウル）

| 手順 | 作業内容 |
|------|----------|
| 1 | 使用済みフレーバーとアルミホイル（またはHMD）を取り除く |
| 2 | ぬるま湯（40℃程度）に5分間浸漬し、焦げ付きをふやかす |
| 3 | ボウル用ブラシまたはスポンジで内側を洗浄 |
| 4 | 穴の詰まりを爪楊枝やピンで取り除く |
| 5 | 流水で十分にすすぐ |
| 6 | 清潔なタオルで水気を拭き、自然乾燥 |

**注意点**:
- クレイボウルは使い込むと匂いが染みつくため、フレーバー系統ごとにボウルを分けると味が混ざらない
- シリコンボウルは熱湯（100℃）で煮沸消毒可能

### 2-2. HMD（ヒートマネジメントデバイス）

| 手順 | 作業内容 |
|------|----------|
| 1 | 炭の残り・灰を除去 |
| 2 | ぬるま湯に10分間浸漬（焦げ付きをふやかす） |
| 3 | 金属たわし（ステンレス製）またはスチールウールで焦げを落とす |
| 4 | 中性洗剤で洗浄 |
| 5 | 流水ですすぎ、完全に乾燥 |

**注意点**:
- アルミ製HMDにアルカリ洗剤（ハイター等）は使わない（腐食する）
- 焦げ付きがひどい場合はクエン酸水（水500mlにクエン酸大さじ1）に一晩浸漬

### 2-3. ステム（金属パイプ）

| 手順 | 作業内容 |
|------|----------|
| 1 | ベースから取り外す |
| 2 | 片方の端を親指で塞ぎ、ぬるま湯を注入 |
| 3 | もう片方も塞いで数回振り、内部のヤニ・グリセリンを浮かせる |
| 4 | ステム用ロングブラシを通して物理的に汚れを落とす |
| 5 | 中性洗剤を付けたブラシで再度洗浄 |
| 6 | 流水で十分にすすぐ（両端から水を流す） |
| 7 | 立てかけて水を切り、自然乾燥 |

**注意点**:
- ステム内部にヤニが蓄積すると味が劣化するため、**毎セッション後に必ず洗浄**
- パージバルブ（逆止弁）も取り外して洗浄する
- ステムのゴムパッキン（グロメット）は消耗品。ひび割れや弾力低下があれば交換

### 2-4. ベース（ガラスボトル）

| 手順 | 作業内容 |
|------|----------|
| 1 | 使用済みの水を捨てる |
| 2 | ぬるま湯を1/3〜1/2まで注ぐ（**熱湯厳禁** → ガラスが割れる） |
| 3 | ベース専用ブラシ（柄の長いもの）で底面と内壁を磨く |
| 4 | 中性洗剤で洗浄 |
| 5 | 流水で十分にすすぐ |
| 6 | 逆さにして水を切り、自然乾燥 |

**週次メンテナンス（水垢・カビ対策）**:
- キッチンハイター（次亜塩素酸ナトリウム）を規定濃度に希釈し、30分浸漬
- または、重曹（大さじ2）＋ぬるま湯を入れて一晩放置
- いずれも使用後は流水で念入りにすすぐ

## 3. ホースの清掃・交換基準

### ホースの種類と洗浄可否

| ホースの種類 | 水洗い | 推奨度 |
|-------------|--------|--------|
| **シリコンホース** | 可能 | 営業用に最適。洗浄しやすく耐久性高い |
| **ウォッシャブルホース（PVC等）** | 可能 | コスパ良好。ただしシリコンより劣化が早い |
| **伝統的布/革ホース** | 不可 | 営業用には不適。洗えないためカビ・臭いの原因に |

### 推奨: シリコンホースを使用

営業用には**シリコンホース一択**。理由:
- 水洗い・煮沸消毒が可能
- 匂い移りが少ない
- 耐久性が高い（適切なメンテナンスで6ヶ月〜1年使用可能）
- 内部の汚れが視認しやすい（透明・半透明タイプの場合）

### シリコンホースの洗浄手順

| 手順 | 作業内容 |
|------|----------|
| 1 | ホースの両端を蛇口に当て、ぬるま湯を十分に流す |
| 2 | 中性洗剤を少量入れ、ホースを揺すって内壁に行き渡らせる |
| 3 | ホース用の細長いブラシがあれば通す |
| 4 | 流水で十分にすすぐ（洗剤残りがないよう両端から） |
| 5 | ホースを振って水気を切る |
| 6 | 吊るして通気性のよい場所で自然乾燥 |

### ホースの交換基準

| 状態 | 対応 |
|------|------|
| 匂いが取れない（洗浄しても残る） | **即交換** |
| 内部にカビが確認できる | **即交換** |
| ホースが硬化・ひび割れしている | **即交換** |
| 変色が著しい | **交換推奨** |
| 使用期間が6ヶ月を超えた | **交換推奨**（使用頻度による） |
| 使用期間が1年を超えた | **必ず交換** |

## 5. 清掃頻度の基準

### 頻度別清掃マトリクス

| パーツ | 毎セッション後 | 毎日（営業終了後） | 週1回 | 月1回 |
|--------|:-----------:|:-------------:|:-----:|:-----:|
| ボウル | 残渣除去・水洗い | 洗剤洗浄 | - | 煮沸消毒（シリコン） |
| HMD | 灰除去 | 焦げ落とし | クエン酸浸漬 | - |
| ステム | - | ブラシ洗浄 | - | 分解洗浄 |
| ベース | 水交換・すすぎ | 洗剤洗浄 | 漬け置き消毒 | - |
| ホース | 水通し | 洗剤洗浄・乾燥 | - | 状態確認・交換判断 |
| グロメット | - | - | 状態確認 | 交換判断 |
| ホースアダプター | - | 水洗い | 消毒 | - |

## 6. カビ・臭い対策

### カビが発生しやすい箇所

1. **ベース内部** — 水を入れっぱなしにすると発生しやすい
2. **ホース内部** — 乾燥不足が主因
3. **ステム内部** — グリセリン残留による

### 予防策

| 対策 | 内容 |
|------|------|
| **使用後は必ず水を捨てる** | ベースに水を残したまま放置しない |
| **完全乾燥の徹底** | すべてのパーツを洗浄後、十分に乾燥させてから保管 |
| **保管場所の通気性確保** | 密閉された棚ではなく、通気性のある場所で保管 |
| **ホースは吊るして保管** | 水平に置くと水分が溜まりカビの原因に |
| **週次の漬け置き消毒** | ベースは週1回、ハイターまたは重曹で漬け置き |
| **梅雨〜夏季は頻度を上げる** | 盛岡の梅雨時期（6〜7月）は特に注意。乾燥が遅いため清掃頻度を上げる |

### カビ・臭いが発生した場合の対処

| 症状 | 対処法 |
|------|--------|
| ベースの水垢・軽いカビ | キッチンハイター希釈液に30分浸漬 → 流水すすぎ |
| ベースの頑固な汚れ | 重曹 + ぬるま湯で一晩放置 → ブラシ洗浄 |
| ホースの臭い | ぬるま湯 + 中性洗剤で複数回洗浄。改善しなければ交換 |
| ホースのカビ | **即廃棄・交換**（カビの完全除去は困難） |
| ステムの臭い | クエン酸水に一晩浸漬 → ブラシ洗浄 |
| ボウルの臭い染み | 煮沸消毒（シリコン製）またはハイター浸漬 |

## 7. 保健所対応メモ

保健所検査時に提示できるよう、以下を整備しておく。

- [ ] 清掃マニュアル（本ドキュメント）を印刷して店舗に掲示
- [ ] 洗浄に使用する洗剤・消毒液のSDS（安全データシート）を保管
- [ ] 清掃記録簿を作成し、毎日の清掃実施を記録
- [ ] パーツ交換の記録（ホース・グロメット等の交換日と理由）
- [ ] 使用する洗剤が食品衛生法に適合していることの確認
