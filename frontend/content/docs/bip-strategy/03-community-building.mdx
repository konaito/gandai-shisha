## 1. フォロワー → 応援者 → 参加者のファネル

### ファネルの全体像

```
フォロワー（認知）  ← SNSで見つけて「面白そう」
    ↓
応援者（関心）      ← 継続的にウォッチ。「頑張れ」とDMをくれる
    ↓
参加者（行動）      ← 実際に来店。口コミで広めてくれる
    ↓
コアファン（推進）   ← 常連。友達を連れてくる。サブスク加入
```

### 各層の規模感（目標）

| 層 | 開業前の目標 | 開業後の目標（3ヶ月） |
|----|-----------|-------------------|
| フォロワー | 300人 | 1,000人 |
| 応援者 | 50人 | 150人 |
| 参加者 | — | 100人（累計来店者数） |
| コアファン | — | 20人（サブスク加入者） |

### フォロワー → 応援者の転換

**鍵: 「一方通行」→「双方向」にする**

| 施策 | 具体的なアクション |
|------|------------------|
| **質問を投げかける** | Instagram Storiesで「どんなフレーバーが好き？」「この物件どう思う？」 |
| **DMに返信する** | DMやコメントには必ず返信。「ありがとう」だけでも |
| **意思決定に巻き込む** | 「内装の色、AとBどっちがいい？」のアンケート |
| **名前を呼ぶ** | リアルで会ったときに「あ、Instagramでコメントくれた人ですよね」 |

### 応援者 → 参加者の転換

**鍵: 「行く理由」を作る**

| 施策 | 具体的なアクション |
|------|------------------|
| **プレオープン招待** | 開業前にBIPを見てくれていた人を優先招待 |
| **初回割引** | 「BIPフォロワー限定で初回500円OFF」 |
| **友達誘導** | 貸切なので「3人で来ると一番お得」→ 友達を誘う動機 |
| **Instagramに来店を報告しやすい空間設計** | フォトスポット、映える空間 |

### 参加者 → コアファンの転換

**鍵: 「帰属意識」を作る**

| 施策 | 具体的なアクション |
|------|------------------|
| **サブスク加入** | シーシャパス3,000円/月、フリーパス5,000円/月 → 定期的に来る理由 |
| **常連の名前を覚える** | 貸切で少人数だからこそ可能 |
| **フィードバックの反映** | 「○○さんの意見でフレーバーメニューを変えました」 |
| **コアファン限定イベント** | 新フレーバー試飲会、内装変更前の投票権 |

## 2. オフラインイベントとの連動

### 開業前のオフラインイベント

物件が決まる前でも、オフラインでの接点を作ることでBIPの効果が倍増する。

| イベント | 内容 | 期待効果 |
|---------|------|---------|
| **シーシャ会（既存店で）** | 盛岡の既存シーシャ店で友達と集まる。「こういう店を岩大の近くに作りたい」と語る | 共感者の獲得。シーシャ未経験者の教育 |
| **BIP報告会** | 岩手大学のフリースペースや学食で、進捗を友達に共有 | 口コミの起点 |
| **内見同行** | 友達を物件の内見に連れていく。「ここがシーシャ屋になるかも」 | 当事者意識の醸成 |

### 開業後のオフラインイベント

| イベント | 内容 | 期待効果 |
|---------|------|---------|
| **プレオープンイベント** | BIPフォロワー限定で無料体験。感想をSNSに投稿してもらう | UGC（ユーザー生成コンテンツ）の獲得 |
| **新フレーバー試飲会** | 新しいフレーバーの感想を聞く。投票で採用するフレーバーを決定 | 参加感。コアファンの育成 |
| **シーシャ × 勉強会** | テスト前の勉強スペースとして貸切利用 | 新規層の開拓 |
| **誕生日パーティプラン** | 誕生日の友達を連れてくると装飾サービス | グループ利用の促進 |

### オフライン → オンラインの循環

```
オフラインイベントを開催
    ↓
参加者がInstagramに投稿（UGC）
    ↓
UGCを見た人がフォロワーになる
    ↓
フォロワーが次のイベントに参加
    ↓
（ループ）
```

## 3. 大学コミュニティとの連携

### 岩手大学内の接点

| 接点 | アプローチ方法 | 期待効果 |
|------|-------------|---------|
| **サークル・部活** | 打ち上げ・親睦会での利用提案。グループ貸切プランを訴求 | 大量の新規顧客獲得 |
| **ゼミ** | ゼミの懇親会利用を提案 | 教授・先輩経由の口コミ |
| **学園祭実行委員会** | 学園祭の打ち上げ利用。学園祭へのブース出展（シーシャ体験） | 大規模な認知獲得 |
| **起業系サークル・ビジコン** | BIPの取り組み自体をプレゼン。事業計画のフィードバックを得る | 起業志向の学生との繋がり |
| **留学生コミュニティ** | シーシャは中東・東南アジア文化。留学生にとっては馴染みがある | 多様な顧客層 |

### 口コミの導線設計

岩手大学は学生数約5,500人。このうちシーシャのターゲット（20歳以上）は約3,500人。全員にリーチする必要はない。

```
まず友達10人に来てもらう
    ↓
その10人がそれぞれ3人に話す → 30人が知る
    ↓
そのうち10人が来店 → 計20人
    ↓
20人がそれぞれ3人に話す → 60人が知る
    ↓
（指数関数的に広がる）
```

大学のコミュニティは密度が高い。一つのサークルで話題になれば、関連サークルにも自然と広がる。

### 連携時の注意点

- **20歳未満には絶対に利用させない**: 大学1〜2年生は未成年の可能性。年齢確認を徹底
- **タバコを吸わない人にも居心地の良い情報発信**: シーシャに興味がない人を排除しない
- **大学の公式機関との距離感**: 大学側にテナントの宣伝を依頼するのはNG。あくまで学生同士の口コミで

## 4. メディア露出の獲得方法

### BiPシーシャのメディアストーリー

メディアが取り上げたくなるストーリーの要素:

1. **大学生が一人で事業を立ち上げる** — 若者の挑戦ストーリー
2. **AIと一緒に全部やる** — テクノロジー × リアル事業の話題性
3. **Build in Public** — 事業構築プロセスの全公開という珍しさ
4. **地方都市での挑戦** — 盛岡という地方でのチャレンジ
5. **東京でAIに仕事を奪われて帰ってきた** — ドラマチックな背景

### ターゲットメディア

| メディア | アプローチ方法 | タイミング |
|---------|-------------|-----------|
| **岩手日報** | 企画持ち込みメール → 記者との面談 | 物件決定時 or 開業時 |
| **盛岡タイムス** | 同上 | 同上 |
| **テレビ岩手（TVI）** | 報道部への企画持ち込み | 開業時（映像映えする内装完成後） |
| **岩手めんこいテレビ** | 同上 | 同上 |
| **岩手朝日テレビ** | 同上 | 同上 |
| **note / はてなブログ** | 自分でBIPの詳細記事を書く | いつでも |
| **X（旧Twitter）** | BIPの過程を投稿。テック系アカウントに拾われる可能性 | いつでも |
| **地元フリーペーパー** | 開業後にプレスリリースを送付 | 開業時 |

### 地方メディアへのアプローチの実際

盛岡の地方メディアは東京メディアと比べて距離が近い。

- **記者は少人数**: 岩手日報の記者は個人で複数分野を担当。一人と繋がればOK
- **ネタ不足**: 地方は全国紙ほどネタが潤沢ではない。「大学生の挑戦」は記事になりやすい
- **対面が強い**: メールより電話。電話より対面。可能なら新聞社に直接行く
- **タイミング**: 開業直前〜直後が最もニュース性が高い。構想段階での取材は「特集」枠を狙う

## 5. BIPを通じたブランディング

### BiPシーシャのブランドの核

| 要素 | 内容 |
|------|------|
| **透明性** | 数字も失敗も全部見せる。隠し事がない |
| **AI-Native** | AIと一緒に作った事業。テクノロジーに対する前向きな姿勢 |
| **学生の挑戦** | 完璧じゃなくていい。やりながら学ぶ |
| **盛岡ローカル** | 東京の真似ではなく、盛岡の学生のための場所 |

### ブランドの一貫性

すべてのタッチポイントで、上記のブランドの核を一貫させる。

| タッチポイント | ブランドの表現方法 |
|-------------|-------------------|
| **Webサイト** | 数字と分析の透明性。AIとの協業プロセスの可視化 |
| **Instagram** | カジュアルだけど嘘がない。日々の進捗をリアルに |
| **物件の内装** | 学生が居心地よいカジュアルな空間。高級志向ではない |
| **接客** | フレンドリー。友達の家に来た感覚 |
| **料金** | 透明。原価率まで公開済み。ぼったくりではない証拠 |

### BIPブランドの長期的な価値

BiPシーシャのBIPは、単にシーシャ店の集客手段ではない。

1. **「内藤剛汰」個人のブランド構築**: シーシャ店が仮にうまくいかなくても、BIPの過程で構築した個人ブランドは残る
2. **次の事業へのレバレッジ**: BIPで獲得したフォロワー・メディア関係・信頼は、次の事業にも使える
3. **「AIで事業を作る」のケーススタディ**: シーシャに興味がない人にも「AIの使い方」としての価値がある
4. **盛岡のスタートアップ文化への貢献**: 地方でBIPをやる先例を作ることで、他の学生にも「自分もできるかも」と思わせる

### 競合との差別化マトリクス

| | 既存シーシャ店 | BiPシーシャ |
|--|-------------|-----------|
| **情報公開** | メニューと営業時間のみ | 原価・家賃・失敗まで全公開 |
| **顧客との関係** | 来店 → 接客 → 退店 | フォロー → 応援 → 来店 → コミュニティ |
| **ブランドの源泉** | 雰囲気・立地・メニュー | ストーリー・透明性・共感 |
| **認知の獲得方法** | 看板・食べログ・口コミ | BIPコンテンツ・SNS・メディア |
