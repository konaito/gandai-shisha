## 1. Googleマップの3つの基本ランキング要因

Googleはローカル検索の順位を決定する際、以下の3つの要因を公式に公表している。

### 1-1. 関連性（Relevance）

ユーザーの検索クエリとビジネスプロフィールの内容がどれだけ一致するか。

| 施策 | 具体的なアクション |
|------|-------------------|
| カテゴリ最適化 | メインカテゴリに「水たばこ店」等を設定 |
| ビジネス説明文 | 「シーシャ」「水たばこ」「貸切」「盛岡」等のキーワードを自然に含める |
| サービス登録 | 提供サービス（シーシャ、ドリンク等）を個別に登録 |
| 投稿 | 最新情報投稿でシーシャ関連のキーワードを定期的に使用 |
| 属性設定 | 可能な属性を全て埋める |

### 1-2. 距離（Distance）

ユーザーの検索位置（または検索クエリに含まれる地名）からビジネスまでの物理的な距離。

- 距離は直接コントロールできない
- ただし、住所・サービスエリアの正確な設定が重要
- 「盛岡 シーシャ」で検索するユーザーの多くは盛岡市内にいるため、盛岡市内の立地であれば不利にはならない
- 岩手大学周辺の立地はターゲット層との距離を最小化する点で有利

### 1-3. 知名度（Prominence）

ビジネスがどれだけ広く知られているか。オンライン・オフライン両方の知名度を評価。

| 評価要素 | 内容 |
|---------|------|
| Web上のサイテーション | ポータルサイト、SNS、ブログ等での言及数 |
| 口コミの量と質 | 口コミ件数、平均評価、口コミの鮮度 |
| SEOシグナル | 公式サイトの検索順位やドメイン評価 |
| リンク | 他サイトからの被リンク |
| ブランド検索 | 「BiPシーシャ」でのダイレクト検索数 |

## 2. 口コミの重要性とアルゴリズムへの影響

### 口コミの重要度が過去最高に

2025年11月のアルゴリズム変更で、**口コミ（レビュー）の重要性がランキング要因の2位に上昇** した。Googleビジネスプロフィールの最適化に次ぐ重要項目。

### 口コミが影響する指標

| 指標 | ランキングへの影響 |
|------|-------------------|
| 口コミ件数 | 多いほど有利 |
| 平均評価 | 高いほど有利（4.0以上を維持が目安） |
| 口コミの鮮度 | 直近の口コミが重視される。定期的に新しい口コミが増えることが重要 |
| 口コミのキーワード | ユーザーが口コミに含めたキーワードが関連性評価に影響 |
| オーナーの返信 | 全ての口コミに返信することでエンゲージメントが評価される |
| 口コミの詳細さ | 長文で具体的な口コミほど評価が高い |

### BiPシーシャでの口コミ戦略の方向性

- 開業直後の3ヶ月で **最低20件** の口コミ獲得を目標にする
- 友人や初期の来店客に「シーシャ」「盛岡」「貸切」等のキーワードを含む口コミをお願いする（ただしGoogleガイドライン遵守。詳細は `03-review-strategy.md` 参照）

## 3. 最新のアルゴリズム変更（2025-2026年）

### 2025年の主要変更

| 変更内容 | 影響 |
|---------|------|
| レビュー重要度が2位に上昇 | 口コミ獲得が最優先施策に |
| 営業時間がランキングに直接影響 | 営業中の店舗が明確に優遇 |
| SNSシグナルの復活 | 8年ぶり。ただし優先度は低い |
| 13のマイナス要因追加 | キーワード詰め込み、広すぎるサービスエリア等がペナルティ対象 |
| 投稿UIの刷新 | 複数プロフィールへの同時投稿が可能に |

### 2026年のトレンド

| トレンド | 詳細 |
|---------|------|
| **リアルタイム性の重視** | 「今営業中か」だけでなく「今受け入れ可能か」を評価。混雑状況や直前の予約枠開放も要因に |
| **動画コンテンツの優遇** | 短尺動画（15〜30秒）を含むプロフィールがクリック率・ランキングで優遇 |
| **AIチャットボットの搭載** | GBPにAIチャットが搭載され、ユーザーの問い合わせに自動対応 |

### 新たなマイナス要因（ペナルティ対象）

以下の行為はランキング低下のリスクがある:

- ビジネス名へのキーワード・地名の詰め込み（例: 「BiPシーシャ 盛岡最安値 水タバコ」）
- 広すぎるサービスエリアの設定
- 虚偽の住所・電話番号
- 自作自演の口コミ
- 金銭・割引と引き換えの口コミ誘導
- 重複リスティング（同一店舗の複数登録）

## 4. 「盛岡 シーシャ」で上位表示するための施策

### 競合状況

盛岡市内の主要シーシャ店:

| 店舗名 | 特徴 |
|--------|------|
| popo sheesha&drink | 盛岡市内丸。若い男女に人気 |
| hookapooka（ふーかぷーか） | 中央通。シーシャ+お酒+ボードゲーム70種類以上 |
| Sports & sheesha Bar BLACK Rabbit | スポーツバー併設 |
| シーシャヤ ミント | シーシャ専門 |

盛岡のシーシャ市場は競合が少なく（4〜5店舗程度）、MEO対策をしっかり行えば上位表示のチャンスが大きい。

### 上位表示のためのアクションプラン

#### フェーズ1: 開業前（基盤構築）

- [ ] Googleビジネスプロフィールを登録・オーナー確認完了
- [ ] 全ての基本情報を漏れなく入力
- [ ] 写真20枚以上、動画1本以上をアップロード
- [ ] 公式サイト（shisha.konaito.com）にLocalBusiness構造化データを実装
- [ ] 主要ポータルサイト・SNSにNAP情報を統一して登録

#### フェーズ2: 開業直後（1〜3ヶ月）

- [ ] 週1〜2回の最新情報投稿を開始
- [ ] 口コミ20件を目標に獲得（全件返信）
- [ ] Instagram連携投稿（@konaito5 等）
- [ ] 地域メディア・大学メディアへの掲載を目指す

#### フェーズ3: 成長期（3ヶ月〜）

- [ ] 口コミ50件突破を目標
- [ ] 季節イベント・新フレーバーの定期投稿
- [ ] 写真・動画の定期追加（月4回以上）
- [ ] 「盛岡 シーシャ」「岩大 シーシャ」「盛岡 水タバコ」等のキーワードでの検索順位モニタリング
- [ ] 競合店の動向チェック（新しい口コミ、投稿内容等）

### 狙うべきキーワード

| キーワード | 検索意図 | 優先度 |
|-----------|---------|--------|
| 盛岡 シーシャ | 盛岡でシーシャ店を探している | 最高 |
| 盛岡 水タバコ | 同上（別の呼び方） | 高 |
| 岩手 シーシャ | 県全体で探している | 高 |
| 盛岡 貸切 シーシャ | 貸切ニーズ | 高（差別化ポイント） |
| 岩手大学 シーシャ | 大学生の検索 | 中 |
| 盛岡 シーシャバー | バー寄りの検索 | 中 |
| 盛岡 シーシャ 安い | 価格訴求 | 中 |
| 盛岡 デート シーシャ | デートユース | 中 |
