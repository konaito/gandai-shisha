## 1. アカウント作成と登録手順

### 前提条件

- Googleアカウント（Gmail）を用意
- 店舗の住所が確定していること（物件契約後）
- 店舗の電話番号が確定していること

### 登録ステップ

| ステップ | 作業内容 | 備考 |
|---------|---------|------|
| 1 | [Googleビジネスプロフィール](https://business.google.com/)にアクセス | PCからの操作を推奨 |
| 2 | 「ビジネスの管理を開始」をクリック | |
| 3 | ビジネス名を入力 | 正式名称のみ。「BiPシーシャ」等。キーワードや地名を含めるのはガイドライン違反 |
| 4 | ビジネスカテゴリを選択 | 下記「カテゴリ選択」参照 |
| 5 | 店舗の住所を入力 | 番地まで正確に |
| 6 | 電話番号・WebサイトURLを入力 | `https://shisha.konaito.com` |
| 7 | オーナー確認（本人確認） | ハガキ / 電話 / メールのいずれか |

### オーナー確認の注意点

- **ハガキ確認**: 2〜3週間かかる。届くまでビジネス名や住所を変更しないこと
- **電話確認**: 一部業種で利用可能。即日完了
- **開業前でも登録可能**: 「準備中」として登録し、開業日に公開に切り替える

## 2. カテゴリ選択

カテゴリはMEOランキングに最も影響する設定項目の一つ。メインカテゴリ1つ + サブカテゴリ（最大9つ）を設定できる。

### 推奨カテゴリ構成

| 種別 | カテゴリ名 | 理由 |
|------|-----------|------|
| **メイン** | `水たばこ店` / `Hookah bar` | シーシャ業態に最も直接的。Googleカテゴリに「水たばこ」が存在する場合はこれを使用 |
| サブ1 | `ラウンジ` | 貸切スペースとしての側面を表現 |
| サブ2 | `カフェ` | ドリンク提供がある場合 |
| サブ3 | `バー` | 飲食を提供する場合に補完 |

### カテゴリ選択時の注意

- メインカテゴリは「最も核となる業態」を1つ選ぶ
- Googleの管理画面で「シーシャ」「水たばこ」「hookah」等で検索してヒットするカテゴリを確認する
- カテゴリが見つからない場合は「ラウンジ」または「バー」をメインにし、ビジネス説明文でシーシャ業態であることを明記
- 競合店（popo、hookapooka等）のカテゴリ設定をGoogleマップで確認して参考にする

## 3. ビジネス情報の最適化（NAP統一）

### NAPとは

| 項目 | 意味 | BiPシーシャの例 |
|------|------|----------------|
| **N** (Name) | ビジネス名 | BiPシーシャ |
| **A** (Address) | 住所 | 岩手県盛岡市〇〇町X-X-X |
| **P** (Phone) | 電話番号 | 019-XXX-XXXX |

### NAP統一の原則

NAP情報は **全てのWebサイト・SNS・ポータルサイトで完全に一致** させる必要がある。

- ビジネス名: 「BiPシーシャ」で統一。「BIPシーシャ」「ビップシーシャ」等の揺れを作らない
- 住所: 「丁目」「番」「号」等の表記を統一。半角/全角も揃える
- 電話番号: ハイフンの有無を統一

### ビジネス説明文（750文字以内）

説明文にはビジネスの特徴と検索されたいキーワードを自然に含める。

**記載すべき要素:**

- 何のお店か（貸切シーシャスペース）
- 場所（盛岡市、岩手大学近く等）
- ターゲット（大学生、少人数グループ）
- 特徴（貸切、リラックス空間、フレーバーの種類等）
- 提供サービス（シーシャ、ドリンク等）

**説明文の例:**

> 盛岡市の岩手大学近くにある貸切シーシャスペースです。少人数でゆったりとシーシャ（水たばこ）を楽しめるプライベート空間を提供しています。豊富なフレーバーと落ち着いた雰囲気の中で、友人との時間やリラックスタイムをお過ごしください。大学生向けのリーズナブルな料金設定で、初めての方にも丁寧にご説明いたします。

### 属性の設定

Googleビジネスプロフィールの「属性」も漏れなく設定する。

- 支払い方法（現金、クレジットカード、電子マネー、QRコード決済）
- バリアフリー対応
- Wi-Fi有無
- 予約の可否
- 喫煙/禁煙情報
- 駐車場の有無

## 4. 写真・動画の投稿戦略

### 写真の種類と推奨枚数

| カテゴリ | 推奨枚数 | 内容例 |
|---------|---------|--------|
| カバー写真 | 1枚 | 店舗の雰囲気が最も伝わる写真 |
| ロゴ | 1枚 | BiPシーシャのロゴ |
| 外観 | 3枚以上 | 入口、看板、建物外観（昼・夜） |
| 内装 | 5枚以上 | 座席エリア、照明、装飾、全体の雰囲気 |
| シーシャ | 5枚以上 | シーシャ本体、フレーバー、煙の様子 |
| メニュー | 3枚以上 | 料金表、フレーバーリスト、ドリンクメニュー |
| スタッフ | 2枚以上 | 接客風景、準備風景 |

### 写真の仕様

- 解像度: 720px x 720px 以上推奨
- ファイルサイズ: 10KB〜5MB
- 形式: JPG / PNG
- 加工しすぎない。実際の店舗と乖離のない写真を使用
- フリー素材やスクリーンショットは使用禁止（Googleポリシー違反）

### 動画の投稿

- **長さ**: 最大30秒
- **ファイルサイズ**: 最大75MB
- **2026年のトレンド**: 短尺動画（15〜30秒）がクリック率を大きく向上させる要因になっている
- **おすすめコンテンツ**: シーシャの煙が立ち上る様子、店内の雰囲気、フレーバー紹介

### 投稿頻度

- 開業時: 最低20枚以上の写真を一括アップロード
- 運営開始後: **週1回以上** 新しい写真や動画を追加
- 季節ごとのメニュー変更、イベント時は必ず写真を更新

### 「最新情報」投稿（旧・投稿機能）

- 週1〜2回のペースで投稿
- 新フレーバー入荷、イベント告知、営業時間変更等
- 画像サイズ: 16:9（横長）または 1:1（正方形）推奨
- CTA（行動喚起）ボタンを活用: 「予約」「詳細」「電話」等

## 5. 営業時間・特別営業日の設定

### 通常営業時間

- 正確な営業時間を設定する
- 営業中の店舗はランキングで明確に優遇される（2025年のアルゴリズム変更）
- ターゲット（大学生）の行動パターンに合わせた時間帯を設定

### 特別営業日

以下のタイミングで必ず更新する:

| タイミング | 対応 |
|-----------|------|
| 祝日・祝前日 | 営業時間変更を事前設定 |
| 年末年始 | 休業日・短縮営業を設定 |
| 大学の試験期間 | 営業時間延長等があれば反映 |
| 臨時休業 | 必ず事前に設定。来店したのに閉まっていたという体験は低評価口コミに直結 |
| イベント時 | 特別営業時間を設定 |

### 営業時間に関する注意

- Googleマップ上で「営業中」「まもなく閉店」等のステータスが表示されるため、実際の営業時間と一致させることが極めて重要
- 営業時間の不一致は低評価口コミやユーザー離脱の原因になる
- 2026年のアルゴリズムでは「今すぐ受け入れ可能か」というリアルタイム性も重視されている
