## 1. 在庫管理の基本（発注点・安全在庫）

### 用語の定義

| 用語 | 定義 | BiPシーシャでの例 |
|------|------|------------------|
| **発注点** | この在庫量を下回ったら発注する基準値 | ミントフレーバー残り100g以下 |
| **安全在庫** | 欠品を防ぐためのバッファ在庫 | 炭2kg（約1週間分） |
| **発注リードタイム** | 発注から納品までにかかる日数 | 通販: 2〜5営業日 |
| **適正在庫** | 過不足なく運営できる在庫量 | 炭: 5〜10kg / フレーバー: 各100〜250g |

### 発注点方式（定量発注方式）

在庫が一定量を下回ったら、決まった量を発注する方式。BiPシーシャの規模に適している。

```
発注点 = 1日あたりの平均使用量 × リードタイム + 安全在庫

例: 炭の場合
  1日平均使用量: 150g（約2セッション分）
  リードタイム: 5日
  安全在庫: 1kg
  → 発注点 = 150g × 5日 + 1,000g = 1,750g（約1.75kg）
  → 炭の残りが1.75kgを下回ったら発注
```

## 2. フレーバー・炭・消耗品の発注サイクル

### カテゴリ別の発注管理

#### フレーバー

| 区分 | フレーバー例 | 在庫目安 | 発注点 | 発注量 | 発注頻度 |
|------|-----------|---------|--------|--------|---------|
| **定番（高回転）** | ダブルアップル、グレープ、ミント、レモンミント | 250g〜1kg | 残り100g | 250g or 1kg | 月1〜2回 |
| **準定番（中回転）** | ブルーベリー、ピーチ、マンゴー | 100〜250g | 残り50g | 100g | 月1回 |
| **スポット（低回転）** | Tangiers各種、季節もの | 50〜100g | 残り30g | 50g | 2〜3ヶ月に1回 |

#### 炭

| 項目 | 数値 |
|------|------|
| 月間使用量（40セッション想定） | 約3〜4kg |
| 適正在庫 | 5〜10kg |
| 発注点 | 残り2kg |
| 発注量 | 10kgボックス |
| 発注頻度 | 2〜3ヶ月に1回 |

#### 消耗品

| アイテム | 月間使用量 | 適正在庫 | 発注点 | 発注頻度 |
|---------|-----------|---------|--------|---------|
| 使い捨てマウスピース | 100〜200個 | 500個 | 残り100個 | 3〜6ヶ月に1回 |
| アルミホイル | 1〜2ロール | 3ロール | 残り1ロール | 3ヶ月に1回 |
| 電熱コンロのコイル（予備） | - | 1個 | 在庫0になったら | 都度 |
| ボトルブラシ | - | 2本 | 劣化したら | 半年に1回 |
| シリカゲル（乾燥剤） | 2〜3個 | 10個 | 残り3個 | 半年に1回 |

### 在庫チェックのルーティン

| 頻度 | 作業 |
|------|------|
| **毎セッション後** | 使ったフレーバー・炭の量をメモ（ざっくりでOK） |
| **週1回** | スプレッドシートの在庫数を更新。発注点を下回っているものがないかチェック |
| **月1回** | 全在庫の棚卸し。実数とスプレッドシートの数値を照合。廃棄品の確認 |

## 4. 廃棄ロスを減らすコツ

### フレーバーの廃棄ロス対策

| 対策 | 詳細 |
|------|------|
| **小容量で試す** | 新しいフレーバーは必ず50g or 100gで試す。人気が出たら大容量に切り替え |
| **FIFO（先入れ先出し）** | 古いフレーバーから先に使う。開封日をラベルに記載 |
| **開封後の期限管理** | 開封から3ヶ月以内に使い切ることを目標。3ヶ月を超えたら味をチェックし、劣化していたら廃棄 |
| **密封保存の徹底** | ジップロック + シリカゲルで保存。空気を抜いて密封 |
| **不人気フレーバーのミックス活用** | 単体では出にくいフレーバーはミックスのサブとして使い切る |
| **季節を意識した仕入れ** | 夏にミント系を多め、冬にスイーツ系を多めに仕入れ。季節外れの過剰在庫を防ぐ |
| **お客様の好みをデータ化** | どのフレーバーが何回注文されたか記録し、需要予測に活用 |

### 炭の廃棄ロス対策

| 対策 | 詳細 |
|------|------|
| **湿気防止** | 密閉容器 + シリカゲルで保管。湿った炭は使用不可 |
| **まとめ買いしすぎない** | 3ヶ月分（約10kg）を目安に。半年以上の在庫は湿気リスクが増す |
| **保管場所の温度管理** | 結露しない場所に保管。温度変化の少ない場所が理想 |

### 消耗品の廃棄ロス対策

| 対策 | 詳細 |
|------|------|
| **必要最低限の在庫** | マウスピースは半年分以上は持たない（劣化しないが場所を取る） |
| **汎用品を選ぶ** | 特定メーカー専用品より汎用品を選ぶと、機材変更時に無駄にならない |

### 廃棄コストの目安

月40セッション運営の場合、適切な在庫管理をしていれば:

| カテゴリ | 月間廃棄コスト目安 | 備考 |
|---------|------------------|------|
| フレーバー | 500〜1,500円 | 劣化・不人気フレーバー分 |
| 炭 | 0〜500円 | 湿気による廃棄分 |
| 消耗品 | ほぼ0円 | 劣化しにくい |
| **合計** | **500〜2,000円/月** | 売上の1〜3%以内が目標 |

## 5. 月次の在庫管理チェックリスト

毎月末に以下を確認:

- [ ] 全フレーバーの残量確認・スプレッドシート更新
- [ ] 開封から3ヶ月以上経過したフレーバーの味チェック
- [ ] 炭の残量確認・湿気チェック
- [ ] 使い捨てマウスピースの残数確認
- [ ] 電熱コンロのコイル状態確認
- [ ] ホース・ボウル等の劣化チェック
- [ ] 先月の使用量実績を記録
- [ ] 翌月の発注計画を立てる
- [ ] 発注履歴と支出の確認
