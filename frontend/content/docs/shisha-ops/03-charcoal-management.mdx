## 1. 炭の種類

シーシャ用の炭は大きく分けて2種類ある。**店舗運営ではココナッツチャコール一択**。

### 比較表

| 項目 | ココナッツチャコール（ナチュラル炭） | クイックライト（イージーチャコール） |
|------|-----------------------------------|----------------------------------|
| **原料** | ココナッツの殻 | 木炭 + 着火剤（硝石等） |
| **着火時間** | 8〜15分（電熱コンロ使用） | 30秒〜1分（ライターで着火） |
| **燃焼時間** | 60〜90分 | 20〜30分 |
| **温度安定性** | 非常に安定 | 不安定（急速に温度低下） |
| **異臭** | ほぼなし | 着火剤由来の異臭あり（特に着火時） |
| **一酸化炭素** | 少ない | 多い |
| **味への影響** | クリーンでフレーバーの味を損なわない | 着火剤の味が混ざることがある |
| **コスト** | やや高い | 安い |
| **推奨度** | ★★★★★（店舗必須） | ★★☆☆☆（非推奨） |

### 主要ココナッツチャコールブランド

| ブランド | 特徴 | 形状 | 価格帯（1kg） |
|---------|------|------|--------------|
| **CocoNara** | 世界的に最も有名。品質安定。燃焼時間約60分 | キューブ（26mm） | 800〜1,200円 |
| **Tom Cococha** | ヨーロッパで人気。低灰。安定した火力 | キューブ（25mm/26mm） | 900〜1,300円 |
| **Titanium** | 高品質。灰が少ない。長時間燃焼 | キューブ / フラット | 1,000〜1,500円 |
| **Coco Brico** | 安定品質。炭の表面温度約430度。ボウル内温度120〜130度 | キューブ（26mm） | 800〜1,200円 |
| **Black Coco's** | ドイツブランド。プレミアム品質 | キューブ（26mm） | 1,200〜1,800円 |

### 推奨

- **メイン**: CocoNara または Tom Cococha（コスパと品質のバランス）
- **形状**: 26mmキューブが最も汎用的。HMDにフィットしやすい
- 10kgボックスでまとめ買いすると1kgあたり600〜800円程度まで下がることがある

## 2. 炭の着火方法と適正温度

### 着火に必要なもの

| アイテム | 用途 | 注意点 |
|---------|------|--------|
| **電熱コンロ（シングルバーナー）** | 炭の着火（推奨） | IHコンロは不可。ニクロム線のコイル式を使用 |
| **炭用トング** | 炭の移動・裏返し | 金属製で長めのものを選ぶ |
| **耐熱マット** | コンロの下に敷く | テーブル保護 |

### 着火手順

```
1. 電熱コンロを最大火力でON
2. 炭をコンロの上に並べる（間隔を空けすぎない）
3. 片面が赤くなるまで約4〜5分待つ
4. トングで炭を裏返す
5. 反対面も赤くなるまで約4〜5分待つ
6. 全面が均一に赤く（灰色がかった赤）なったら着火完了
   → 合計8〜15分
```

### 重要ポイント

- **全面が赤くなるまで待つこと**。黒い部分が残っていると:
  - 一酸化炭素の発生量が増加
  - 異臭がする
  - フレーバーの味が悪くなる
  - 頭痛の原因になる
- **着火中はシーシャのセッティング（フレーバー盛り等）を並行して行う**と効率的
- 電熱コンロは2台用意し、セッション中の炭交換用にも1台稼働させておく

### 適正温度の目安

| 計測ポイント | 温度 |
|-------------|------|
| 炭の表面温度 | 約400〜450℃ |
| ボウル内部温度（HMD使用時） | 約120〜180℃ |
| フレーバーの理想加熱温度 | 約100〜200℃（ブランドにより異なる） |

> 温度が高すぎるとフレーバーが焦げて苦味が出る。低すぎると煙が出ない。HMDの蓋の開閉で微調整する。

## 3. 炭の保管方法（湿気対策）

### 湿気が炭に与える影響

- **着火しにくくなる**: 水分を含むと着火に時間がかかる
- **パチパチと弾ける**: 湿った炭は加熱時に水蒸気で弾けて危険
- **燃焼温度の低下**: 水分蒸発にエネルギーを使い、本来の温度に達しない
- **異臭の発生**: カビや雑菌が繁殖する可能性

### 保管のルール

| 項目 | 推奨 |
|------|------|
| **保管容器** | 密閉できるプラスチックコンテナまたはジップロック |
| **乾燥剤** | シリカゲルを容器内に同封（100均で購入可） |
| **保管場所** | 直射日光を避けた涼しく乾燥した場所。床に直置きしない |
| **開封後の扱い** | 使う分だけ取り出し、すぐに密封 |
| **梅雨〜夏場** | 特に注意。除湿剤を多めに入れる。エアコン除湿の部屋が理想 |

### 盛岡での注意点

岩手県盛岡市は冬の寒さが厳しく、暖房使用時に結露が発生しやすい。

- 暖房器具の近くに炭を保管しない（温度変化で結露）
- 窓際に保管しない（結露リスク）
- 冬場は暖房のない部屋（倉庫等）に保管し、使用前に室温に戻す

## 4. コンロの選び方

### 電熱コンロの種類

| タイプ | 特徴 | 価格帯 | 推奨度 |
|--------|------|--------|--------|
| **シーシャ専用電熱コンロ** | コイル型。シーシャ炭に最適化。高温 | 3,000〜6,000円 | ★★★★★ |
| **家庭用電熱コンロ（ニクロム線）** | 代用可能。ただし火力がやや弱い場合あり | 2,000〜4,000円 | ★★★☆☆ |
| **ガスコンロ + 網** | 家庭用ガスコンロに焼き網を置いて使用。火力は十分 | 手持ちのもの + 網500円 | ★★★☆☆ |
| **IHコンロ** | **使用不可**。炭は磁性体でないため加熱できない | - | 使用不可 |
| **カセットコンロ + 網** | 持ち運び可能。ただし炭落下時のガスボンベ爆発リスクあり | 1,000〜3,000円 | ★★☆☆☆ |

### 推奨構成

- **メイン**: シーシャ専用電熱コンロ × 2台
  - 1台目: セッション開始時の炭着火用
  - 2台目: セッション中の炭交換用（常時稼働）
- **予備**: 家庭用ガスコンロ + 網（電熱コンロ故障時のバックアップ）

### 電熱コンロ使用時の注意

- コンロの下に耐熱マットを敷く
- 使用後は完全に冷めるまで放置（火傷防止）
- コイルが劣化したら早めに交換（通常6ヶ月〜1年で交換目安）
- 炭のカスがコイルに溜まるので定期的に清掃

## 5. 1セッションあたりの炭使用量

### 基本的な使用量

| HMD | 初回セット | 炭交換 | 交換回数 | 合計使用量 |
|-----|-----------|--------|---------|-----------|
| **Kaloud Lotus** | 3個 | 3個 | 1〜2回 | 6〜9個 |
| **Provost** | 2個 | 2個 | 1〜2回 | 4〜6個 |
| **アルミホイル直置き** | 3個 | 3個 | 2〜3回 | 9〜12個 |

### セッション時間別の目安

| セッション時間 | 炭交換回数 | 炭使用量（Lotus） | 炭使用量（Provost） |
|--------------|-----------|------------------|-------------------|
| 45分 | 0〜1回 | 3〜6個 | 2〜4個 |
| 60分（標準） | 1回 | 6個 | 4個 |
| 90分 | 1〜2回 | 6〜9個 | 4〜6個 |
| 120分 | 2回 | 9個 | 6個 |

### コスト計算

26mmキューブの場合、1kgで約72個。

| 項目 | 数値 |
|------|------|
| 炭の仕入れ価格（1kg） | 約800〜1,200円 |
| 1個あたりのコスト | 約11〜17円 |
| 1セッション（60分・Lotus）の炭コスト | 約66〜102円 |
| 月間炭コスト（月40セッション想定） | 約2,640〜4,080円 |
| 月間炭使用量 | 約240個（約3.3kg） |

### まとめ買いの推奨

- **10kgボックス**で仕入れると1kgあたりの単価が下がる
- 月間使用量3〜4kgを考慮し、**3ヶ月分（約10kg）をまとめ買い**が効率的
- 10kgボックスの価格目安: 6,000〜10,000円
