title: "シーシャ事業の原価構造"
---

BiPシーシャの原価管理に必要な情報をまとめる。シーシャ事業は飲食業の中でも原価率が低い業態であり、正しく原価を把握すれば高い利益率を実現できる。

---

## 1. 材料別の原価内訳

### フレーバー（たばこ葉）

| 項目 | 数値 |
|------|------|
| 小売価格（50g） | 1,300〜2,500円 |
| 小売価格（250g） | 4,500〜9,000円 |
| 卸売価格（50g目安） | 800〜1,800円（卸契約・ロットによる） |
| 1セッション使用量 | 10〜15g |
| 1セッションあたり原価 | 約160〜540円 |

- 250gパッケージで仕入れると50gあたり単価が30〜40%下がる
- 人気フレーバー（Al Fakher, Fumari, Azure等）は卸でも単価が高め
- コスパ重視なら Adalya, Al Waha, Afzal 等が候補

### 炭（ココナッツチャコール）

| 項目 | 数値 |
|------|------|
| 1kg価格（小売） | 800〜1,500円 |
| 1kg価格（卸/箱買い） | 500〜900円 |
| 1セッション使用量 | 3〜4個（約40〜60g） |
| 1セッションあたり原価 | 約30〜80円 |

- 定番はCocoNara, Tom Coco, Kaloud等
- 10kg単位の箱買いで大幅にコストダウン可能

### 使い捨てマウスピース

| 項目 | 数値 |
|------|------|
| 100個入り | 500〜1,200円 |
| 1個あたり | 5〜12円 |
| 1セッション使用数 | 1〜4個（人数分） |
| 1セッションあたり原価 | 約5〜48円 |

### アルミホイル / HMS（ヒートマネジメント）

| 項目 | 数値 |
|------|------|
| アルミホイル（ロール） | 100〜300円で数十回分 |
| HMS（Kaloud Lotus等） | 3,000〜6,000円（繰り返し使用） |
| 1セッションあたり原価 | HMSなら実質0円に近い / アルミなら5〜10円 |

### その他消耗品

| 項目 | 1セッションあたり目安 |
|------|----------------------|
| ボウルパッキン（グロメット） | 5〜10円（消耗時のみ交換） |
| 水（ベース用） | ほぼ0円 |
| 清掃用品（ブラシ、洗剤等） | 5〜10円按分 |

## 2. 1セッションあたりの原価計算モデル

### 標準セッション（1台 / 60〜90分）

| 費目 | 最小 | 標準 | 最大 |
|------|------|------|------|
| フレーバー（12g） | 160円 | 300円 | 540円 |
| 炭（4個） | 30円 | 50円 | 80円 |
| マウスピース（2個） | 10円 | 20円 | 48円 |
| アルミ or HMS按分 | 0円 | 5円 | 10円 |
| その他消耗品 | 5円 | 10円 | 20円 |
| **合計** | **205円** | **385円** | **698円** |

### 想定売価との関係

| 料金設定 | 材料原価 | 原価率 |
|----------|----------|--------|
| 1,500円/台 | 385円 | 25.7% |
| 2,000円/台 | 385円 | 19.3% |
| 2,500円/台 | 385円 | 15.4% |

## 3. 仕入れルート別の価格比較

### 主要卸売サイト

| サイト名 | 特徴 | 卸価格の目安 |
|----------|------|-------------|
| C.S.B Online | 国内最大級。機材・フレーバー・炭すべて揃う | 会員登録後に卸価格表示 |
| シーシャ卸.com | 卸売専門。20社以上のメーカー取扱い | 小売の20〜30%OFF目安 |
| Tokyo Shisha | 店舗向け卸対応。LINE/メールで問合せ | 要見積もり |
| LA SHISHA | 卸売エントリーフォームあり | 要問合せ |
| ニューエモシーシャ | 通販・卸売対応 | 量による |

### 仕入れ方法の比較

| 方法 | メリット | デメリット |
|------|----------|-----------|
| 卸売サイト（会員制） | 安定した卸価格、品揃え豊富 | 最低ロットあり、初回審査 |
| メーカー直取引 | 最安値の可能性 | 大ロット必須、交渉コスト |
| 一般通販（Amazon, 楽天等） | 少量から購入可、すぐ届く | 割高（小売価格） |
| 海外直輸入 | 単価最安 | たばこ輸入規制、手続き煩雑、関税 |

### 仕入れ時の注意

- **たばこ小売販売業許可**が必要（財務省管轄）
- 仕入先によっては許可証のコピー提出が求められる
- フレーバーは「製造たばこ」に該当するため、無許可販売は法律違反

## 4. 原価率の目安と飲食業界との比較

| 業態 | 一般的な原価率 |
|------|---------------|
| **シーシャカフェ/バー** | **10〜25%** |
| 居酒屋 | 28〜35% |
| カフェ | 25〜35% |
| ラーメン店 | 30〜35% |
| 焼肉店 | 40〜50% |
| バー（酒類中心） | 15〜25% |

### シーシャ事業の原価構造の特徴

- **フード提供がない（または最小限）** → 食材ロスがほぼゼロ
- **チャージ料の原価はゼロ** → 純粋な空間提供収益
- **FL比率（Food + Labor）が低い** → 30%前後（飲食業平均は55〜65%）
- **在庫管理が比較的容易** → フレーバーは密閉保存で長期間保管可能
- 一方で **客単価が飲食店より低くなりがち** → 回転率または付加価値で補う必要あり

### BiPシーシャにおける目標原価率

貸切制・少人数運営の場合、以下を目標にする:

| 項目 | 目標値 |
|------|--------|
| 材料原価率 | 15〜20% |
| 人件費率（自分の人件費除く） | 0〜10% |
| FL比率 | 15〜30% |
| 営業利益率 | 20〜40% |
