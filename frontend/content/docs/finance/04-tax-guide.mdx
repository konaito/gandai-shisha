---
title: "個人事業主の税務ガイド"
---

BiPシーシャを個人事業主として運営するための税務知識。開業届から確定申告まで、必要な手続きと節税のポイントをまとめる。

---

## 1. 開業届と青色申告承認申請

### 開業届（個人事業の開業・廃業等届出書）

| 項目 | 内容 |
|------|------|
| 提出先 | 納税地の所轄税務署（盛岡税務署） |
| 提出期限 | 事業開始日から1ヶ月以内 |
| 費用 | 無料 |
| 提出方法 | 税務署窓口、郵送、e-Tax |
| 届出書様式 | 国税庁Webサイトからダウンロード可能 |

**記入のポイント**:
- 職業欄: 「飲食サービス業」または「シーシャカフェ運営」
- 届出の区分: 「開業」にチェック
- 開業日: 実際に営業を開始した日（準備段階でもOK）
- 事業の概要: 「シーシャ（水たばこ）の提供およびドリンク販売」

### 青色申告承認申請書

| 項目 | 内容 |
|------|------|
| 提出先 | 納税地の所轄税務署 |
| 提出期限 | **開業日から2ヶ月以内**（1/1〜1/15開業の場合は3/15まで） |
| 費用 | 無料 |
| 提出方法 | 税務署窓口、郵送、e-Tax |

**記入のポイント**:
- 簿記方式: 「複式簿記」を選択（65万円控除の条件）
- 備付帳簿名: 「仕訳帳」「総勘定元帳」にチェック（最低限この2つ）

### 提出スケジュール例

```
開業日を4月1日とした場合:
  4月1日  — 事業開始
  4月中   — 開業届 + 青色申告承認申請書を同時提出（期限: 5月31日）
  翌年2月 — 確定申告準備開始
  翌年3月15日 — 確定申告期限
```

**開業届と青色申告承認申請書は同時に提出するのがベスト**。提出を忘れると初年度は白色申告（10万円控除のみ）になってしまう。

## 2. 青色申告の65万円控除を受ける条件

### 3段階の控除額

| 控除額 | 条件 |
|--------|------|
| **65万円** | 複式簿記 + e-Tax申告 or 電子帳簿保存 |
| **55万円** | 複式簿記 + 紙で申告 |
| **10万円** | 簡易簿記 |

### 65万円控除の全条件（すべて満たす必要あり）

1. **事業所得がある**（不動産所得でも可だが事業規模が必要）
2. **青色申告承認申請書を期限内に提出済み**
3. **複式簿記で記帳している**
4. **貸借対照表と損益計算書を確定申告書に添付**
5. **法定申告期限内に確定申告書を提出**（3月15日まで）
6. **以下のいずれかを満たす**:
   - e-Taxで確定申告書を提出する **（推奨）**
   - 仕訳帳と総勘定元帳を電子帳簿保存する

### 実務的なアドバイス

- **クラウド会計ソフトを使えばほぼ自動で複式簿記になる**
  - freee（無料〜月額1,298円）
  - マネーフォワードクラウド確定申告（無料〜月額1,078円）
  - やよいの青色申告オンライン（初年度無料あり）
- **e-Taxでの申告が最も簡単に65万円控除を受ける方法**
  - マイナンバーカード + スマホ or ICカードリーダーが必要
- **65万円控除の節税効果（大学生の場合）**:
  - 所得税率5%の場合: 65万円 x 5% = 32,500円の節税
  - 住民税10%も加味すると: 65万円 x 15% = **約97,500円の節税**

## 3. シーシャ事業で経費にできるもの一覧

### 直接的な事業経費

| 勘定科目 | 具体例 | 備考 |
|---------|--------|------|
| 仕入高 | フレーバー、炭、マウスピース | 主要な変動費 |
| 消耗品費 | ボウル、ホース、アルミホイル、清掃用品 | 10万円未満の備品 |
| 器具備品 | シーシャ本体、HMS、電気コンロ | 10万円以上は減価償却 |
| 地代家賃 | テナント家賃 | 自宅兼用は按分 |
| 水道光熱費 | 電気、水道、ガス | 自宅兼用は按分 |
| 通信費 | Wi-Fi、携帯電話（事業使用分） | 按分が必要 |
| 広告宣伝費 | Instagram広告、チラシ、名刺 | SNS運用費含む |
| 旅費交通費 | 仕入れの交通費、展示会への参加 | |
| 接待交際費 | 取引先との打合せ飲食費 | 領収書必須 |
| 損害保険料 | 火災保険、賠償責任保険 | |
| 租税公課 | 事業用の印紙税、固定資産税（按分） | |
| 支払手数料 | 決済手数料、振込手数料 | |
| 新聞図書費 | シーシャ関連の書籍、業界情報 | |
| 研修費 | シーシャの技術習得、セミナー参加 | |
| 雑費 | 上記に分類できない少額経費 | |

### 自宅兼事業所の場合の按分（家事按分）

自宅の一部を事業に使う場合、使用割合に応じて経費にできる。

| 経費項目 | 按分の考え方 | 例 |
|---------|-------------|-----|
| 家賃 | 事業使用面積 ÷ 総面積 | 1Kで1部屋を事業用 → 50% |
| 電気代 | 使用時間 or 面積按分 | 事業使用30% → 電気代の30% |
| 水道代 | 使用割合 | 事業使用20% |
| 通信費 | 使用割合 | 事業使用50% |

**按分の根拠を記録しておくこと**（面積の計算、使用時間の記録など）。税務調査で質問された際に説明できるようにする。

### 開業前の経費（開業費）

開業前に支出した費用も「開業費」として経費にできる:
- 物件探しの交通費
- 市場調査のための飲食費（他店視察）
- 開業準備のための備品購入
- Webサイト制作費用

**開業費は繰延資産として計上し、任意の年に償却可能**（利益が出た年にまとめて経費化できる）。

## 4. 消費税の免税事業者とインボイス制度

### 免税事業者の条件

| 条件 | 内容 |
|------|------|
| 基準期間の課税売上高 | **1,000万円以下**なら免税 |
| 基準期間 | 2年前の1月1日〜12月31日 |
| 新規開業の場合 | **開業から2年間は原則免税** |

BiPシーシャの想定売上規模（年間100〜300万円程度）であれば、**当面は免税事業者に該当する可能性が高い**。

### インボイス制度との関係

| 項目 | 内容 |
|------|------|
| 制度開始 | 2023年10月1日〜 |
| インボイス発行事業者 | 課税事業者のみ登録可能 |
| 免税事業者の場合 | インボイスを発行できない |

### BiPシーシャにおけるインボイス制度の影響

**結論: 現時点ではインボイス登録は不要と考えられる**

理由:
1. **顧客は一般消費者（大学生）** — 仕入税額控除を必要としない
2. **BtoC取引がメイン** — インボイスの需要がない
3. **免税事業者のままでいるメリット** — 消費税の申告・納付が不要で事務負担が軽い
4. **売上規模が小さい** — 1,000万円を超える見込みがない間は免税

### インボイス登録を検討すべきケース

- 法人向けの貸切イベント（会議費・福利厚生費として処理する企業から依頼がある場合）
- 他の事業者との取引が増えた場合

### 2割特例（経過措置）

| 項目 | 内容 |
|------|------|
| 対象期間 | 2023年10月1日〜**2026年9月30日**を含む課税期間 |
| 内容 | 納税額を売上税額の2割に軽減 |
| 対象者 | 免税事業者からインボイス発行事業者になった事業者 |

**2026年9月30日以降、この特例は終了する**。もしインボイス登録する場合は、特例終了後の負担増を考慮すること。

## 5. 確定申告のスケジュール

### 年間スケジュール

| 時期 | やること |
|------|---------|
| **開業時** | 開業届 + 青色申告承認申請書を提出 |
| **毎月** | 売上・経費を会計ソフトに記帳、領収書の整理・保存 |
| **6月頃** | 住民税の通知が届く（前年分） |
| **7月** | 所得税の予定納税（第1期）※該当者のみ |
| **11月** | 所得税の予定納税（第2期）※該当者のみ |
| **1月** | 年間の帳簿を締める、決算整理仕訳 |
| **2月16日〜3月15日** | **確定申告期間**（所得税） |
| **3月15日** | 所得税の納付期限 |
| **3月31日** | 消費税の申告・納付期限（課税事業者のみ） |

### 確定申告に必要な書類

| 書類 | 備考 |
|------|------|
| 確定申告書B | e-Taxで作成・提出が便利 |
| 青色申告決算書 | 損益計算書 + 貸借対照表 |
| 各種控除証明書 | 国民健康保険、国民年金、生命保険等 |
| マイナンバーカード | e-Tax送信に必要 |
| 銀行口座情報 | 還付がある場合の振込先 |

### 帳簿・書類の保存期間

| 書類の種類 | 保存期間 |
|-----------|---------|
| 仕訳帳、総勘定元帳 | 7年 |
| 損益計算書、貸借対照表 | 7年 |
| 領収書、請求書 | 7年（※一部5年） |
| 見積書、契約書 | 5年 |

### 初年度の確定申告チェックリスト

- [ ] 会計ソフトに全取引を記帳済み
- [ ] 領収書・レシートをすべて保存（電子保存 or 紙保存）
- [ ] 売上の計上漏れがないか確認
- [ ] 経費の計上漏れがないか確認（開業費含む）
- [ ] 家事按分の比率を決定・記録
- [ ] 減価償却費の計算（10万円以上の資産）
- [ ] 棚卸し（期末在庫のフレーバー・炭の金額を把握）
- [ ] 貸借対照表のバランスが合っているか確認
- [ ] e-Taxの利用準備（マイナンバーカード、利用者識別番号）
- [ ] 3月15日までに申告書を提出

## 6. その他の税務ポイント

### 事業用口座の分離

- **事業専用の銀行口座を開設する**（個人口座と分ける）
- 事業の入出金をすべてこの口座に集約
- 会計ソフトと口座連携すれば自動仕訳が可能

### 少額減価償却資産の特例

- **30万円未満の資産**は一括で経費にできる（青色申告者の特例）
- 年間合計300万円まで
- シーシャ本体（1台1〜3万円）は即時経費化可能

### 国民健康保険・国民年金

- 個人事業主は自分で加入・支払い
- **全額が社会保険料控除の対象**
- 学生であれば国民年金の学生納付特例制度を利用可能

### 住民税の非課税ライン

- 盛岡市の場合、所得が一定額以下なら住民税が非課税になる可能性あり
- 青色申告特別控除65万円を使えば、課税所得を大幅に圧縮できる
- 例: 売上200万円 - 経費100万円 - 青色控除65万円 = 課税所得35万円
