## 1. 店舗用賃貸契約と住居用の違い

シーシャカフェの出店にあたっては、必ず**店舗用（事業用）の賃貸借契約**を締結すること。住居用物件での営業は契約違反となり、退去を求められるリスクがある。

| 項目 | 住居用賃貸 | 店舗用（事業用）賃貸 |
|------|-----------|---------------------|
| **契約形態** | 普通借家契約が多い | 普通借家または定期借家契約 |
| **敷金・保証金** | 家賃1〜2ヶ月分 | 家賃6〜12ヶ月分（高額） |
| **礼金** | 0〜2ヶ月分 | 0〜2ヶ月分 |
| **更新料** | 1ヶ月分程度 | 契約による |
| **原状回復** | 通常使用の経年劣化は貸主負担 | **借主が全額負担が原則**（スケルトン戻し等） |
| **消費税** | 非課税 | **課税（10%）** |
| **用途変更** | 不可（住居のみ） | 契約で定めた用途のみ |
| **造作（内装）** | 原則禁止 | 契約範囲内で可能 |
| **中途解約** | 1〜2ヶ月前の通知 | 3〜6ヶ月前の通知（契約による） |

## 2. シーシャ（煙・匂い）に関する特約

### 2-1. なぜ特約が重要か

シーシャカフェは大量の煙（水蒸気）が発生するため、以下のリスクがある。

- 壁・天井への煙の付着
- 近隣テナント・住居への匂いの流出
- 換気設備のダクトから発生する騒音
- 退去時の原状回復費用の増大

これらのリスクについて、契約前に大家と合意し、**特約として契約書に明記**しておくことが極めて重要。

### 2-2. 契約書に盛り込むべき特約事項

以下の項目について、大家と交渉し契約書に記載する。

| 特約項目 | 記載すべき内容 |
|----------|---------------|
| **用途の明記** | 「シーシャ（水たばこ）カフェとしての営業を認める」と明記 |
| **煙・匂いの許容** | 「営業に伴う煙（水蒸気）及び匂いの発生を了承する」 |
| **換気設備の設置** | 「借主は自己負担で換気設備を設置できる。退去時の扱いを明記」 |
| **原状回復の範囲** | 「原状回復の範囲を具体的に定める（スケルトン戻し or 現状渡し等）」 |
| **ダクト工事の許可** | 「外壁への穴あけ・ダクト設置を許可する」 |
| **造作買取請求権** | 「造作買取請求権の行使の可否を明記」 |

### 2-3. シーシャの煙の実態（大家への説明材料）

| 比較項目 | 紙巻きたばこ | シーシャ（水たばこ） |
|----------|:------------:|:-------------------:|
| 煙の成分 | タール・ニコチン含有 | 大部分が水蒸気。水を通してろ過されている |
| 壁の黄ばみ | 強い（ヤニ） | **ほぼなし**（タールが大幅に除去されている） |
| 匂いの残留 | 強い（数日〜数週間） | **弱い**（数時間で消える） |
| 火災リスク | 吸い殻からの出火 | 炭を使用（管理された火気） |

## 3. 原状回復義務

### 3-1. 店舗用賃貸の原則

事業用賃貸では、**借主が原状回復費用を全額負担するのが原則**。住居用とは異なり、経年劣化・通常損耗も借主負担となることが多い。

### 3-2. 原状回復の範囲パターン

| パターン | 内容 | 費用目安 |
|----------|------|----------|
| **スケルトン戻し** | 内装をすべて撤去し、躯体（コンクリート）のみの状態に戻す | 坪あたり3万〜10万円 |
| **原状回復（入居時の状態）** | 入居時と同じ状態に戻す | 坪あたり2万〜5万円 |
| **居抜き渡し** | 内装・設備を残して退去（次の借主に引き渡す） | 最も安い（交渉次第） |

### 3-3. 費用を抑えるポイント

- **入居前の状態を写真で記録**しておくこと（退去時のトラブル防止）
- **原状回復の範囲を契約書に具体的に明記**すること
- **居抜き退去**ができる条項を契約に入れる交渉をする
- 大規模な造作は大家の許可を得て、退去時の扱いを事前に決めておく

## 4. 大家への説明のコツ（シーシャへの偏見対策）

### 4-1. 大家が懸念するポイント

| 懸念点 | 実態と回答 |
|--------|-----------|
| 「タバコの煙で建物が汚れる」 | シーシャの煙は水蒸気が主成分。紙巻きたばこのようなヤニ汚れはほぼ発生しない |
| 「匂いが残る」 | シーシャのフルーティーな香りは数時間で消散。換気設備の設置で対策する |
| 「火事が心配」 | 消防法に完全準拠。消火器設置、CO検知器設置、消防検査をクリアする |
| 「客層が悪い」 | 岩手大学生向けの貸切制。完全予約制で不特定多数の出入りはない |
| 「違法ではないか」 | たばこ事業法に基づく正規の許可を取得して営業。合法的な事業 |
| 「近隣からクレームが来る」 | 換気ダクトの排気方向に配慮。防音対策も実施 |

### 4-2. 説明時のアプローチ

1. **事業計画書を作成して持参する**
   - 事業概要、ターゲット顧客、営業時間、取得する許可一覧を記載
   - プロフェッショナルな印象を与える

2. **「シーシャカフェ」ではなく「プライベートラウンジ」として説明する**
   - 「たばこ屋」のイメージを避ける
   - 「大学生向けの会員制リラクゼーションスペース」と説明

3. **法令遵守をアピールする**
   - 取得予定の許可一覧を見せる
   - 消防法・健康増進法への対応を具体的に説明

4. **換気設備の設計図を見せる**
   - 煙・匂いへの対策が具体的であることを示す
   - 専門業者の見積もりがあれば説得力が増す

5. **既存のシーシャカフェの事例を見せる**
   - 国内の成功事例を写真で紹介
   - おしゃれで清潔感のある空間であることを伝える

6. **賃料の上乗せや保証金の積み増しを提案する**
   - 大家のリスクを補填する姿勢を見せる

## 5. 居抜き物件のチェックポイント

前テナントの内装・設備をそのまま引き継ぐ「居抜き物件」は、初期費用を大幅に削減できる。特に飲食店の居抜きは、厨房設備やトイレなどがそのまま使えるメリットが大きい。

### 5-1. 内見時のチェックリスト

#### 設備関連

| チェック項目 | 確認ポイント |
|-------------|-------------|
| **換気設備** | 排気ファンの動作確認。ダクトの状態。シーシャには大容量の換気が必要 |
| **電気容量** | 契約アンペア数。チャコールバーナー（電気コンロ）を複数台使用するため、電気容量が足りるか |
| **給排水設備** | 水道の蛇口数、排水管の状態。飲食店営業許可の施設基準を満たすか |
| **ガス設備** | ガスの種類（都市ガス or プロパン）。必要に応じて |
| **空調設備** | エアコンの動作確認。シーシャの煙で劣化している場合あり |
| **トイレ** | 手洗い設備の有無。飲食店営業許可の要件 |
| **消防設備** | 消火器・火災報知器・誘導灯の有無と状態 |

#### 建物・構造関連

| チェック項目 | 確認ポイント |
|-------------|-------------|
| **天井の高さ** | シーシャの煙は上に溜まる。天井が低いと圧迫感がある。最低2.5m以上推奨 |
| **壁・天井の状態** | 前テナントの喫煙によるヤニ汚れがないか |
| **床の状態** | 水漏れの跡、傾きがないか |
| **防音性能** | 近隣への音漏れ。深夜営業する場合は特に重要 |
| **外壁への穴あけ可否** | 換気ダクトの設置に外壁への穴あけが必要になることが多い |
| **耐荷重** | 2階以上の場合、シーシャ台やソファの重量に耐えられるか |

#### 契約関連

| チェック項目 | 確認ポイント |
|-------------|-------------|
| **造作譲渡の条件** | 前テナントから設備を買い取る場合の金額と範囲 |
| **造作の所有権** | 譲渡される設備の所有権が明確か |
| **設備の動作保証** | 引き渡し後に故障した場合の責任の所在 |
| **前テナントの退去理由** | 事業不振か、その他の理由か（物件自体に問題がないか確認） |
| **リース設備の有無** | 前テナントがリースで使っていた設備がないか（引き継げない） |
| **原状回復の起点** | スケルトンからの原状回復か、居抜き入居時の状態への回復か |

### 5-2. 居抜き物件の注意点

- **「造作譲渡料」と「保証金」は別物**。造作譲渡料は前テナントへの支払い、保証金は大家への預け金
- **設備の「現状有姿渡し」**が一般的。引き渡し後の故障は自己責任
- **リース設備が含まれていないか確認**。リース品は前テナントに返却義務がある
- **大家の承諾**がなければ居抜きでの引き渡しはできない

## 6. 契約前の確認チェックリスト

| # | 確認事項 | 確認先 |
|---|---------|--------|
| 1 | 用途地域が飲食店営業可能なエリアか | 盛岡市都市計画課 |
| 2 | 物件の用途が「飲食店」として使用可能か | 大家・管理会社 |
| 3 | シーシャ（喫煙目的施設）としての営業を大家が了承しているか | 大家・管理会社 |
| 4 | 換気ダクトの設置（外壁への穴あけ含む）が可能か | 大家・管理会社 |
| 5 | 電気容量は十分か（30A以上推奨） | 電力会社・管理会社 |
| 6 | 原状回復の範囲が明確か | 契約書 |
| 7 | 消防法上の問題がないか | 管轄消防署 |
| 8 | 保健所の施設基準を満たせるか | 盛岡市保健所 |
| 9 | 近隣テナント・住居からのクレームリスク | 管理会社・現地確認 |
| 10 | 定期借家契約の場合、契約期間は十分か | 契約書 |
