## 1. 個人事業の開業届

### 1-1. 開業届（個人事業の開業・廃業等届出書）

| 項目 | 内容 |
|------|------|
| **届出先** | 盛岡税務署 |
| **届出期限** | 事業開始日から1ヶ月以内 |
| **届出費用** | 無料 |
| **届出方法** | 税務署窓口への持参 / 郵送 / e-Tax（電子申告） |
| **必要書類** | 個人事業の開業・廃業等届出書、マイナンバーカード（またはマイナンバー通知カード＋身分証明書） |

### 1-2. 青色申告承認申請書

開業届と同時に提出するのが望ましい。

| 項目 | 内容 |
|------|------|
| **届出先** | 盛岡税務署 |
| **届出期限** | 事業開始日から2ヶ月以内（1月1日〜1月15日開業の場合は3月15日まで） |
| **届出費用** | 無料 |
| **メリット** | 最大65万円の特別控除、赤字の3年繰越、30万円未満の一括経費計上 |

## 2. 各種届出・許可の時系列フロー

### 開業6ヶ月前〜

```
[開業6ヶ月前]
  ├─ 食品衛生責任者の資格取得（講習会 or eラーニング）
  ├─ 出店候補エリアの用途地域調査
  └─ シーシャフレーバー卸業者の選定・契約交渉

[開業4〜5ヶ月前]
  ├─ 物件探し・内見
  ├─ 大家への事業説明・交渉
  └─ 管轄消防署への事前相談

[開業3〜4ヶ月前]
  ├─ 物件の賃貸借契約締結
  ├─ たばこ出張販売許可の申請（卸業者側）← 審査に約2ヶ月
  ├─ 保健所への事前相談（図面持参）
  └─ 内装設計・施工業者の選定

[開業2〜3ヶ月前]
  ├─ 内装工事開始
  ├─ 換気設備（ダクト・排気ファン）の設置
  ├─ 消防設備の設置（消火器・誘導灯・感知器）
  └─ 厨房設備の設置（手洗い・冷蔵庫等）

[開業1〜2ヶ月前]
  ├─ 内装工事完了
  ├─ 飲食店営業許可の申請（保健所）
  ├─ 保健所の施設検査
  ├─ たばこ出張販売許可の取得（約2ヶ月後）
  └─ 個人事業の開業届・青色申告承認申請書の提出（税務署）

[開業10日前]
  ├─ 防火対象物使用開始届の提出（消防署）← 7日前まで
  ├─ 深夜営業届出（酒類提供する場合）← 10日前まで
  ├─ 消防検査
  └─ 喫煙目的施設の標識掲示

[開業日]
  └─ 営業開始！
```

## 7. 最終チェックリスト（開業前の確認）

営業開始前に、以下のすべてにチェックが入っていることを確認する。

- [ ] 用途地域が飲食店営業可能なエリアであることを確認した
- [ ] 店舗用の賃貸借契約を締結した（シーシャ営業の了承あり）
- [ ] 食品衛生責任者の資格を取得した
- [ ] たばこ出張販売許可（または小売販売業許可）を取得した
- [ ] 内装工事が完了し、施設基準を満たしている
- [ ] 換気設備が設置され、気流基準（0.2m/秒以上）を満たしている
- [ ] CO検知器を設置した
- [ ] 消火器を設置した
- [ ] 飲食店営業許可を取得した（保健所の施設検査クリア）
- [ ] 防火対象物使用開始届を提出した
- [ ] 消防検査をクリアした
- [ ] 喫煙目的施設の標識を掲示した
- [ ] 20歳未満立入禁止の表示を掲示した
- [ ] 個人事業の開業届を提出した
- [ ] 青色申告承認申請書を提出した
- [ ] 深夜営業届出を提出した（酒類を24時以降に提供する場合）
- [ ] 各種保険に加入した（火災保険・賠償責任保険等）
