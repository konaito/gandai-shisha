title: "CRM・リピーター育成"
---

BiPシーシャのLINE公式アカウントを活用したCRM（顧客関係管理）とリピーター育成戦略。大学生がメインターゲットであることを踏まえた実践的な運用ガイド。

## 1. セグメント配信の活用

### 1-1. セグメント配信とは

全員に同じメッセージを送るのではなく、ユーザーの属性や行動に応じてグループ分けし、それぞれに最適なメッセージを配信する手法。**ブロック率低減**と**反応率向上**の両方に効果がある。

### 1-2. LINE公式の標準セグメント（絞り込み配信）

LINE公式アカウントの管理画面で使えるフィルタ:

| フィルタ | 選択肢 | BiPシーシャでの活用例 |
|---------|--------|---------------------|
| 友だち期間 | 7日未満 / 7〜30日 / 30日以上 | 新規向け・常連向けの出し分け |
| 性別 | 男性 / 女性 | — |
| 年齢 | 15-19 / 20-24 / 25-29… | 大学生（20-24）に絞って配信 |
| OS | iOS / Android | — |
| エリア | 都道府県 | 岩手県に絞って配信 |

> **注意**: 標準セグメントはLINEの推定データに基づくため精度に限界がある。ターゲット数が100人以上必要。

### 1-3. タグによるセグメント管理

チャットでやり取りしたユーザーには**タグ**を付けて管理できる:

| タグ名 | 用途 |
|--------|------|
| `来店済み` | 1回以上来店したユーザー |
| `リピーター` | 2回以上来店したユーザー |
| `初回未来店` | 友だち追加のみで未来店 |
| `シーシャ初心者` | 初心者向けフォローが必要 |
| `グループ利用` | 複数人で来店した人 |
| `誕生月_○月` | 誕生月クーポン配信用 |

**タグ付けの運用ルール**:
- 来店後にチャットでお礼を送る際にタグ付け
- 予約時のやり取りで得た情報をタグに反映
- 1人運営なのでタグは最小限に。上記6種程度から始める

### 1-4. セグメント配信の活用例

| ターゲット | メッセージ内容 | タイミング |
|-----------|--------------|-----------|
| 初回未来店 | 「まだ来たことない方限定！初回500円OFFクーポン」 | 友だち追加1週間後 |
| 来店済み（1回） | 「2回目の方限定！友だちと来たらドリンク無料」 | 来店1週間後 |
| リピーター | 「常連さん限定！新フレーバー先行体験」 | 新メニュー追加時 |
| グループ利用 | 「貸切パーティープランのご案内」 | 不定期 |

## 2. ステップ配信（来店後フォロー）

### 2-1. ステップ配信とは

特定のトリガー（友だち追加、タグ付与など）をきっかけに、あらかじめ設定したメッセージを時間差で自動配信する機能。

### 2-2. 友だち追加後のステップ配信シナリオ

| 経過日数 | メッセージ内容 | 目的 |
|---------|--------------|------|
| 即時 | あいさつメッセージ（自動応答で設定） | 第一印象・特典案内 |
| 1日後 | 「BiPシーシャってこんなところ！」コンセプト紹介 | 興味喚起 |
| 3日後 | 「シーシャが初めてでも大丈夫！」初心者向けQ&A | 不安解消 |
| 7日後 | 「今週末、予約空いてます！」+ 初回クーポンリマインド | 来店促進 |

### 2-3. ステップ配信の設定手順

1. LINE Official Account Manager → ステップ配信 → 作成
2. 開始条件を設定（例: 友だち追加）
3. 待ち時間を設定（例: 1日後）
4. メッセージ内容を入力
5. 条件分岐を追加（任意: 属性で分岐可能）
6. 「利用開始」をオン

### 2-4. 来店後フォローシナリオ（タグトリガー）

来店後に「来店済み」タグを付与 → ステップ配信が起動:

| 経過日数 | メッセージ内容 |
|---------|--------------|
| 当日 | 「今日はありがとうございました！気に入ったフレーバーはありましたか？」 |
| 3日後 | 「次回使える10%OFFクーポンをプレゼント🎫」 |
| 14日後 | 「また吸いに来ませんか？新フレーバー入荷しました！」 |
| 30日後 | 「1ヶ月ぶり！久しぶりクーポンです」（条件: 2回目未来店の人のみ） |

> **1人運営のコツ**: ステップ配信を設定しておけば、来店後のフォローが自動化される。手動でやるべきことは「来店後にタグを付ける」だけ。

## 3. クーポン機能

### 3-1. LINE公式のクーポン機能

LINE公式アカウントの標準機能として、デジタルクーポンを作成・配布できる。

**作成手順**: 管理画面 → クーポン → 作成

| 設定項目 | 内容 |
|---------|------|
| クーポン名 | わかりやすく（例:「初回500円OFF」） |
| 有効期限 | 期間を設定（短めが行動を促す） |
| クーポンタイプ | 割引 / 無料 / キャッシュバック / プレゼント / その他 |
| 使用回数 | 1回のみ推奨 |
| クーポンコード | 任意（店頭確認用） |

### 3-2. BiPシーシャで使えるクーポン案

| クーポン名 | 内容 | 配布タイミング |
|-----------|------|--------------|
| 友だち追加特典 | 初回500円OFF | 友だち追加時（あいさつメッセージ） |
| リピートクーポン | 2回目10%OFF | 初回来店後ステップ配信 |
| 友だち紹介 | 紹介者・被紹介者に各300円OFF | 常時（リッチメニューから） |
| 誕生月クーポン | ドリンク1杯無料 | 誕生月にセグメント配信 |
| 平日限定 | 平日15:00-18:00 20%OFF | 月初に配信（閑散時間帯対策） |
| 雨の日クーポン | ドリンク無料 | 当日（手動配信） |

### 3-3. クーポン運用のコツ

- **有効期限は短めに**（2週間程度）: 「いつか使おう」ではなく「今使わないと」の心理を作る
- **使用条件を明記**: 「他クーポンとの併用不可」「1組1回限り」など
- **店頭での確認方法**: スマホ画面の提示 or クーポンコード読み上げ
- **利用データの確認**: 管理画面で開封率・使用率を定期チェック

## 4. ショップカード機能

### 4-1. ショップカードとは

LINE上で発行・管理できるデジタルポイントカード。紙のスタンプカードの代替。

**メリット**:
- 印刷コスト不要
- ユーザーがカードを忘れない（LINEアプリ内に保存）
- 利用データを管理画面で確認可能
- 友だち追加の動機になる

### 4-2. BiPシーシャ向け設定案

| 設定項目 | 推奨値 |
|---------|--------|
| ゴールポイント数 | 5ポイント（来店5回でゴール） |
| 付与ポイント | 1来店 = 1ポイント |
| ゴール特典 | 1時間無料 or 2,000円割引 |
| 中間特典（任意） | 3ポイント到達でドリンク無料 |
| カードデザイン | ブランドカラー + ロゴ |
| 有効期限 | 設定なし or 1年（大学在学中に使い切れる範囲） |

### 4-3. ポイント付与方法

来店時に店頭でQRコードを読み取ってもらう:

1. 管理画面 → ショップカード → ポイント付与QRコード取得
2. QRコードを印刷して店内に掲示
3. 来店者にスマホでQRコードを読み取ってもらう
4. 自動でポイント加算

> **不正防止**: ポイント付与は来店確認後のみ。QRコードは定期的にリセット可能。

### 4-4. ショップカードで友だち追加を促す

ショップカードはLINE友だちでなくても利用できるが、「友だち追加していないユーザーがQRコードを読み取ると、友だち追加画面が表示される」仕組みを利用して、自然に友だちを増やせる。

## 5. ブロック率を下げるメッセージ運用

### 5-1. ブロックされる主な原因

| 原因 | 対策 |
|------|------|
| メッセージが多すぎる | 配信頻度を週1回以下に |
| 興味のない内容 | セグメント配信で必要な人だけに |
| 宣伝ばかり | 「役立つ情報」と「宣伝」を7:3で |
| 友だち追加特典だけが目的だった | 継続的な価値提供が必要 |

### 5-2. 推奨配信頻度

| 配信タイプ | 頻度 | 内容例 |
|-----------|------|--------|
| 一斉配信 | 月2〜4回 | 新フレーバー情報、イベント告知 |
| セグメント配信 | 月1〜2回 | リピーター限定情報、初回未来店者向け |
| ステップ配信 | 自動 | 友だち追加後・来店後フォロー |
| 個別チャット | 随時 | 予約対応・問い合わせ対応 |

> **目安**: 全体で**週1回以下**を守る。大学生は通知が多いとすぐブロックする。

### 5-3. ブロック率の目安

| ブロック率 | 評価 |
|-----------|------|
| 〜20% | 優秀 |
| 20〜30% | 平均的 |
| 30〜40% | 要改善 |
| 40%以上 | 配信内容・頻度の見直しが急務 |

## 6. 大学生に刺さるメッセージの書き方

### 6-1. 基本原則

| 原則 | NG例 | OK例 |
|------|------|------|
| 短くカジュアルに | 「拝啓、時下ますますご清祥のこととお喜び申し上げます」 | 「お疲れ様！今週末ヒマ？」 |
| 絵文字は控えめに | 「🎉🎊🔥✨💯素敵なお知らせ🎉🎊🔥✨💯」 | 「新フレーバー入荷しました！🆕」 |
| メリットを最初に | 「当店では新しいサービスを開始しました」 | 「友だちと来たらドリンク無料！」 |
| 行動を促す | 「ぜひご検討ください」 | 「このトークから予約できるよ👇」 |
| 限定感を出す | 「クーポンを配布します」 | 「先着10名限定！今月だけのクーポン」 |

### 6-2. メッセージテンプレート集

**新フレーバー告知**:
```
🆕 新フレーバー入荷！

「ブルーベリーマフィン」が仲間入り！
甘くてスモーキーな新感覚フレーバーです。

📅 今週末まだ空きあります
👇 予約はメニューの「予約」から！
```

**閑散期の集客**:
```
☀️ 平日の昼シーシャ、最高です。

テスト勉強の息抜きにどうぞ。
平日15-18時は20%OFFクーポン使えます！

[クーポンを見る]
```

**イベント告知**:
```
🎊 フレーバー全種類制覇チャレンジ！

全フレーバーを制覇した人に
「BiPマスター認定証」+ 1時間無料券をプレゼント。

ショップカードで進捗管理できます。
あと何種類かな？ 確認してみて👇
```

**季節ネタ**:
```
🌸 新学期お疲れ様！

履修登録終わった？
頑張った自分にご褒美シーシャ。

新入生の友だちと来たら
紹介者もドリンク無料です！
```

### 6-3. 配信タイミング

大学生の生活リズムに合わせる:

| 曜日・時間帯 | 開封率 | 備考 |
|-------------|--------|------|
| 平日 18:00-21:00 | 高 | 授業終わり・夕食後 |
| 金曜 17:00-19:00 | 最高 | 週末の予定を決めるタイミング |
| 土曜 12:00-14:00 | 高 | 昼に予定を確認する |
| 月曜 朝 | 低 | 避ける |
| テスト期間中 | 低 | 配信を控える or 息抜き系に |

## 7. KPI管理

### 7-1. 追跡すべき指標

| KPI | 目標値 | 確認頻度 |
|-----|--------|---------|
| 友だち数 | 月+20人 | 週次 |
| ブロック率 | 20%以下 | 月次 |
| メッセージ開封率 | 60%以上 | 配信毎 |
| クーポン使用率 | 30%以上 | クーポン毎 |
| リピート率 | 40%以上 | 月次 |
| ショップカード利用率 | 来店者の80%以上 | 月次 |

### 7-2. 確認方法

- LINE Official Account Manager → 分析 → メッセージ配信
- 友だち数の推移、メッセージの開封率・クリック率を確認
- クーポンの開封数・使用数も管理画面で確認可能
