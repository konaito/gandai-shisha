## AIキャプション自動生成

### 効果的なプロンプトテンプレート

```
あなたはInstagramの熟練コピーライターです。
[画像の説明]に対する魅力的なキャプションを作成してください。

条件:
- 150文字以内
- スクロールを止めるフック（質問形式）で始める
- 個人的なストーリーか共感できるエピソードを含める
- コメントを促すCTA（質問）で締める
- ニッチ: [あなたのニッチ]
- トーン: [カジュアル/プロフェッショナル/ウィット]
- ターゲット: [ターゲット層]
```

### バリエーション生成用プロンプト

```
[トピック]について5つの異なるInstagramキャプションを書いてください。
各キャプションは:
1) スクロールを止める大胆なフックで始まる
2) 個人的なストーリーか共感ポイントを含む
3) コメントを促す明確なCTAで終わる
```

### 統計
- 80%のマーケターがAIをコンテンツ作成に使用（HubSpot 2026）
- AI活用で1コンテンツあたり平均3時間の時短

## ハッシュタグ最適化（2026年）

### 重要な変化
- Instagramは**3-5個**の高度に関連するハッシュタグを推奨（30個→大幅削減）
- 2024年12月にハッシュタグのフォロー機能が廃止
- キーワード豊富なキャプションがハッシュタグ多用投稿と比べて**リーチ30%増、いいね2倍**

### 最適化プロンプト

```
以下の投稿に最適なInstagramハッシュタグを5個提案してください:
- 投稿内容: [内容]
- ニッチ: [業界/カテゴリ]
- ターゲット: [ターゲット層]
- 目的: [リーチ拡大/エンゲージメント向上]
各ハッシュタグの選定理由も説明してください。
```

## コンテンツカレンダーAI生成

### Claude APIで1ヶ月分を生成するプロンプト

```
あなたはSNSマーケティングの専門家です。
以下の条件で1ヶ月分のInstagramコンテンツカレンダーを作成してください:

ビジネス: [ビジネス名・業種]
コンテンツの柱: [教育, エンタメ, プロモーション, ビハインドシーン]
投稿頻度: 週5回
フォーマット: フィード投稿3回、リール2回/週

出力形式:
| 日付 | 投稿タイプ | テーマ | キャプション案 | ハッシュタグ(5個) | CTA |
```

## コンテンツアイデア大量生成

### 30日分を一括生成するプロンプト

```
あなたはInstagramのコンテンツストラテジストです。
[ニッチ/業種]のアカウント向けに、30日分のコンテンツアイデアを生成してください。

各アイデアに以下を含めてください:
1. 投稿タイプ（フィード/リール/ストーリーズ/カルーセル）
2. テーマとフック
3. キャプションの方向性
4. 期待されるエンゲージメントタイプ（保存/コメント/シェア）

コンテンツミックス:
- 教育系 40%
- エンタメ系 25%
- プロモーション 20%
- ビハインドシーン 15%
```

## A/Bテスト用バリエーション生成

```
以下のInstagram投稿のA/Bテストバリエーションを作成してください:

元の投稿: [キャプション]

テストする要素:
A) フック（質問形式 vs 衝撃的事実 vs ストーリー導入）
B) CTA（質問 vs DM促進 vs 保存促進）
C) キャプション長（短い vs 長い）

各バリエーションの仮説と測定すべきKPIも提示してください。
```

## バッチ制作の効率

- AI活用で**週6-8時間で5-7本のリール＋デイリーストーリーズ**を制作可能
- 手動の15-20時間から大幅短縮
- `INSTAGRAM_VOICE.md` のようなボイスファイルでトーン統一

## 注意事項

- 1日2回以上のAPI投稿はスパムフィルターの可能性あり
- 人間の監視なしの高頻度自動投稿はリーチ減少につながりやすい
- 2026年のInstagramはAIコンテンツフィルターが強化 → Claudeのような自然な文章生成が有利
- SNS向けAI市場: 2025年26.9億ドル → 2031年113.7億ドル
