## 導線設計の全体像

```
リール/フィード投稿 → 認知獲得（発見タブ経由）
  ↓
プロフィール → 予約リンク・住所・営業時間・アクションボタン
  ↓
ストーリーズ → リンクスタンプで予約/クーポンページへ誘導
  ↓
ハイライト → メニュー、アクセス、口コミ、限定特典
  ↓
来店
```

### コンバージョン率の参考値
- Instagram平均CVR: 0.3%〜2.6%
- フォロワー1万人未満: 平均1.3%（最も高い）
- Instagram広告の平均CVR: 14.29%（リスティング広告の約4倍）

## Instagram限定クーポン

### 配布形式
- **画像クーポン型**: 投稿画像を店舗で提示→特典
- **ストーリーズ限定型**: 24時間限定で緊急性・限定感
- **フォロー&いいねキャンペーン型**: フォロー条件でクーポン
- **ハッシュタグキャンペーン型**: 来店写真投稿で次回クーポン

### メリット
- 発行コストゼロ
- 利用率が高い（スマホで即座に提示）
- 配布数・利用数がデジタルで追跡可能

## DM予約の運用

### Instagram公式機能
- 「保存済み返信（クイック返信）」でテンプレート作成可能
- 外部予約システム（RESERVA等）と連携→「予約する」ボタン

### 運用ルール
- 初回返信目標: 1時間以内
- 営業時間外はテンプレートで自動案内
- Wブッキング防止のワークフロー設計

## LINE公式アカウントとの連携

### 役割分担
- **Instagram** = 認知拡大・新規顧客獲得
- **LINE** = 既存顧客の育成・リピート促進

### 連携フロー
1. Instagramで新商品を紹介
2. 「LINE友だち追加で初回10%OFFクーポン」
3. LINE登録後、セグメント配信でリピート促進
4. 来店後、LINEでお礼メッセージ＆次回クーポン

### 誘導方法
- プロフィールリンクにLINE友だち追加URL
- ストーリーズのリンクスタンプ
- ハイライトに「LINE特典」カテゴリ

## 口コミ・UGCを増やす施策

### 統計データ
- 生活者の63%が購入前にSNS上のUGCを検索
- 20-30代の53%がUGCが購買行動に影響

### 具体的施策
| 施策 | 内容 |
|------|------|
| 撮影スポット設置 | フォトジェニックな壁・オブジェ |
| ハッシュタグ告知 | 店内POP・メニュー・レシートに記載 |
| 投稿特典 | 指定ハッシュタグ投稿でドリンク無料 |
| リポスト | お客様投稿をストーリーズでシェア |
| 口コミカード | 会計時にInstagram投稿を依頼 |
| スタッフの声かけ | マニュアル化 |

## リピーター獲得

### 新規獲得コスト = リピーター育成の約5倍（1:5の法則）

### Instagram施策
1. 会計時に「フォローで次回10%OFFクーポン」
2. 毎日ストーリーズで「思い出してもらう」接点
3. コメント返信・DM対応で関係性構築
4. 「親しい友達リスト」で常連客に先行情報配信
5. お客様投稿のリポストで愛着→再来店

## UGCサイクル（UDSSAS）

```
UGC発見 → Like → Search → 来店(Action) → 投稿(Spread) → 新たなUGC発見...
```

### サイクルを回す4ステップ
1. **投稿したくなる店舗体験の設計**: 映える盛り付け・内装、目の前で仕上げる演出
2. **投稿を促す仕掛け**: テーブルにハッシュタグPOP、投稿インセンティブ
3. **投稿を拡散する仕組み**: ストーリーズでリポスト、月間ベストフォト企画
4. **新たな来店を生む循環**: リポスト→別ユーザーが発見→来店→自分も投稿

## 来店数の計測方法

| 方法 | 精度 | コスト |
|------|------|--------|
| Instagram限定クーポン消込数 | 高 | 低 |
| 専用予約リンク（UTMパラメータ） | 中 | 低 |
| 来店時アンケート | 中 | 低 |
| Instagramインサイト分析 | 低 | 無料 |
| Instagram広告の来店CV | 高 | 中 |

### 推奨KPI
- KGI: Instagram経由の来店数（月間）
- KPI: プロフィールアクセス数、Webサイトクリック数、クーポン利用数、DM予約数
