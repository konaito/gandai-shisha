## リールが最重要な理由

- フォロワー外への拡散力がフィード投稿の3〜5倍
- 月間20億ユーザーにリーチ可能
- 全スクリーンタイムの35%を占める
- 2026年はオリジナルコンテンツが「必須」に格上げ

## バズるリールの黄金構成

```
[0-3秒] フック（つかみ）── 視聴者を釘付けにする
[3-25秒] 本題 ── 価値を提供する
[最後2-3秒] CTA ── 次のアクションを促す
```

### 最初の1-3秒のフック例
- 「知らないと損する〇〇」
- 「実は間違っている△△」
- 「〇〇する人の9割が知らない事実」
- 「3秒で分かる〇〇」
- 「これ見て行かない人いないでしょ」
- 衝撃的なビジュアル（炎が上がる、煙が広がるなど）

### 最適な動画の長さ
- **シェアされやすい: 15〜30秒**
- エンタメ系: 7〜15秒（リテンション率60-80%達成しやすい）
- 教育系: 15〜30秒

## 店舗系でバズりやすいリールのジャンル

| ジャンル | 具体例 | バズる理由 |
|---|---|---|
| 調理・制作シーン | 炭を載せる瞬間、フレーバーミックス | 視覚的満足感・ASMR |
| ビフォーアフター | 開店準備→営業中の華やかさ | ギャップで惹きつける |
| 裏側・舞台裏 | 仕込み、開店前ルーティン | 親近感・信頼感 |
| スタッフ紹介 | スタッフの人柄、技術 | 人に会いに来店する動機 |
| お客様の反応 | 初体験リアクション | 社会的証明・UGC |
| 道案内 | 最寄り駅→店舗の早送り | 来店ハードル低下（19万回再生事例） |
| 限定メニュー | 期間限定・裏メニュー | 希少性・FOMO |
| 豆知識・How to | 正しい吸い方、フレーバーの選び方 | 保存されやすい |

### シーシャ店に特に有効なネタ
- 煙のリング（スモークリング）のスローモーション
- フレーバーミックスの紹介（レシピ形式）
- 炭の火付けから提供までの一連の流れ（ASMR風）
- 店内の雰囲気映像（照明・音楽・煙のある空間）
- 「初めてのシーシャガイド」教育系コンテンツ

## 音源の選び方

### トレンド音源の見つけ方
1. リール閲覧中の「↗アイコン」= 急上昇中
2. 音源選択画面の「おすすめ」タブ
3. 「人気」タブ
4. バズっている動画の音源をタップ

### ポイント
- トレンド音源でリーチが3〜5倍に
- アップテンポ・メリハリのある曲を優先
- 80%以上のユーザーがサウンドONで視聴

## 編集テクニック

### テキスト挿入のコツ
- 冒頭2秒で興味を引くフレーズ
- 大きく、シンプルに、読みやすいフォント
- 音声オフでも内容が伝わるようにテロップを入れる

### BGMのビートに合わせてカット・テキストを配置

### 推奨編集ツール

## 再生回数を伸ばすTips

- **投稿時間**: 平日7-9時、12時、19-22時
- **投稿頻度**: 週3〜5回が理想
- **投稿後1時間以内**にコメント返信でエンゲージメント↑
- **シリーズ化**: 「Vol.1」「Part2」で視聴者の期待を作る
- **ストーリーズでリールをシェア**: 初動反応を稼ぐ
- **TikTokのウォーターマーク付き動画はNG**
- **カバー画像**: リール一覧で目を引くサムネイルを設定
