---

## 1. 接客の基本方針

### コンセプト: 「友達の家に遊びに来た感覚」

BiPシーシャは1人運営かつ大学生がターゲット。ホテルのような丁寧すぎる接客ではなく、**先輩の家に招かれたような自然な距離感**を目指す。

| やるべきこと | やらないこと |
|------------|------------|
| 名前で呼ぶ（「○○さん」or ニックネーム） | 「お客様」という呼び方 |
| タメ口混じりの敬語（親しみやすさ） | 完全なタメ口（初対面で馴れ馴れしい） |
| 自分もシーシャを楽しんでいる雰囲気 | 作業感のあるサーブ |
| 会話を振るが、放っておく時間も大切に | ずっと話しかけ続ける / 完全に放置 |
| 「困ったら声かけて」と伝える | 黙って奥に引っ込む |

## 2. 1人運営の接客フロー

### タイムライン

```
予約時間30分前  準備開始
  ├─ 空間セットアップ（照明・BGM・換気）
  ├─ シーシャ機材の準備
  └─ テーブルセッティング

予約時間      お客様到着
  ├─ 出迎え・案内
  ├─ 荷物置き場の案内
  └─ 飲み物の提供

到着後5分     シーシャの説明・フレーバー選び
  ├─ 初心者なら吸い方レクチャー
  └─ フレーバーのヒアリング・提案

到着後10-15分  シーシャ提供開始
  ├─ 最初の一口で感想を聞く
  └─ 「いい感じですか？」と確認

以降15-20分毎  炭替え
  ├─ 煙の出具合を確認
  ├─ 軽く会話（味の感想、フレーバー変更の提案）
  └─ 必要ならフレーバーチェンジ

終了15分前     お知らせ
  ├─ 「あと15分くらいですが、延長もできますよ」
  └─ 延長の意思確認

終了時        退店対応
  ├─ 会計（PayPay / 現金）
  ├─ 次回クーポンの案内
  ├─ LINE登録の案内（未登録者）
  ├─ 写真撮影の提案
  └─ 「また来てね！」で見送り
```

### 1人運営で気をつけるポイント

| 課題 | 対策 |
|-----|------|
| トイレに行けない | お客様が落ち着いたタイミングで「ちょっと失礼します」 |
| 炭替えが忙しい | シーシャの台数を同時に2台までに制限 |
| 急なトラブル対応 | 基本的なトラブル対応フローを事前に頭に入れておく |
| 体調不良 | 予約を少なく取る日を設けてバッファを持つ |
| 売上管理 | 当日の売上はその場でスマホに記録（後回しにしない） |

## 3. 初心者向けシーシャ説明テンプレート

### ステップ1: シーシャとは？（30秒）

> 「シーシャって水タバコとも言うんだけど、普通のタバコとは全然違って、フレーバーの蒸気を水に通して吸うんだよね。だから煙がすごくまろやかで、フルーツとかの味がする。ニコチンは入ってるけど、タバコみたいにガツンとくる感じじゃなくて、リラックスする感じ。」

### ステップ2: 吸い方のレクチャー（1分）

> 「吸い方は簡単で、このホース（マウスピース）をくわえて、ゆっくり深呼吸するみたいに吸い込む。コツは**思ったよりゆっくり、深く**吸うこと。急いで吸うと味がしないし、むせやすい。」
>
> 「最初は短めに吸って、慣れてきたらだんだん深く吸ってみて。煙を出すときも、ゆっくりフーッと。」

### ステップ3: 注意点（30秒）

> 「注意点としては、空腹のときに吸うとクラクラしやすいから、何か食べてから来るか、ドリンク飲みながらがおすすめ。あと、水分はこまめに取ってね。気分悪くなったら遠慮なく言って！」

### ステップ4: よくある質問への回答

| 質問 | 回答例 |
|-----|-------|
| 「タバコ吸わないけど大丈夫？」 | 「全然大丈夫！むしろシーシャから入る人の方が多い。タバコとは別物だよ」 |
| 「クラクラしない？」 | 「最初は軽くフワッとすることはあるかも。水飲みながらゆっくりやれば大丈夫。無理はしないでね」 |
| 「1台をみんなでシェアするの？」 | 「そうそう、回し飲みのイメージ。マウスピースは1人1個あるから衛生面は安心して」 |
| 「何分くらいもつの？」 | 「1台で大体60-90分くらい楽しめるよ。途中でフレーバー変えることもできる」 |
| 「肺に入れるの？」 | 「肺には入れなくてOK。口に含んでゆっくり出す感じ。ふかすだけで十分味わえる」 |

## 4. フレーバー提案の方法

### ヒアリングの流れ

```
Step 1: 経験を聞く
  「シーシャ吸ったことある？」
     ├─ ある → 「前はどんな味吸った？好きだった？」
     └─ ない → Step 2へ

Step 2: 好みを聞く（初心者向け）
  「普段どんな飲み物が好き？」
  「フルーツ系とスッキリ系、どっちが気分？」
     ├─ フルーツ系 → ベリー、ピーチ、マンゴー等を提案
     ├─ スッキリ系 → ミント、柑橘系を提案
     └─ わからない → おすすめミックスを提案

Step 3: 提案する
  「じゃあ○○と○○のミックスはどう？ △△な味になるよ」
```

### 初心者おすすめフレーバーマップ

| 好みのタイプ | おすすめフレーバー | 説明トーク例 |
|------------|-----------------|-------------|
| 甘いもの好き | ピーチ+バニラ | 「桃のジュースみたいな甘さだよ」 |
| フルーツ好き | ダブルアップル+ミント | 「シーシャの王道。りんごの甘さにミントのスッキリ感」 |
| さっぱり派 | レモン+ミント | 「レモネードっぽい爽やかさ。一番吸いやすい」 |
| 冒険したい | ブルーベリー+ラベンダー | 「ちょっと大人っぽい。香りがめっちゃいい」 |
| お酒好き | グレープ+ミント | 「ワインっぽい深みがあるよ」 |
| カフェ好き | チャイ+シナモン | 「チャイラテの香り。秋冬に人気」 |

### フレーバーチェンジの提案タイミング

- 炭替え2回目（来店30-40分後）が自然なタイミング
- 「味に慣れてきたでしょ？次は違うの試してみる？」
- 2台目のシーシャを注文してもらえると客単価UP

## 5. クレーム対応の基本

### 基本姿勢

大学生同士の距離感だからこそ、問題が起きた時の対応で信頼が決まる。

**3ステップ**: 共感 → 対応 → フォロー

### 想定されるクレームと対応

| クレーム | 原因 | 対応 |
|---------|------|------|
| 「煙が出ない / 味がしない」 | 炭の配置不良、フレーバーの水分過多 | すぐに炭を調整 or 新しく作り直す。「ごめん、すぐ直すね！」 |
| 「気分が悪い」 | ニコチン酔い（特に初心者） | シーシャを止めて換気。水を提供。「無理しないで休んで」 |
| 「思ったより高い」 | 料金の事前認識不足 | 「説明不足でごめんね」と謝罪。次回クーポンで誠意を見せる |
| 「予約時間より短い」 | 準備の遅れ | 無料で延長。「準備遅くなってごめん。その分延長するね」 |
| 「部屋が暑い / 寒い」 | 空調管理の不備 | エアコン調整+ブランケット提供。「言ってくれてありがとう」 |
| 「シーシャの台が汚い」 | 清掃不足 | 即座に交換。清掃フローを見直し |

### 絶対にやってはいけないこと

- 言い訳をする（「1人でやってるんで...」）
- 問題を軽視する（「そんなもんですよ」）
- SNSでのネガティブ投稿を恐れて態度を変える
- 対応を後回しにする

### 事後フォロー

- 問題があった顧客にはLINEで「今日はごめんね。次回○○円オフにするから、リベンジさせて！」
- クレーム内容は必ず記録し、再発防止策を講じる

## 7. 衛生管理

### シーシャ特有の衛生対策

| 項目 | 対応 |
|-----|------|
| マウスピース | 1人1個の使い捨てマウスピースを提供。共用は絶対にしない |
| ホース | 毎回アルコール消毒。定期的に交換 |
| ボトル（水） | 毎回新しい水に交換。ボトル内部は毎回洗浄 |
| ボウル | 使用後に洗浄。フレーバーの残りカスをしっかり除去 |
| テーブル・クッション | 毎回アルコール消毒。カバーは定期的に洗濯 |
| 空間 | 換気を十分に行い、消臭剤で匂いをリセット |

### 清掃チェックリスト（お客様退店後）

- [ ] シーシャ機材の洗浄・消毒
- [ ] テーブルの拭き上げ
- [ ] クッション・ラグの確認（汚れがあれば対処）
- [ ] 灰・炭の処理
- [ ] ゴミの回収
- [ ] 換気（最低15分）
- [ ] 消臭
- [ ] トイレの清掃
- [ ] 次の予約の準備物確認

## 8. 緊急対応マニュアル

| 事態 | 対応 |
|-----|------|
| お客様が体調不良 | シーシャ中止→換気→水提供→休ませる→改善しなければ119番 |
| 炭による火傷 | 流水で冷やす→応急処置→必要なら医療機関へ |
| 炭の落下・火災リスク | 耐熱シートで消火→換気→安全確認 |
| 酔っ払い客のトラブル | 冷静に対応→エスカレートしたら退店を促す→最悪は110番 |
| 機材の故障 | 予備機材で対応。なければ正直に説明し、割引 or 無料で次回招待 |

## 参考情報

- シーシャは炭替えのタイミングで自然に会話が生まれるため、通常の飲食店より継続的なコミュニケーション機会がある
- 初心者にはフルーツ系・ミント系から始めることが推奨される（業界標準）
- フレーバーの好みは「普段飲む飲み物」からのアナロジーで提案すると伝わりやすい
