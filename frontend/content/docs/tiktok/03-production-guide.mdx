## 撮影機材

### 結論: スマホだけで十分

2026年の TikTok では「プロ品質」は求められていない。むしろスマホ撮影の「リアル感」が好まれる。ただし最低限の品質は必要。

### 必須機材（予算0円〜5,000円）

| 機材 | おすすめ | 予算 | 必要度 |
|---|---|---|---|
| **スマホ** | iPhone 13以降 or 同等の Android | 手持ちでOK | ★★★★★ |
| **三脚/スマホスタンド** | 100均のスマホスタンドで可 | 110円〜 | ★★★★★ |
| **照明** | 自然光（窓際）で十分。夜間はリングライト | 0〜2,000円 | ★★★★ |
| **外部マイク** | ピンマイク（ASMR動画用） | 1,000〜3,000円 | ★★★ |

### あると便利な機材（予算5,000円〜）

| 機材 | 用途 | 予算 |
|---|---|---|
| **リングライト** | 夜間撮影・顔出し時の照明 | 2,000〜5,000円 |
| **ジンバル（スタビライザー）** | 歩き撮り・ルームツアー時の手ブレ防止 | 5,000〜15,000円 |
| **ワイヤレスマイク** | 動きながらの解説動画 | 3,000〜8,000円 |
| **背景布（黒）** | 煙トリック動画の背景 | 1,000〜2,000円 |

### スマホ撮影の設定

```
解像度: 1080p（フルHD）で十分。4K は不要
フレームレート: 通常は30fps、スローモーション用に60fps or 120fps
アスペクト比: 9:16（縦長）
グリッド線: ON（構図の補助）
HDR: OFF（編集で色が崩れることがある）
```

### シーシャ動画の撮影テクニック

#### 煙の撮影

- **逆光 or サイドライトで煙を際立たせる**: 光源を煙の後ろ or 横に配置
- **黒背景**: 黒い布 or 暗い部屋で煙が映える
- **スローモーション**: 120fps で撮影し、編集で 0.25x 再生
- **接写**: 煙がパイプから出る瞬間をアップで

#### 空間の撮影

- **広角レンズ（スマホ内蔵）**: 部屋全体を映す時に使用
- **ゆっくりパン（横移動）**: 空間の雰囲気を伝える
- **定点タイムラプス**: 改装過程、セッティング過程

## 編集アプリの比較

### メイン推奨: CapCut（キャップカット）

TikTok 公式の編集アプリ。無料で高機能。大学生が1人で使う前提ならこれ一択。

| 項目 | 内容 |
|---|---|
| **価格** | 基本無料（Pro版あり。無料で十分） |
| **対応** | iOS / Android / PC |
| **長所** | TikTok連携、豊富なテンプレート、AI字幕、トレンドエフェクト |
| **短所** | Pro限定素材あり。ロゴが入る場合あり（無料版） |

#### CapCut の必須機能

| 機能 | 用途 |
|---|---|
| **自動字幕（Auto Caption）** | 話した内容を自動でテロップ化。手動修正は必要 |
| **テンプレート** | トレンド動画の型をそのまま使える |
| **速度変更** | スローモーション・倍速 |
| **トランジション** | カット間の切り替え効果 |
| **背景除去** | グリーンバックなしで背景を消せる |
| **テキスト to ビデオ** | テキストから AI で動画素材を生成 |

### 比較表

| アプリ | 価格 | 難易度 | TikTok連携 | おすすめ度 |
|---|---|---|---|---|
| **CapCut** | 無料 | ★★（簡単） | 最高 | ★★★★★ |

### 推奨ワークフロー

```
1. スマホで撮影（TikTokアプリ or カメラアプリ）

4. TikTok にアップロード（キャプション、ハッシュタグ追加）
5. 同じ動画を Instagram Reels、YouTube Shorts にも投稿
```

## フック・本編・CTAの構成テンプレート

### 基本構成: 3パート構造

すべてのショート動画は「フック → 本編 → CTA」の3パートで構成する。

```
┌─────────────────────────────────────┐
│ フック（0-2秒）                      │
│ → 視聴者を「止める」                │
├─────────────────────────────────────┤
│ 本編（2-20秒）                      │
│ → 価値を「届ける」                  │
├─────────────────────────────────────┤
│ CTA（20-25秒）                      │
│ → 行動を「促す」                    │
└─────────────────────────────────────┘
```

### フック（冒頭0-2秒）のパターン集

フックが全て。最初の1秒で視聴者を止められなければ、その動画は存在しないのと同じ。

| フックの型 | 例文 | 効果 |
|---|---|---|
| **疑問型** | 「盛岡にシーシャ屋がないの、知ってた？」 | 好奇心を刺激 |
| **数字型** | 「大学生が50万円でシーシャ屋を作った話」 | 具体性で引きつける |
| **衝撃型** | 「大学3年でシーシャ屋を開業します」 | 驚きで止める |
| **結論先出し型** | 「シーシャ屋、初月の売上は○万円でした」 | 結果を先に見せる |
| **否定型** | 「シーシャ屋を開くのに飲食店営業許可は要りません」 | 常識を覆す |
| **呼びかけ型** | 「岩大生、これ知ってる？」 | ターゲットに直接語りかける |
| **映像フック** | （煙トリックの最もインパクトのある瞬間） | 視覚で止める |

### 本編の構成パターン

#### パターンA: リスト型（情報系）

```
フック: 「シーシャ初心者が知るべき3つのこと」
① ポイント1（3秒）
② ポイント2（3秒）
③ ポイント3（3秒）
CTA: 「詳しくはプロフのリンクから」
```

#### パターンB: ストーリー型（Build in Public系）

```
フック: 「大学生がシーシャ屋の物件を探した結果…」
状況説明（3秒）
課題・困難（5秒）
解決・成果（5秒）
CTA: 「続きはフォローして待っていてください」
```

#### パターンC: Before/After型（改装系）

```
フック: Before の映像（ボロい空き部屋）
転換: トレンド音源に合わせてカット
After: 完成した映像
CTA: 「遊びに来てね！場所はプロフに」
```

#### パターンD: リアクション型（エンタメ系）

```
フック: 「友達を初めてシーシャに連れてきた」
リアクション映像（10-15秒）
CTA: 「あなたの初シーシャの感想もコメントで教えて」
```

### CTA（行動喚起）のパターン

| CTA の型 | 例文 | 目的 |
|---|---|---|
| **フォロー誘導** | 「続きが気になる人はフォロー」 | フォロワー増加 |
| **コメント誘導** | 「どっちのフレーバーが気になる？コメントで教えて」 | エンゲージメント |
| **保存誘導** | 「保存しておくと後で見返せるよ」 | 保存数アップ |
| **プロフ誘導** | 「予約はプロフのリンクから」 | 来店・Web誘導 |
| **シェア誘導** | 「シーシャ好きの友達に送ってあげて」 | シェア数アップ |

## BGM・トレンド音源の活用

### トレンド音源の探し方

| 方法 | 手順 |
|---|---|
| **TikTok の「おすすめ」をチェック** | FYP で流れてくる動画の音源を保存 |
| **TikTok Creative Center** | `ads.tiktok.com/business/creativecenter` → Trends → Songs |
| **検索バーで🎵アイコン** | 使いたいジャンルで検索し、使用回数が多い音源を選ぶ |
| **他のシーシャアカウントを参考** | 同業のバズ動画で使われている音源をチェック |

### シーシャ動画に合う音源ジャンル

| ジャンル | 用途 | 雰囲気 |
|---|---|---|
| **ローファイ/チルホップ** | 煙動画、空間紹介 | リラックス、おしゃれ |
| **アラビック/オリエンタル** | シーシャの文化感を出す時 | エキゾチック、本格派 |
| **トレンドポップ** | あるある系、リアクション系 | キャッチー、若者向け |
| **ASMR（無音 or 環境音）** | ASMR系、煙の音 | 没入感 |
| **トレンドリミックス** | Before/After、トランジション系 | テンポ、切り替え |

### 音源選びの鉄則

- **トレンド音源は鮮度が命**: 流行り始めて1-2週間以内に使う
- **使用回数10万〜50万の音源が狙い目**: 少なすぎると認知度が低く、多すぎると競合が激しい
- **オリジナルサウンドも有効**: シーシャのブクブク音など、独自の音をオリジナルサウンドとして登録
- **著作権に注意**: TikTok内の音源ライブラリから選べば問題なし。外部音源は著作権を確認

## テロップ・エフェクトの効果的な使い方

### テロップの基本ルール

| ルール | 説明 |
|---|---|
| **必ずテロップを入れる** | ミュートで視聴する人が多い（電車内など）。テロップがないと内容が伝わらない |
| **フォントは読みやすく** | ゴシック体が基本。明朝体は小さいと読みにくい |
| **文字サイズは大きめ** | スマホの小さい画面で読めるサイズ |
| **1画面の文字量は少なく** | 1フレーズ15文字以内が目安 |
| **セーフゾーンを守る** | 上下左右の端にテロップを置かない（UIに隠れる） |

### テロップのセーフゾーン

```
┌──────────────────────┐
│   ▲ 上10%は空ける    │  ← ユーザー名が被る
│                      │
│  ┌──────────────┐    │
│  │              │    │
│  │  テロップは  │    │
│  │  この範囲に  │    │
│  │              │    │
│  └──────────────┘    │
│                      │
│   ▼ 下20%は空ける    │  ← キャプション・UIが被る
└──────────────────────┘
```

### テロップの演出テクニック

| テクニック | やり方 | 効果 |
|---|---|---|
| **キーワード強調** | 重要な単語だけ色を変える（赤・黄色） | 視線誘導 |
| **縁取り（アウトライン）** | 白文字 + 黒縁取り | どんな背景でも読める |
| **ポップイン** | テロップが弾むように表示 | テンポ感 |
| **手書き風フォント** | 感情的なコメント時に使用 | 親近感 |
| **カウントダウン** | 「3… 2… 1…」の表示 | 期待感 |

### エフェクトの使い方

| エフェクト | 用途 | 注意 |
|---|---|---|
| **スピードランプ** | 一部分だけスロー or 倍速に | メリハリをつける。やりすぎ注意 |
| **ズームイン** | 重要な瞬間を強調 | 2-3秒に1回程度 |
| **画面切り替え（トランジション）** | シーン変更時 | シンプルなカットが基本。派手すぎない |
| **フィルター** | 動画全体の色味調整 | 暖色系がシーシャの雰囲気に合う |
| **グリーンスクリーン** | 背景を変える（解説動画時） | 画像やWebサイトを背景に解説 |

### CapCut でのテロップ追加手順

```
1. CapCut で動画を読み込む
2. 「テキスト」→「自動キャプション」をタップ
3. 言語を「日本語」に設定して生成
4. 誤字を手動で修正
5. フォント・サイズ・色・位置を調整
6. 強調したいキーワードの色を変更
```

## 撮影チェックリスト

### 撮影前

- [ ] スマホの充電は80%以上あるか
- [ ] ストレージに空きはあるか（最低2GB）
- [ ] レンズは綺麗か（指紋を拭く）
- [ ] 撮影場所の照明は十分か（自然光 or リングライト）
- [ ] 騒音はないか（エアコン音、外の音）
- [ ] 三脚/スタンドは安定しているか
- [ ] 撮影する内容・構成は決まっているか（フック→本編→CTA）

### 撮影中

- [ ] 9:16 縦向きで撮影しているか
- [ ] 1080p 30fps に設定されているか
- [ ] 顔・商品が中央〜やや上に位置しているか
- [ ] セーフゾーンを意識しているか
- [ ] 1テイクで完璧を目指さない（複数テイク撮ってベストを選ぶ）

### 編集時

- [ ] フックは1秒以内に入っているか
- [ ] 無駄な「間」は全てカットしたか
- [ ] テロップは全編に入っているか
- [ ] テロップのセーフゾーンは守られているか
- [ ] BGM は動画の雰囲気に合っているか
- [ ] 動画の長さは15-30秒に収まっているか
- [ ] 最後にCTAは入っているか

### 投稿時

- [ ] キャプションにキーワードを含めたか（盛岡、シーシャ、岩大 等）
- [ ] ハッシュタグは3-5個か
- [ ] 投稿時間はゴールデンタイム（19-22時）か
- [ ] 他プラットフォームのウォーターマークは入っていないか
