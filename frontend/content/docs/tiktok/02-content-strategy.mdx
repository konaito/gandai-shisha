## バズりやすいシーシャコンテンツの型

### 型一覧

| # | コンテンツの型 | バズ度 | 制作難度 | 例 |
|---|---|---|---|---|
| 1 | **煙アート・トリック** | ★★★★★ | ★★★ | 煙リング、竜巻、ジェリーフィッシュ等をスローモーションで |
| 2 | **ASMR系** | ★★★★ | ★★ | シーシャの炭を載せる音、ブクブク音、吸引音 |
| 3 | **シーシャ入門・解説** | ★★★★ | ★★ | 「シーシャって何？」「初めてのシーシャガイド」 |
| 4 | **フレーバーレビュー** | ★★★ | ★ | 「今日のフレーバーは○○！味の感想は…」 |
| 5 | **Before/After** | ★★★★★ | ★★ | 空き物件 → 改装 → 完成の変化を見せる |
| 6 | **あるある系** | ★★★★ | ★ | 「シーシャ初めての友達あるある」 |
| 7 | **ルームツアー** | ★★★ | ★★ | 完成した空間を映像で紹介 |
| 8 | **裏側・メイキング** | ★★★★ | ★ | フレーバーのミックス作業、セッティング風景 |

### 各型の詳細と制作ポイント

#### 1. 煙アート・トリック動画

シーシャコンテンツで最もバズりやすい鉄板。視覚的なインパクトが強く、ループ再生されやすい。

**制作ポイント**:
- 黒背景 + 間接照明で煙を際立たせる
- スローモーション撮影（iPhone のスロモ機能で十分）
- BGM はローファイ or チル系トレンド音源
- 15秒以内で1つのトリックに集中

#### 2. ASMR系

シーシャ特有の「音」は ASMR と相性抜群。ヘッドホンで聴きたくなる音。

**制作ポイント**:
- 炭をアルミに載せる「ジュッ」という音
- 水パイプのブクブク音
- フレーバーを混ぜる音
- 外部マイク推奨（安価な ASMR マイクで可）

#### 3. シーシャ入門・解説系

「TikTok で検索する若者」に刺さる。TikTok SEO を意識した構成が重要。

**制作ポイント**:
- テロップで「シーシャとは？」「害はあるの？」など疑問形を入れる
- 30秒以内で簡潔に
- キャプションに「#シーシャ初心者」「#シーシャとは」を含める

#### 4. Before/After（改装過程）

BiPシーシャ最大の武器。物件探し→契約→改装→完成の過程は圧倒的なストーリー性がある。

**制作ポイント**:
- 同じアングルで定点撮影し、変化を際立たせる
- トレンド音源に合わせてカットを切り替え
- 「Day 1」「Day 30」「Day 60」のようにシリーズ化

## Build in Public × TikTok の掛け合わせ

### なぜ Build in Public が TikTok と相性が良いのか

| 要素 | 説明 |
|---|---|
| **ストーリー性** | 起業の過程には山あり谷あり。続きが気になる＝フォローされる |
| **リアルさ** | 2026年のTikTokは「リアルな人間味」を重視。作り込まれた広告より日常が伸びる |
| **シリーズ化** | アルゴリズムがシリーズ投稿を優遇。「#BiPシーシャ ep.1」のようにナンバリング |
| **コミュニティ** | 応援してくれるフォロワーが保存・シェアしてくれる |
| **検索流入** | 「大学生 起業」「シーシャ 開業」で検索する人にリーチ |

### Build in Public コンテンツの具体例

#### フェーズ1: 物件探し・準備期

| コンテンツ | フック例 | 想定反応 |
|---|---|---|
| 物件内覧レポート | 「岩大生が盛岡でシーシャ屋を開きたいので物件を見てきた」 | 応援コメント |
| 開業資金の公開 | 「大学生がシーシャ屋を開くのにいくらかかる？」 | 保存されやすい |
| 競合調査 | 「盛岡のシーシャ事情を調べてみた」 | ローカル検索流入 |
| 失敗・壁にぶつかった話 | 「物件審査落ちた…でも諦めない」 | 共感・応援 |

#### フェーズ2: 改装・準備期

| コンテンツ | フック例 | 想定反応 |
|---|---|---|
| DIY改装タイムラプス | 「大学生が1人でシーシャ部屋を作る」 | Before/After バズ |
| 機材選び | 「シーシャの機材、何を買えばいい？」 | 保存されやすい |
| フレーバー試作 | 「オリジナルミックス作ってみた」 | コメントで意見もらえる |
| 内装のこだわり | 「1万円で作るチルい空間」 | インテリア系にもリーチ |

#### フェーズ3: オープン後

| コンテンツ | フック例 | 想定反応 |
|---|---|---|
| 初日のリアル | 「大学生シーシャ屋、オープン初日の売上は…」 | 数字系はバズる |
| お客さんの反応 | 「初めてシーシャを吸う友達の反応」 | あるある・リアクション系 |
| 1ヶ月の振り返り | 「開業1ヶ月で分かったこと5選」 | リスト系は保存される |
| 売上公開 | 「大学生シーシャ屋の1ヶ月の売上を公開」 | 起業系コミュニティに刺さる |

### Build in Public 投稿のテンプレート構成

```
[フック] 0-2秒: 衝撃的な事実 or 疑問で引きつける
  例: 「大学生がシーシャ屋を開いたら、初月の売上は○万円だった」

[背景] 2-5秒: 自分が何者かを簡潔に説明
  例: 「岩大3年の内藤です。盛岡でシーシャスペースを開業しました」

[本編] 5-20秒: 今回のアップデート・学び
  例: 改装の進捗、数字の公開、直面した課題

[CTA] 20-25秒: 次のアクションを促す
  例: 「続きが気になる人はフォロー」「コメントで意見ください」
```

## 投稿頻度とベストな投稿時間

### 投稿頻度の目安

| 頻度 | レベル | 効果 |
|---|---|---|
| **毎日投稿** | 理想 | アルゴリズムに最も好まれる。最速で伸びる |
| **週5回** | 推奨 | 大学生が現実的に続けられるライン |
| **週3回** | 最低ライン | これ以下だとアルゴリズムに忘れられる |

**大学生の現実的な運用**: 週3-5回を目標に、「撮り溜め → 予約投稿」で運用負荷を下げる。日曜にまとめて3-5本撮影し、平日に分散投稿。

### ベストな投稿時間（日本・大学生ターゲット）

| 時間帯 | おすすめ度 | 理由 |
|---|---|---|
| **7:00-9:00** | ★★★ | 通学時間。電車・バスで TikTok を見る |
| **12:00-13:00** | ★★★★ | 昼休み。食堂でスマホを見る時間 |
| **19:00-22:00** | ★★★★★ | ゴールデンタイム。最もアクティブユーザーが多い |
| **22:00-24:00** | ★★★★ | 就寝前の TikTok タイム |

**BiPシーシャの推奨**: **20:00-21:00 投稿**を基本にする。大学生が夕食後にくつろいでスマホを見る時間帯。シーシャの「チル」な雰囲気とも合う。

### 曜日別の傾向

| 曜日 | 傾向 |
|---|---|
| 月-木 | 安定した視聴数。解説系・情報系に向く |
| 金 | 週末の期待感。「週末シーシャどう？」系に最適 |
| 土日 | 閲覧数が最も多い。エンタメ系・映え系に向く |

## 大学生クリエイターのポジショニング

### 「岩大生がシーシャ屋を作る」唯一無二のポジション

TikTok で成功するには「この人しか発信できない内容」を持つことが重要。BiPシーシャには以下の差別化要素がある:

| 差別化ポイント | 説明 |
|---|---|
| **現役大学生 × 起業** | 大学生の起業は珍しく、応援されやすい |
| **地方（盛岡）** | 東京ではなく地方でやることのリアルさ |
| **シーシャという珍しさ** | カフェやバーではなく「シーシャ」の専門性 |
| **Build in Public** | 過程を全公開するスタイルが共感を生む |
| **貸切スペース** | 一般的なシーシャバーとは異なるモデル |

### ペルソナの設計

BiPシーシャの TikTok ターゲットは主に3層:

| ペルソナ | 特徴 | 刺さるコンテンツ |
|---|---|---|
| **岩大生・盛岡の大学生** | メインターゲット。来店に直結 | ローカルネタ、岩大あるある、店舗情報 |
| **シーシャ好き（全国）** | シーシャ文化に興味。フォロワー拡大 | フレーバーレビュー、煙トリック、機材紹介 |
| **起業に興味のある学生** | 同世代の起業に刺さる。Build in Public の本丸 | 開業過程、売上公開、失敗談 |

### クリエイターとしてのブランディング

- **顔出しを推奨**: 2026年のTikTokは「人」が見えるアカウントが優遇される。内藤剛汰本人が出ることで信頼性が増す
- **話し方**: 岩手弁を適度に混ぜると親近感が出る。ただしテロップは標準語で
- **服装**: 清潔感があればOK。ブランドロゴ入りのTシャツを作ると統一感が出る
- **キャラ設定**: 「理工学部なのにシーシャ屋を開く数理系大学生」のギャップが面白い

## ローカルビジネス × TikTok の成功事例

### 事例1: 自動車学校「だつりん先生」（滋賀県）

| 項目 | 内容 |
|---|---|
| **業種** | 自動車学校 |
| **戦略** | 「教習所あるある」コンテンツを投稿 |
| **成果** | 若年層の入校数が増加。「指導官に怒られそう」というネガティブイメージを払拭 |
| **学び** | ターゲット層の不安を「あるある」で笑いに変えると集客に直結する |

**BiPシーシャへの応用**: 「シーシャ初心者あるある」で、シーシャに対する不安・ハードルを下げる。

### 事例2: 地方飲食店の広告費ゼロ集客

| 項目 | 内容 |
|---|---|
| **業種** | 地方の飲食店 |
| **戦略** | 「現場音」×「商品紹介」のリール戦略 |
| **成果** | 広告費ゼロでフォロワー1万人突破、売上150%UP |
| **学び** | 高品質な映像より「リアルな現場感」が地方ビジネスでは効く |

**BiPシーシャへの応用**: シーシャを準備する「現場音」（炭の音、水の音）で ASMR 的な動画を制作。

### 事例3: SideHustleStack（TikTok × Build in Public）

| 項目 | 内容 |
|---|---|
| **業種** | Webサービス（副業マッチング） |
| **戦略** | 起業過程を TikTok で公開。若者向け B2C |
| **成果** | TikTok で複数回バイラル化 |
| **学び** | Build in Public × TikTok は若者向け B2C と非常に相性が良い |

**BiPシーシャへの応用**: まさに同じ構図。若者向けサービスの構築過程を公開する戦略はそのまま使える。
